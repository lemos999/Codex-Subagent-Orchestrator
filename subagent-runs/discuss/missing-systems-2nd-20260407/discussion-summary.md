### 철학 관점 (Elara Voss / Codex)
1. 트라우마/치유 — 비선형 감정 잔류, 과거의 현재 침범
2. 양심/도덕 — 이득 계산 외부의 내부 제약
3. 호기심/탐구 — 무목적 탐색 충동
4. 유머/놀이 — 존재 긍정, 관계 분위기 변수
5. 노화/시간감각 — 주관적 생애 위치, 목표 재편

### 운영 관점 (Rex Baek / Opus) — 코드 분석 기반
1. **틱 데몬** — 가장 치명적. 시뮬레이션을 구동하는 주체가 없음. 현재는 "사람이 명령할 때만 움직이는 시스템"
2. 상태 내구성 — JSON SOT에 백업/트랜잭션 보장 없음. 원자적 쓰기+스냅샷+복구 CLI 필수
3. 이벤트 버스 + 알림 — push 경로 부재. WebSocket + 외부 웹훅(Discord 등)
4. 멀티세션 격리 — 시뮬레이션 코어 = LLM 없이 돌아가는 순수 로직, LLM = 고수준 개입만
