- 시장(행정 권력) vs 가문/컴퍼니 수장(무력/실권 권력) 이원 구조

### Step 5
마지막으로 미궁 구조, 외부 환경, 벨트, 도시 감정선, 서사적 역할을 정교화한다.

## 10. 작업 방식 규칙

1. 없는 말을 만들지 말 것.
2. 문서와 채팅에 없는 새 설정은 **무단 확정 금지**.
3. 좋은 아이디어가 있어도 **사용자 승인 전에는 ‘제안’ 상태**로 둘 것.
4. 설정끼리 연결할 때는 반드시
   - 철학
   - 행위
   - 이득
   - 당위성
   을 추적할 것.
5. 강한 설정은 삭제보다 **강등 보존 / 전승화 / 역사화 / 오해화**를 우선 검토할 것.
6. 종족과 조직의 말투, 행동, 정책은 철학에서 나오게 만들 것.
