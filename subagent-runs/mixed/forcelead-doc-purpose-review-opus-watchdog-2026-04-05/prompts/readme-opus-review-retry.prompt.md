## 4. 현재 확정된 핵심 캐논

### 4-1. 세계 질서
- 현행 세계 질서는 **지구권 4권역 12도시 + 중립 허브 이르카 + WLO** 중심으로 운영된다.

### 4-2. 4권역 12도시 + 1중립
- 아신 제국: 금오 / 미리내 / 한가람
- 엘시움 연합: 올리브 / 에덴 / 아카디아

- 이르카: 진정한 중립 허브

### 4-3. 중립 도시 원칙
사용자 최신 지시에 따라 **진정한 중립 도시**는 다음 조건을 만족해야 한다.

- 갈등/협력/경쟁/종속 관계에 직접 연루되지 않음
- 외교적 완충지대 역할 수행

현재 이에 가장 부합하는 도시는 **이르카**다.

### 4-4. 도깨비
- 도깨비는 최신 매뉴얼 기준으로 **노바에 남은 프리오리안 잔존 개체**다.
- 도술은 **의지 기반 자연 조작**이다.
- 도깨비 철학은 **자연주의 / 우주론 / 인식론**을 중심으로 작동한다.
- 도깨비는 인간처럼 도시를 만들지 않으며, 본거지는 **창공의 육지**다.
- 인간/요괴/뱀파이어 전쟁에 직접 개입하지 않지만, 질서 붕괴급 사건에는 개입한다.

### 4-5. 밤손님의 찬가
- 도깨비에게 “맡겨진 책”이 아니다.
- 프리오리안이 계약상 활용할 수 없었고, 노바에 **남겨진 기록**이다.
- 도깨비는 수탁자가 아니라, 결과적으로 그 주변에 남게 된 존재들이다.

### 4-6. 뱀파이어
- 뱀파이어는 외부 유입 종족이 아니라 **인간 내부에서 갈라진 변이 종**으로 정리한다.
- 진보파/전통파/효율/지배의 철학 축이 중요하다.
- Solar 보정 체계는 유지한다.
- 단, 매뉴얼 일부의 “노바인 변이” 서술은 지구 배경 확정과 충돌 여지가 있어 후속 정리가 필요하다.

### 4-7. 인간
- 현재 서사의 기본 인간은 **지구 인류권** 기준으로 본다.
- 매뉴얼 내부의 일부 노바 중심 인간 서술은 후속 최신화가 필요하다.
- 마르스 인류는 가이드 시스템의 기원 층으로 유지한다.
