# 🎨 바이브 콘텐츠 크리에이터 커뮤니티

> AI를 활용해서 만든 콘텐츠들을 보여주고, 공유하고, 함께 성장하는 공간

---

## 📌 프로젝트 개요

| 항목 | 내용 |
|------|------|
| **목적** | AI 도구를 활용해 만든 콘텐츠를 전시·공유하는 커뮤니티 |
| **대상** | 바이브 콘텐츠 창작자 (Vibe Coders, AI 아티스트, 프롬프트 엔지니어 등) |
| **핵심 가치** | 자랑 + 정보 공유 + 영감 |

---

## 🧩 핵심 기능

### 1. 갤러리 + 상세페이지
- AI 콘텐츠 카드 그리드 레이아웃
- 클릭 시 상세 보기 (사용 도구, 프롬프트, 제작 과정 등)
- 콘텐츠 성격에 따른 차별화된 카드 디자인

### 2. 커뮤니티 게시판
- 글 작성 / 댓글 / 좋아요
- 카테고리별 분류 (자유, 질문, 팁 공유, 프로젝트 모집 등)

### 3. 프로필 & 포트폴리오
- 크리에이터별 프로필 페이지
- 작품 모음 (포트폴리오)
- 활동 히스토리

### 4. AI 도구 리뷰 / 가이드
- AI 도구 사용법, 팁
- 도구별 리뷰 섹션
- 추천 워크플로우

### 5. GitHub 연동 UI/UX
- GitHub 프로젝트 링크 / 임베드
- 코드 스니펫 미리보기
- 레포지토리 연결

---

## 🎯 디자인 원칙

- **일관성 있는 구성** — 통일된 디자인 시스템
- **콘텐츠 성격별 차별화** — 이미지/영상/코드/음악 등 타입별 최적화된 표현
- **GitHub 친화적 UI/UX** — 개발자 커뮤니티 특성 반영
