# 작품 상세 + 제작과정 (Core 2)

> 최종 갱신: 2026-03-24
> 주관 섹션: 작품 상세
> 참조 섹션: 갤러리(Core 1), 등급 시스템(Core 3)

## [확정] 상세 페이지 레이아웃

원스크롤(One-scroll) — 세로 배치 순서:

| 순서 | 섹션 |
|------|------|
| 1 | 작품 (이미지/결과물) |
| 2 | 제작과정 (마크다운 에디터 입력 내용) |
| 3 | 댓글 |

## [확정] 제작과정 입력 방식

자유 텍스트 — 마크다운 에디터 사용

## [확정] 컴포넌트 인터페이스

| 연결 컴포넌트 | 입력 (값, 범위) | 출력 (값, 범위) | 실패 시 기본값 |
|-------------|---------------|---------------|-------------|
| 갤러리(Core 1) | 카드 클릭 이벤트 | `postId` 수신 | — |
| 상세 페이지 진입 시 | `postId` | `viewCount` +1 (비동기) | 실패 시 카운트 유지 |
| 댓글 작성 시 | `userId`, `postId` | `commentCount` +1 → 등급 시스템 | 실패 시 스킵 |
| 댓글 수신 시 | `postId` (작성자) | `commentReceived` +1 → 등급 시스템 | 실패 시 스킵 |

## [확정] viewCount 처리 규칙

- 상세 페이지 진입 시 `viewCount` +1 (표시용)
- `viewCount`는 `activityScore`에 포함하지 않음 (조작 방지)
- `viewCount`는 명예의 전당 선정 기준에 포함하지 않음
- 수치 정본: [shared-parameter-registry.md](shared-parameter-registry.md)
