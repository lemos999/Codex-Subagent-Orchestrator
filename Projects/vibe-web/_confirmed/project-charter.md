# Project Charter (Phase 1)

> 최종 갱신: 2026-03-24
> 모든 에이전트가 참조하는 프로젝트 근본 정체성.

## [확정] Primary Outcome

AI 콘텐츠를 자랑하고 싶은 욕구 충족 — 핵심 시나리오: 작품 업로드 → 피드백 → 등급 상승 → 명예의 전당

## [확정] Operating Loop

| 레이어 | 시간 범위 | 루프 |
|--------|----------|------|
| 마이크로 | 1회 세션 | 브라우징 → 클릭 → 좋아요 |
| 미들 | 수일~수주 | 업로드 → 피드백 → 등급 상승 |
| 매크로 | 수개월 | 포트폴리오 누적 → 명예의 전당 입성 |

## [확정] Baseline Expectations

포함: 갤러리, 작품 상세+제작과정, 사용자 등급, 명예의 전당, 커뮤니티 게시판, 사용자 인증, 프로필/포트폴리오, 카테고리/태그, 기본 텍스트 검색, 반응형 레이아웃(데스크톱+모바일)
제외: 프레스티지 시스템 → 대체: v2 검토 (MAU 1,000+ 이후)

## [확정] Differentiation Thesis

"AI 창작 갤러리인데 프롬프트·도구·워크플로우 등 제작과정을 함께 공유하기 때문에 재현 가능한 영감을 제공한다"

## [확정] Target Audience

| 항목 | 결정 |
|------|------|
| 대상 | 바이브 코더 / AI 아티스트 |
| 사용 환경 | 데스크톱 + 모바일 |
| 허용 복잡도 | 낮음 (린 MVP) |
| 사용 빈도 | 매일 방문 |
| 개발 규모 | 1인 개발, 린 스코프 |
