# 커뮤니티 게시판 (Core 5)

> 최종 갱신: 2026-03-24
> 주관 섹션: 커뮤니티 게시판
> 참조 섹션: 등급 시스템(Core 3), 명예의 전당(Core 4), 공통 규칙

## [확정] 카테고리 구성

5개 고정 카테고리:

| 번호 | 카테고리명 |
|------|----------|
| 1 | 자유 |
| 2 | 질문 |
| 3 | 팁 공유 |
| 4 | 프로젝트 모집 |
| 5 | 공지 |

## [확정] 컴포넌트 인터페이스

| 연결 컴포넌트 | 입력 (값, 범위) | 출력 (값, 범위) | 실패 시 기본값 |
|-------------|---------------|---------------|-------------|
| 사용자 인증 | `userId` | — | — |
| 카테고리 | `categoryId` (1~5) | — | — |
| 등급 시스템(Core 3) | 댓글 작성 이벤트 | `commentCount` → `activityScore` 반영 | 스킵 |
| 등급 시스템(Core 3) | 좋아요 수신 이벤트 | `likeReceived` → `activityScore` 반영 | 스킵 |
| 명예의 전당(Core 4) | 좋아요 이벤트 | `likeCount` (전체 합산) | 0 |

## [확정] 신고/제재

신고 기능 존재 — 최소 카테고리 4개:

| 번호 | 신고 유형 |
|------|----------|
| 1 | 스팸 |
| 2 | 욕설 |
| 3 | 저작권 침해 |
| 4 | 기타 |

> 제재 절차 세부 구현은 구현자 재량
> 수치 정본: [common-rules.md](common-rules.md)
