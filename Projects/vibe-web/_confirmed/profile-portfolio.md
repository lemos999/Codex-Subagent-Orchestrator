# 프로필 / 포트폴리오 (Support)

> 최종 갱신: 2026-03-24
> 참조 섹션: 등급 시스템(Core 3), 갤러리(Core 1)

## [확정] 프로필 페이지 구성

| 섹션 | 내용 | 데이터 소스 |
|------|------|-----------|
| 사용자 정보 | 닉네임, 아바타, 가입일 | auth-system → OAuth 프로필 |
| 등급 배지 | 현재 단계 + 레벨 (예: "크리에이터 Lv.3") | grade-system → userLevel |
| expPoints 바 | 현재 경험치 / 다음 단계 임계치 진행률 | grade-system → expPoints |
| 작품 목록 | 내가 올린 갤러리 작품 그리드 (메이슨리) | gallery-system → userId 필터 |
| 활동 히스토리 | 최근 댓글, 좋아요, 업로드 이력 (최대 20건) | 각 컴포넌트 → userId 필터 |
| 명예의 전당 이력 | 주간/월간 베스트 선정 횟수 | hall-of-fame → userId |

## [확정] 컴포넌트 인터페이스

| 연결 컴포넌트 | 입력 | 출력 | 실패 시 기본값 |
|-------------|------|------|-------------|
| 등급 시스템(Core 3) | — | `userLevel`, `expPoints` 수신 | 뉴비 Lv.1 표시 |
| 갤러리(Core 1) | `userId` | 작품 목록 (postId[]) | 빈 목록 |
| 명예의 전당(Core 4) | `userId` | 선정 횟수 | 0회 |
