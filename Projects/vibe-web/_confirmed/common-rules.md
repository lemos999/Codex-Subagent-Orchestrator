| 항목 | 처리 방식 |
|------|----------|

## [확정] 좋아요 유일성 규칙

- 계정당 게시물 1좋아요만 허용
- 좋아요 취소 시 `likeCount` -1

## [확정] 업로드 점수 일일 상한

- 1건당 +10pt
- 일일 상한: 30pt (3건/일)
- 목적: 대량 업로드 파밍 방지

## [확정] 댓글 점수 일일 상한

- 1건당 +2pt
- 일일 상한: 20pt

## [확정] 신고/제재 카테고리

최소 4개:

| 번호 | 유형 |
|------|------|
| 1 | 스팸 |
| 2 | 욕설 |
| 3 | 저작권 침해 |
| 4 | 기타 |

> 제재 절차 세부 구현: 구현자 재량

## [확정] HoF 동점 처리 규칙

동점 시 최신 게시물 우선 (게시물 생성 시각 기준)

## [확정] 삭제 쿨다운

삭제 후 24시간 재업로드 1건 제한 (갤러리 전체 기준, 카테고리 무관)

모든 시간 기반 집계는 UTC 기준
