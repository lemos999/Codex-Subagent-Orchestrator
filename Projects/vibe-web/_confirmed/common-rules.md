## [확정] 제재 절차 (DEVELOPMENT-BIBLE §27)

| 단계 | 조건 | 처리 |
|------|------|------|
| 자동 숨김 | 동일 콘텐츠 신고 3건 이상 누적 | 콘텐츠 자동 숨김 + 관리자 큐 등록 |
| 관리자 검토 | 관리자 큐 진입 | 복원 / 삭제 / 계정 경고 중 선택 |
| 경고 | 관리자 판단 | 사용자에게 경고 알림 |
| 정지 | 경고 3회 누적 | 7일 활동 정지 (로그인 가능, 작성 불가) |
| 영구 정지 | 정지 후 재위반 | 계정 영구 정지 (기존 콘텐츠는 유지, 작성자 "[정지된 사용자]") |

차단된 사용자의 기존 콘텐츠: 유지 (BIBLE §27: "처리 정책 사전 결정")
차단 이력: 감사 로그 기록 (BIBLE §27)
