# 갤러리 시스템 (Core 1)

> 최종 갱신: 2026-03-24
> 주관 섹션: 갤러리
> 참조 섹션: 작품 상세(Core 2), 등급 시스템(Core 3), 명예의 전당(Core 4)

## [확정] 레이아웃

메이슨리(Pinterest식) — 카드 높이 가변, 콘텐츠 타입별 자연스러운 배치

## [확정] 카드 정보 구성

리치(Rich) 카드 — 다음 6가지 요소를 모두 포함:

| 요소 | 내용 |
|------|------|
| 썸네일 | 작품 대표 이미지 |
| 제목 | 작품 제목 |
| 작성자 | 사용자명 (→ `userId` 참조) |
| 좋아요 수 | `likeCount` (정본: [shared-parameter-registry.md](shared-parameter-registry.md)) |
| 도구 태그 | 사용 AI 도구 태그 |
| 제작과정 미리보기 | 1줄 요약 |

## [확정] 컴포넌트 인터페이스

| 연결 컴포넌트 | 입력 (값, 범위) | 출력 (값, 범위) | 실패 시 기본값 |
|-------------|---------------|---------------|-------------|
| 작품 상세(Core 2) | 카드 클릭 | `postId` | — |
| 등급 시스템(Core 3) | 업로드 이벤트 | `uploadCount` (갤러리 작품 전용) | 미전달 시 등급 업데이트 스킵 |
| 명예의 전당(Core 4) | 게시물 좋아요 | `likeCount` (갤러리 전체 합산) | 0 |

## [확정] 업로드 점수 규칙

- 업로드 1건당 `activityScore` +10pt
- 일일 업로드 점수 상한: 30pt (3건/일)
- 수치 정본: [common-rules.md](common-rules.md)
