> 최종 갱신: 2026-03-24
> 근거: DEVELOPMENT-BIBLE §27 "DB LIKE 금지, 전문 검색 엔진"

## [확정] 검색 범위

| 대상 | 검색 필드 | 우선순위 |
|------|----------|---------|
| 갤러리 작품 | 제목, 도구 태그, 제작과정 미리보기 | 1순위 |
| 게시판 글 | 제목, 카테고리 | 2순위 |
| 사용자 | 닉네임 | 3순위 |

## [확정] 검색 방식

| 단계 | 방식 | 비고 |
|------|------|------|
| MVP | PostgreSQL Full-text Search (tsvector) | Prisma $queryRaw |
| v2 | Meilisearch 또는 동급 | BIBLE §27 권장 |

## [확정] 결과 정렬

- 기본: 관련도순 (ts_rank)
- 대안: 최신순, 좋아요순 (사용자 선택)

## [확정] 페이지네이션

커서 기반 (BIBLE §27: "오프셋은 대량 데이터에서 느림")

## [확정] 컴포넌트 인터페이스

| 연결 컴포넌트 | 입력 | 출력 | 실패 시 기본값 |
|-------------|------|------|-------------|
| 갤러리(Core 1) | 검색 쿼리 | postId[] (매칭 결과) | 빈 목록 + "결과 없음" |
| 게시판(Core 5) | 검색 쿼리 | boardPostId[] | 빈 목록 |
| 사용자 | 검색 쿼리 | userId[] | 빈 목록 |
