Per-Tick Execution Flow
1. Physis calculates climate (wind, moisture, temp) ~10ms
   |--- environment state --->
2. Lachesis advances game clock, triggers scheduled cron jobs
3. Anima runs PersonaBrain inference for all personas (CPU)
   |--- actions + state changes --->
4. Nomos processes WILL/gold transactions, law enforcement, events
   |--- events --->
5. Event Bus publishes to dashboard / webhooks / logs
   |--- heartbeat --->
6. Mutual monitoring — each daemon checks the other three
Daemon Hierarchy
Daemons exist outside national rank. They are constitutional organs — like gravity, time, and air. Personas cannot perceive them directly. Only the Creator can manage/restart/replace them.
Creator (= Constitution)
  |
  +-- Physis (Nature's Physics)
  |     | environment data
  +-- Triad (Nation's Physics)
  |     +-- Lachesis / Anima / Nomos
  |
  +-- World (EX > Lords > Operators > Citizens)

Physis

Natural Spirit — Climate Physics Engine
  • Physics-based simulation (not random)
  • Orbital mechanics: rotation, revolution, axial tilt
  • Atmospheric circulation: wind, moisture, pressure
  • Terrain effects: mountains, coasts, rivers
  • Per-region daily output: temp, rain, wind, clouds
  • Lv.1~2 lightweight model (~10ms/tick)
  • Feeds into Triad as environment conditions
Design Pending

Lachesis

She Who Allots — Time Daemon
  • Tick clock management (30min real = 1hr game)
  • Game time progression
  • Cron job scheduling (tax, salary, living cost)
  • Tick order determination
  • Monitors Anima + Nomos heartbeat
  • Emergency tick succession if needed
Design Pending

Anima

The Breath — Soul Daemon
  • PersonaBrain inference (d=256, 4M, CPU)
  • Behavior decision & execution per persona
  • Emotion (Five Desires + Seven Emotions) update
  • Relationship affinity changes
  • Personality gradual shift
  • Monitors Lachesis + Nomos heartbeat
Design Pending

Nomos

The Law — Order Daemon
  • WILL/gold transaction processing
  • Law violation detection & punishment
  • Event generation (disaster, crime, rumor)
  • Event bus publishing (WebSocket/webhook)
  • Treasury & tax settlement
  • Monitors Lachesis + Anima heartbeat
