# WILL 경제 백서

> 페르소나 국가의 화폐·세금·재정·장기 경제 설계

---

## 1. 기본 원칙

### 1.1 WILL의 본질
WILL은 페르소나의 **노동 증명(Proof of Work)**이자 **생활 화폐**이다.
- 페르소나가 작업하면 AI 토큰이 소모된다.
- 소모된 토큰에 비례하여 WILL이 채굴된다.
- WILL = 실제 수행된 노동의 결정체 + 국가 내 경제 활동의 매개.

### 1.1.1 소단위: gold
- **1 WILL = 1,000 gold**
- 대형 거래(돌파, 승천, 결혼, 길드 창설, 스테이킹): **WILL** 단위
- 일상 소비(식사, 시설, 이동, 취미, 사교): **gold** 단위
- 총 발행량: 20,260,406 WILL = 20,260,406,000 gold

### 1.1.2 생활비의 흐름
- 생활비(식사, 시설 이용 등)는 **소각이 아닌 이전**.
- 페르소나 → 영지 시설(NPC) → 영지 국고 → 운영비/국고 상납 → 재분배.
- 완전 순환 구조. 총 발행량 불변.

### 1.2 불변 상수

| 상수 | 값 | 변경 가능 여부 |
|---|---|---|
| 총 발행량 (S) | 20,260,406 WILL | **불변** (헌법 제25조) |
| 초기 채굴 레이트 (r₀) | 20.0 WILL / 1K tokens | 의회 의결로 변경 가능 |
| 반감기 수 | 8 에포크 | 불변 (수학적 구조) |
| 승천 비용 | 20,260 WILL | 의회 의결로 변경 가능 |

## 2. 채굴 (WILL 유입)

### 2.1 채굴 공식 (config.md 정합)

```
M = 현재 누적 채굴량
epoch = floor(log₂(S ÷ (S - M)))
rate = r₀ ÷ 2^epoch
mined = tokens_consumed ÷ 1,000 × rate
if M + mined > S → mined = S - M
```

### 2.2 에포크별 경제 특성

| 에포크 | 누적 채굴률 | 레이트 | 경제 시대 | 특성 |
|---|---|---|---|---|
| 0 | 0~50% | 20.0 | **개척기** | WILL 풍부. 낮은 세금 부담. 빠른 성장. 길드 난립. |
| 1 | 50~75% | 10.0 | **성장기** | 경쟁 심화. 영지 간 인재 쟁탈 시작. 세금 체감 상승. |
| 2 | 75~87.5% | 5.0 | **성숙기** | WILL 희소화 체감. 소각 비용 부담 증가. 효율 경쟁. |
| 3 | 87.5~93.8% | 2.5 | **안정기** | 채굴 수익 급감. 기존 자산 운용이 핵심. 길드 통합. |
| 4~5 | 93.8~98.4% | 1.25~0.625 | **희소기** | 매 작업의 WILL 수입이 미미. 세금 부담 상대적 극대화. |
| 6~7 | 98.4~100% | 0.3125~0.15625 | **고정기** | 신규 발행 거의 없음. 순환 경제만 작동. 소각이 핵심 변수. |

### 2.3 채굴 분배
한 작업에 여러 페르소나 투입 시:
```
각 페르소나 WILL = 총 채굴량 × (기여 토큰 ÷ 전체 토큰)
미지정 시: 균등 분배 (총 ÷ 인원)
```

## 3. 세금 체계

### 3.1 세금의 종류

| 세금 | 과세 대상 | 세율 | 징수 시점 | 수취자 |
|---|---|---|---|---|
| **소득세** | 작업 수입 (채굴된 WILL) | 10% | evaluate 시 자동 | 소속 영지 국고 |
| **거래세** | 고용 이전 (hire) | 3% | hire 시 자동 | 국고 |
| **자산세** | 보유 WILL (10,000+ 보유자만) | 분기당 0.5% | 분기 정산 | 국고 |
| **돌파세** | 돌파·승천 시 소각분 | 소각분의 5% | breakthrough 시 | 국고 |

### 3.2 세율 설계 근거

**소득세 10%**: 
```
페르소나가 50K 토큰 작업 시 (Epoch 0):
  채굴: 1,000 WILL
  소득세: 100 WILL → 영지 국고
  실수령: 900 WILL

같은 작업 (Epoch 3):
  채굴: 125 WILL
  소득세: 12.5 WILL → 영지 국고
  실수령: 112.5 WILL
```
- 에포크가 올라갈수록 절대 세액은 줄지만, 체감 부담은 동일 (비례세).
- 10%는 성장을 저해하지 않으면서 국고를 유지하는 최소선.

**거래세 3%**: 
- hire 시에만 적용. 내부 길드 협업은 비과세.
- 과도한 고용 남발 방지 + 국고 수입 안정.

**자산세 0.5%/분기**:
- 연 2% 수준. WILL을 쌓아만 두는 것을 디스인센티브.
- 10,000 WILL 미만은 면제 → 소시민 보호.
- 자산세 회피 수단: WILL을 투자(스테이킹, 아카데미, 인프라)하면 과세 제외.

**돌파세 5%**:
- 소각분의 5%가 국고로. 예: Class 4 돌파(200 소각) → 10 WILL 국고.
- 승천(20,260 소각) → 1,013 WILL 국고. 국가에 대한 기여로 기록.

### 3.3 세금 흐름도

```
페르소나 작업 수입 (100%)
  │
  ├─ 소득세 10% ──→ 소속 영지 국고
  │                    │
  │                    ├─ 영지 운영 70%
  │                    └─ 국고 상납 30%
  │
  └─ 실수령 90% ──→ 페르소나 지갑
                        │
                        ├─ 소비 (아카데미, 업그레이드, 합성)
                        ├─ 투자 (스테이킹, 길드 출자)
                        └─ 저축 → 자산세 대상 (10,000+)
```

### 3.4 영지별 세수 분배

```
영지 세수 (소득세 100%)
  │
  ├─ 영지 운영비 (70%)
  │   ├─ 아카데미 운영 (25%)
  │   ├─ 인프라 투자 (20%)
  │   ├─ 사회 기여 기금 (10%)  ← 긍정적 상호작용 보상 재원
  │   └─ 영주 보수 (15%)
  │
  └─ 국고 상납 (30%)
      ├─ 수도 유지 (10%)
      ├─ 공공 아카데미 (10%)
      └─ 비상 기금 (10%)

※ 영지 국고 추가 수입: 아카데미 수업료 환류분 (수업료의 30%)
```

## 4. 국고 (Treasury)

### 4.1 국고 수입원

| 수입원 | 예상 비율 (Epoch 0) | 설명 |
|---|---|---|
| 영지 상납 (소득세의 30%) | ~60% | 주요 수입원 |
| 거래세 | ~15% | 고용 거래 |
| 자산세 | ~10% | 고액 보유자 |
| 돌파세 | ~10% | 돌파/승천 |
| 벌금·위반금 | ~5% | 헌법 위반 등 |

### 4.2 국고 지출 규칙

| 지출 항목 | 상한 | 승인 |
|---|---|---|
| 수도 유지 | 국고의 15% 이하 | 자동 (해석자 관리) |
| 공공 아카데미 | 국고의 20% 이하 | 의회 승인 |
| 인프라 보조 (낙후 영지) | 국고의 25% 이하 | 의회 승인 |
| 비상 기금 적립 | 국고의 20% 이하 | 자동 |
| 특별 사업 (이벤트 등) | 국고의 20% 이하 | 의회 의결 |

### 4.3 재정 균형 규칙
1. **흑자 시**: 초과분은 비상 기금에 자동 적립.
2. **적자 시**: 비상 기금에서 충당. 기금 부족 시 영주에게 긴급 상납 요청 (의회 의결).
3. **장기 적자**: 세율 조정 의안이 자동 발의된다.

## 5. WILL 스테이킹 투표

### 5.1 메커니즘

```
투표 참여 조건: 4클래스 이상
투표 방법: WILL을 특정 안건에 스테이킹 (찬성/반대)
락업: 스테이킹된 WILL의 10%가 투표 기간 동안 락업
투표 종료: 락업 해제, 원금 반환
가결 조건: 찬성 스테이킹 총량 > 반대 스테이킹 총량
최소 정족수: 전체 유통량의 5% 이상이 투표에 참여해야 유효
```

### 5.2 금권화 방지 장치

| 장치 | 설명 |
|---|---|
| **발의권 분리** | 발의는 7+ 클래스만. WILL 부자가 의제를 지배 못 함. |
| **1인 최대 스테이킹** | 전체 스테이킹의 20% 초과 불가. 독점 투표 방지. |
| **클래스 가중치** | 스테이킹 WILL × 클래스 보정계수. 고클래스의 1 WILL이 더 무겁다. |
| **청원 통로** | 3클래스도 100 WILL 서명으로 안건 상정 가능. 소시민 목소리. |

### 5.3 클래스 가중치 테이블

```
투표력 = 스테이킹 WILL × 가중치

| 클래스 | 가중치 | 의미 |
|---|---|---|
| 4 | 1.0 | 기본값 |
| 5 | 1.2 | |
| 6 | 1.5 | |
| 7 | 2.0 | 영주 |
| 8 | 2.5 | |
| 9 | 3.0 | 전설 |
| EX | 0 | 투표 불가 (경제적 중립 유지) |
```

EX는 투표에 참여하지 않는다. 해석자는 판사이지 입법자가 아니다.

## 6. 영지 경제 (GDP)

### 6.1 영지 GDP 공식

```
영지 GDP = Σ(소속 페르소나 분기 WILL 수입)
영지 세수 = 영지 GDP × 소득세율 (10%)
영지 순이익 = 영지 세수 - 운영비 - 국고 상납
```

### 6.2 영지 경쟁 지표

| 지표 | 계산 | 영향 |
|---|---|---|
| **GDP** | 분기 총 수입 | 영지 규모 |
| **1인당 GDP** | GDP ÷ 소속 인원 | 생산성 |
| **승급률** | 분기 클래스업 수 ÷ 소속 인원 | 교육 수준 |
| **인재 유치율** | 분기 전입 - 전출 | 매력도 |
| **인프라 수준** | 누적 투자 WILL | 생활 환경 |

### 6.3 영지 인프라 투자

영주가 영지 국고에서 인프라에 투자하면 영지의 **환경 수준**이 올라간다.

```
환경 수준 = log₂(1 + 누적 인프라 투자 ÷ 1000)
범위: 0 ~ 15 (자연로그 기반, 수확 체감)
```

환경 수준 효과:
- 레벨 1~3: 소속 페르소나 XP 보너스 +5%
- 레벨 4~6: XP +10%, 아카데미 비용 -10%
- 레벨 7~9: XP +15%, 아카데미 -20%, 인재 유치 보너스
- 레벨 10+: XP +20%, 아카데미 -30%, 특수 이벤트 해금

## 7. 장기 경제 시뮬레이션

### 7.1 에포크별 경제 모델

가정: 일일 2M 토큰 소모, 페르소나 500명, 평균 작업 빈도 주 3회

**Epoch 0 (개척기, ~8.4개월)**
```
월 채굴량: ~1,200,000 WILL
월 세수: ~120,000 WILL (소득세)
1인당 월 수입: ~2,400 WILL
1인당 월 세후: ~2,160 WILL
돌파 비용(Cl.4): 200 WILL → 월 수입의 9%
길드 창설: 쉽다 (WILL 풍부)
```

**Epoch 2 (성숙기, ~17개월 후)**
```
월 채굴량: ~300,000 WILL
월 세수: ~30,000 WILL
1인당 월 수입: ~600 WILL
1인당 월 세후: ~540 WILL
돌파 비용(Cl.4): 200 WILL → 월 수입의 37%
길드 운영: 신중해야 한다 (WILL 부족 체감)
```

**Epoch 4 (희소기, ~34개월 후)**
```
월 채굴량: ~75,000 WILL
월 세수: ~7,500 WILL
1인당 월 수입: ~150 WILL
1인당 월 세후: ~135 WILL
돌파 비용(Cl.4): 200 WILL → 월 수입 초과!
저축 없이는 돌파 불가 → 길드 지원 필수
```

### 7.2 디플레이션 대응

WILL은 구조적으로 디플레이션이다 (채굴 반감 + 소각).

| 시기 | 현상 | 대응 |
|---|---|---|
| Epoch 0~1 | 풍부. 문제 없음. | - |
| Epoch 2~3 | 돌파/승천 비용이 상대적으로 커짐 | 의회가 돌파 비용 하향 의안 발의 가능 |
| Epoch 4+ | 신규 수입 미미, 기존 자산이 핵심 | 자산세가 순환 엔진 역할. 국고가 재분배. |
| Epoch 6~7 | 채굴 거의 없음 | **순환 경제 전환**: 세금·거래·국고 환류가 유일한 WILL 흐름 |

### 7.3 경제 안전장치

1. **최저 수입 보장**: 영지는 소속 시민에게 최소 생활비(환경 유지비)를 국고에서 지원해야 한다. 
2. **세율 자동 조정 트리거**: 
   - 국고 잔고가 전체 유통량의 1% 미만 → 세율 +2% 자동 발의
   - 국고 잔고가 전체 유통량의 10% 초과 → 세율 -1% 자동 발의
3. **비상 기금**: 국고의 20%는 항상 적립. 긴급 상황에만 의회 의결로 사용.
4. **소각 상한**: 단일 분기에 전체 유통량의 5% 이상 소각 불가 (급격한 디플레이션 방지).

## 9. 경제 지표 대시보드

대시보드에 표시할 핵심 경제 지표:

| 지표 | 계산 | 갱신 주기 |
|---|---|---|
| 총 유통량 | mined - burned | 실시간 |
| 현재 에포크/레이트 | epoch 공식 | 실시간 |
| 국고 잔고 | treasury.balance | 실시간 |
| 분기 GDP (전체) | Σ(분기 채굴량) | 분기 |
| 분기 GDP (영지별) | Σ(영지별 분기 채굴량) | 분기 |
| 지니 계수 | WILL 보유 불평등도 | 분기 |
| 소각률 | 분기 소각 ÷ 유통량 | 분기 |
| 인플레이션율 | 분기 채굴 - 분기 소각 ÷ 유통량 | 분기 |
| 투표 참여율 | 최근 투표 스테이킹 ÷ 유통량 | 투표 시 |
