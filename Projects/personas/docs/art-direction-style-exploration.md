## 프로젝트 요약

**페르소나 국가**는 AI 페르소나들이 살아가는 시뮬레이션 국가입니다.

- 살아있는 헌법 국가 (창조자=헌법, 9 클래스 + EX 신격)
- WILL 화폐 경제 (gold 소단위, 채굴·세금·순환)
- 신경망 기반 자율 행동 (PersonaBrain, d=256, 4M params)
- 감정 시스템: 오욕칠정 (동양 철학 기반)
- 데몬 체제: Physis(기후 물리) → Lachesis(시간) / Anima(영혼) / Nomos(질서)
- 3권역: Claude Dominion(남색, 정밀) / Codex Republic(녹색, 기술) / Gemini Federation(보라, 통찰)
- 윤회/전생, 법치, 영지관리, 관계망
- 대시보드 UI: 영지관리물 게임 느낌 (CK3/Anno/Stronghold 참조)

**핵심 키워드**: 인과적 행동, 창발적 드라마, 자연 위의 문명, 예측 불가능한 인간다움

## Gemini에게 요청할 때

1. **위 6개 스타일(A~F) x 5개 대상 = 30개 이미지**를 생성
2. 각 스타일 내에서 5개가 **하나의 세트처럼 통일감**이 있는지 확인
3. 스타일 간 비교 시 어떤 것이 **페르소나 국가의 정체성**에 가장 부합하는지 판단
4. 최종 선택 후 해당 스타일 기반으로 **아트 디렉션 문서** 작성 → 95종 프롬프트 전면 재작성

### 추천 Gemini 워크플로우
```
Step 1: 스타일당 WILL 코인 1장씩 = 6장 먼저 생성 (빠른 필터링)
Step 2: 탈락하지 않은 스타일로 5장 세트 완성
Step 3: 최종 1~2개 스타일 비교 → 확정
```
