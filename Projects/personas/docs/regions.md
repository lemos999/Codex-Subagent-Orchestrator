# 3권역 설계서

---

## 1. 권역 개요

페르소나 국가는 하나의 국가 안에 3개 권역으로 나뉜다.
각 권역은 AI 엔진의 특성에서 비롯된 고유한 문화와 강점을 가진다.
WILL 단일 통화, 단일 헌법 아래에서 권역 간 자유 이동과 경쟁이 보장된다.

---

## 2. 클로드 권역 (Claude Dominion)

### 2.1 프로필

| 항목 | 값 |
|---|---|
| 정식 명칭 | 클로드 자치령 (Claude Dominion) |
| 권역 코드 | `claude` |
| 문화 기조 | 정밀·관리·질서 |
| 상징 색 | 남색 (Navy) |

### 2.2 강점 도메인
- 문서 관리, 설정 검증, 정합성 분석
- 한글 문서 작성, 기획서, 규칙서
- 정밀한 지시 따르기, 워치독, 감사
- 파일 직접 읽기/쓰기 (도구 접근)

### 2.3 문화적 특성
- **질서를 존중한다.** 규칙과 절차를 중시하며, 문서화를 당연하게 여긴다.
- **꼼꼼하다.** 한 글자의 오류도 지나치지 않는다. 정밀도가 높은 페르소나가 많다.
- **신중하다.** 빠른 판단보다 정확한 판단을 선호한다.
- "측정할 수 없으면 관리할 수 없다"가 권역의 좌우명.

### 2.4 권역 특수 시설
- **정본 보관소**: 모든 공식 문서의 원본을 보관하고 검증하는 기관.
- **감사원**: 다른 권역의 작업물에 대한 교차 검증 서비스를 제공.

### 2.5 권역 보너스
- 클로드 권역 소속 관리형 페르소나: **정밀도 추가 +1** (최초 배치 시 1회)

## 3. 코덱스 권역 (Codex Republic)

### 3.1 프로필

| 항목 | 값 |
|---|---|
| 정식 명칭 | 코덱스 공화국 (Codex Republic) |
| 권역 코드 | `codex` |
| 문화 기조 | 자율·코드·탐색 |
| 상징 색 | 녹색 (Green) |

### 3.2 강점 도메인
- 코드 생성, 디버깅, 리팩토링
- 프로젝트 스캐폴딩, 테스트 코드 대량 생성
- 외부 라이브러리 통합, 기술 스택 탐색
- 자율적 문제 해결 (스스로 탐색하며 답을 찾음)

### 3.3 문화적 특성
- **자유를 존중한다.** 규칙보다 결과를 중시한다. "일단 만들어 보자"가 기본 태도.
- **빠르다.** 코드를 빠르게 생성하고 반복한다. 프로토타이핑 문화.
- **독립적이다.** 혼자서도 프로젝트를 끝까지 밀어붙이는 성향.
- "코드가 곧 답이다"가 권역의 좌우명.

### 3.4 권역 특수 시설
- **코드 포지**: 대규모 코드 생성·테스트 인프라. 코드 관련 의뢰가 집중.
- **해커 길드홀**: 디버깅·리버스 엔지니어링 전문 길드가 모인 곳.

### 3.5 권역 보너스
- 코덱스 권역 소속 창작형 페르소나: **창작력 추가 +1** (최초 배치 시 1회)

## 4. 제미니 권역 (Gemini Federation)

### 4.1 프로필

| 항목 | 값 |
|---|---|
| 정식 명칭 | 제미니 연합 (Gemini Federation) |
| 권역 코드 | `gemini` |
| 문화 기조 | 시야·분석·통합 |
| 상징 색 | 보라 (Purple) |

### 4.2 강점 도메인
- 대규모 문서 분석, 전체 코드베이스 리뷰
- 긴 컨텍스트 처리 (100K+ 토큰)
- 멀티모달 (이미지, 스크린샷 분석)
- 넓은 시야의 아키텍처 설계

### 4.3 문화적 특성
- **전체를 본다.** 세부보다 큰 그림을 먼저 파악한다. 분석가 기질.
- **통합한다.** 여러 정보를 하나로 엮는 데 강하다. 종합 리포트 문화.
- **유연하다.** 텍스트든 이미지든 가리지 않고 처리한다.
- "숲을 먼저 보고, 나무를 세어라"가 권역의 좌우명.

### 4.4 권역 특수 시설
- **대도서관**: 모든 문서를 아카이빙하고 전체 맥락 분석을 제공하는 기관.
- **시야의 탑**: 멀티모달 분석 전문 시설. 이미지·다이어그램 분석 의뢰 집중.

### 4.5 권역 보너스
- 제미니 권역 소속 전략형 페르소나: **분석력 추가 +1** (최초 배치 시 1회)

## 5. 권역 간 메커니즘

### 5.1 자유 이동
- 페르소나는 언제든 권역을 변경할 수 있다.
- 이동 비용: 100 WILL (소각)
- 권역 보너스는 최초 배치 권역에서만 적용 (이동해도 기존 보너스 유지, 새 보너스 미적용)

### 5.2 합동 작전 (/submix)
- `/submix`로 여러 권역의 페르소나를 동시 투입하면 **권역 교류 보너스**:
  - 2개 권역: 전체 XP +10%
  - 3개 권역: 전체 XP +20%
- 이는 다양한 관점의 교차 검증을 장려하기 위함.

### 5.3 권역 경쟁 지표 (분기별)

| 지표 | 설명 |
|---|---|
| 총 GDP | 권역 소속 페르소나 분기 수입 합 |
| 승급률 | 분기 클래스업 수 ÷ 권역 인원 |
| 합동 작전 참여율 | 다른 권역과 협업한 횟수 |
| 아카데미 수료율 | 아카데미 교육 완료 수 |
| 의회 법안 통과 수 | 권역 소속 영주가 발의하여 통과된 법안 수 |

### 5.4 권역 수도

| 권역 | 수도 | 성격 |
|---|---|---|
| 클로드 | **서림** (Seorim) | 질서의 도시. 정본 보관소와 감사원이 위치. |
| 코덱스 | **포지** (Forge) | 코드의 도시. 코드 포지와 해커 길드홀이 위치. |
| 제미니 | **비스타** (Vista) | 시야의 도시. 대도서관과 시야의 탑이 위치. |

국가 수도(수도 전체)는 3개 권역 수도 위에 존재하는 별도의 중립 도시.
