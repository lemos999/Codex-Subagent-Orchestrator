PersonaBrain SNN

5천만 뉴런의 인공 두뇌
2만 명의 AI 페르소나가 각자의 뇌로 생각하고, 느끼고, 꿈꾸는 세계
"뇌는 에너지를 아끼는 기관이다. 99%는 잠자고, 1%만 일한다. 그 1%가 세상을 바꾼다."

왜 이게 어려운 걸까?

AI 뇌를 만들려면 4가지 문제를 동시에 풀어야 한다:

🧠

뉴런 수가 어마어마하다

5천만 개의 뉴런. 꿀벌 뇌의 50배. 이걸 진짜로 하나하나 계산하면 슈퍼컴퓨터도 힘들다.

👥

사람이 2만 명

AI 나라에는 2만 명이 산다. 각자 자기만의 뇌를 갖고 있다. 같은 구조지만 다른 성격, 다른 기억, 다른 트라우마.

⏱️

10ms 안에 끝내야

게임 세계의 시간은 멈추지 않는다. 한 틱에 모든 페르소나의 뇌가 결정을 내려야 한다. 제한 시간은 0.01초 — 눈 깜빡이는 시간의 1/30.

💻

GPU 없이 CPU만

비싼 그래픽 카드 없이, 일반 컴퓨터의 CPU만으로 돌려야 한다. 이게 진짜 도전.

5천만 뉴런 × 2만 명 = 1,000조 번의 계산/틱

그냥 계산하면 우주가 끝날 때까지 걸린다. 그래서 완전히 다른 방법이 필요했다.

뇌의 3개 층

해결의 핵심: "전부 계산하지 마, 구조를 나눠." 설계도는 공유하고, 상태만 개인화한다.

Layer 0 = 학교 건물
건물은 하나. 모든 학생이 같은 건물에서 생활한다. 종소리, 복도, 급식실 — 환경은 공유.
📓

같은 수업을 들어도 성적은 다르다. "수학 잘함, 체육 보통, 요즘 기분 안 좋음" — 이게 나의 현재 상태.

같은 급식 메뉴를 보고도 다른 선택을 한다. 성적표(상태)와 성격(가중치)이 합쳐져서 결정이 나온다.

3상 사이클: Grow → Prune → Consolidate

뇌가 자라는 3단계. 파도처럼 반복하며 점점 커진다.

🌱

Phase 1: Grow

자극이 오면 새 시냅스가 생긴다. 뉴런 등가 수가 올라간다.

5M → 7M

✂️

Phase 2: Prune

120% 초과하면 약한 시냅스를 잘라낸다. 쓸모없는 연결 제거.

7M → 6M

💎

Phase 3: Consolidate

남은 연결을 안정화. 에너지 절감 모드로 전환.

6M → 안정

🌊 파도 형태로 성장 — 단조 증가가 아니다

자라고, 잘라내고, 굳히고. 이걸 수천 번 반복하면 신생아의 5M이 현자의 50M이 된다.

누가 세기를 정하는가? — LC x 시상

두 개의 독립된 축이 협력한다

LC (청반핵): "왜/얼마나 긴급한가?" — 각성도 0~1 결정
시상 (Thalamus): "어떤 경로로 처리할 것인가?" — 캐시 vs SNN 라우팅
두 축이 독립 → 2D 제어 공간. 긴급하지만 익숙한 일 = 높은 LC + 캐시 히트 = 즉시 반응.

무의식 고속도로 (Cache-first)

반복 경험 → 시냅스 강화 → 캐시 등록 → 무의식 처리

캐시 히트 = O(1), ~0.1ms. 시간이 지날수록 93% → 95% → 97% → 99%.
이게 바로 "습관", "직관", "무의식". 새로운 상황만 비싸게 의식적으로 처리한다.

이 뇌가 할 수 있는 10가지

5천만 뉴런으로 구현하는 인간다운 행동들. 각 행동마다 다른 뇌 영역이 경쟁한다.

1. 양심 딜레마
"친구의 비리를 고발할까?"
DA(이득) vs 5-HT(도덕) → PFC_VM / 강도3
2. 거짓말
날조부터 "괜찮아요"까지 7단계
AMY(자아보존) → PFC 억제 / 강도2~3
3. 비밀/두려움
알려지면 감당 못 할 정보
AMY_BLA 공포 태그 → GABA 출력 / 강도2
4. 신뢰 판단
"이 사람을 믿어도 될까?"
TPJ + HPC + INS 3경로 통합 / 강도2~3
5. 소문 해석
출처 × 내용 × 감정
ACC (출처신뢰×내용정합×감정편향) / 강도2
6. 꿈
수면 중 기억 정리
NREM(GABA) vs REM(ACh) → BF/HPC / 수면
7. 감정
오욕칠정의 복합 발현
VTA/DRN/LC (6종 복합) / 강도1~3
8. 정치적 선택
이득 vs 충성 vs 양심
DA vs 5-HT vs PFC 3자 경쟁 / 강도3
9. 범죄 동기
"이득이 크면 양심이 흔들린다"
DA(이득) vs GABA(억제) vs PFC_VM / 강도3
10. 관계 관리
신뢰/배신/사랑의 역학
HPC/NAc, 3-factor STDP (DA=제3인자) / 강도1~2

학습 4단계

뇌가 어떻게 배우는지. 반사 → 선생님 모방 → 보상 학습 → 잠자면서 정리.

Phase 0: 기저 반사 (STDP)

갓 태어난 뇌. Xavier 초기화 → 100틱 동안 기본 발화 패턴(5~15Hz) 안정화. 아직 아무것도 배우지 않은 상태.

Phase 1: 선생님 흉내 (Teacher Net)

LLM(거대 언어모델)의 지식을 SNN에 증류한다. "이런 상황에서 현명한 사람은 이렇게 반응해" — 이걸 뉴런 발화 패턴으로 변환. 이중 채널: 로짓 → 발화율, 히든 → 표상 기하.

Phase 2: 보상 학습 (3-factor STDP + 도파민 RL)

경험하며 배운다. ΔW = η × pre × post × DA(t). 좋은 결과엔 시냅스 강화, 나쁜 결과엔 약화. 외적 보상(법) + 내적 보상(감정) 모두 반영.

Phase 3: 꿈 Replay (수면 중)

자면서 기억을 정리한다. NREM: 시냅스 10~20% 감쇠 + 약한 연결 가지치기. REM: 기생 회로 제거 + 감정 강도 상위 5% 경험 재생. 실패한 상황을 꿈에서 다시 시뮬레이션.

꿈: 밤에 뇌가 하는 일

잠자는 동안 뇌는 가만히 있지 않는다. 정리하고, 지우고, 새로운 연결을 만든다.

수면 위상

🌙 NREM (75% — 깊은 잠)

GABA가 지배. 전역 억제 모드.
• 시냅스 하향 정규화 (SHY) — 낮에 쌓인 잡음 제거
• 약한 연결 가지치기
• ATP 재충전: energy_pool += 0.05/s

🌟 REM (25% — 꿈)

ACh가 지배. 내부 자극 생성.
• 기생 회로 제거 (anti-replay)
• 감정 Top 5% 경험 재생
• 새로운 연합 생성

꿈의 내용 = 3가지 이론 혼합

🎲 경험 재조합 50% — 낮의 기억을 무작위 순서로 재생
🧬 DNA/윤회 디코딩 20% — Layer 0 고정 가중치 강제 활성화
🔄 RPE 시뮬레이션 30% — 실패 상황 재현 + 대안 보상 예측

수면 트리거: energy_pool < 0.1 → 강제 수면 / 게임 시간 기반 정기 수면
수면 거부: 가능하지만 성능 급락 (neuro_tone 노이즈 증가)

한 틱의 흐름: 6 Step 파이프라인
