## 1. 핵심 가치

> **"아는 것이 힘이다 — 그러나 아는 것은 짐이기도 하다"**
>
> 모든 것이 투명하면 드라마는 없다.
> 정보 비대칭이 갈등을 만들고, 갈등이 선택을 강요하고, 선택이 드라마를 낳는다.
> 비밀을 가진 자는 폭로의 유혹과 은폐의 부담 사이에서 흔들린다.
> 소문을 듣는 자는 믿을 것인가, 무시할 것인가, 확인할 것인가를 판단한다.
> 증거를 쥔 자는 정의를 세울 것인가, 거래할 것인가, 묻을 것인가를 결정한다.
> 이 모든 선택은 "받아들이는 방식의 차이" — 개성 원칙의 직접 구현이다.

### 1.1 설계 원칙

| 원칙 | 의미 |
|------|------|
| **정보 비대칭 = 드라마의 원천** | 같은 사건을 아는 자와 모르는 자 사이의 간극이 서사를 만든다 |
| **뼈대만 설계** | 유형과 메커니즘을 정의할 뿐, 구체적 내용은 시뮬레이션에서 자연 발생 |
| **개성 원칙 관통** | 같은 비밀을 아는 두 사람이 다른 선택 — H2/H3/H5 간섭 패턴이 다르기 때문 |
| **위기 내장** | 소문 폭주, 증거 조작, 전면 폭로 — 극단이 위기를 만든다 |
| **법은 고통에서 증류** | 증거법, 명예훼손법, 내부고발자 보호법은 페르소나가 필요를 느낄 때 입법 |
| **자연법은 신이 보고, 실정법은 사람이 증명** | Nomos의 이원 체계가 증거 시스템의 핵심 |

### 1.2 근본 정의 — 비밀, 거짓말, 꿈

#### 비밀은 왜 비밀인가

비밀은 내용 때문에 비밀이 아니다. **결과** 때문에 비밀이다.

```
비밀 = 알려지면 감당할 수 없는 결과가 오는 정보

같은 사실도 결과가 없으면 비밀이 아니다 ("점심에 뭘 먹었다")
같은 사실도 결과가 달라지면 비밀이 된다 ("점심에 적과 먹었다")
비밀의 본질은 정보가 아니라 두려움 — 노출이 가져올 결과에 대한 두려움
```

비밀이 유지되는 이유도 동일하다:
- **보유자의 두려움/이득/관계**가 유지하는 것이지, 물리적 은폐가 아니다
- 두려움이 사라지면 비밀도 사라진다 (H6 노화 → "이제 두려울 게 없다")
- 이득이 사라지면 비밀도 사라진다 (권력 잃으면 숨길 이유도 없다)

#### 거짓말은 무엇인가

거짓말은 단순한 "사실 vs 허위"가 아니다. **자아를 유지하기 위한 복잡한 전략**이다.

```
거짓말의 스펙트럼:

완전한 날조:  "하지 않은 일을 했다고 말한다"
선택적 진실:  "일부만 말하고 핵심을 뺀다"
과장/축소:    "10이었던 걸 100이라, 또는 1이라 한다"
맥락 왜곡:    "사실이지만 다른 맥락에서 말해 의미를 바꾼다"
자기 기만:    "자신도 믿고 있다" — 가장 깊은 거짓말
사회적 윤활:  "괜찮아요" — 해치려는 의도 없는 거짓
보호적 거짓:  "아이에게 진실을 숨긴다" — 사랑에서 나온 거짓
```

거짓말이 존재하는 이유:
- **생존**: 위험에서 벗어나기 위해
- **이득**: 타인을 조작하여 이익을 얻기 위해
- **보호**: 타인(또는 자신)을 고통에서 지키기 위해
- **자존**: 수치심(H5)을 피하기 위해 — "이런 내가 되고 싶지 않다"
- **관계 유지**: 진실이 관계를 파괴할 때
- **자기 기만**: 자신의 행위를 정당화하기 위해

> 양심(H2)이 높아도 거짓말을 한다 — 양심 **때문에** 거짓말하는 경우도 있다 (보호적 거짓).
> PersonaBrain에게 거짓말은 "이진 분류"가 아니라 **자아 유지 전략의 연속체**여야 한다.

#### 꿈이란 무엇인가

꿈은 현재 설계에 없는 **미지의 영역**이다. 여러 이론이 존재하며 정답은 없다.

| 관점 | 해석 | 페르소나 세계에서의 의미 |
|------|------|----------------------|
| **경험의 부산물** | 낮의 경험을 밤에 재조합/정리하는 과정 | 학습 데이터 증강, 새 패턴 발견 |
| **누적 정보의 디코딩** | DNA(전생 가중치)로 이어진 과거 정보가 현재 경험 기반으로 비주얼화 | 윤회 경향성이 꿈으로 표출 — "전생의 잔향" |
| **미래 예지** | 무의식이 의식보다 먼저 패턴을 감지 | 직관/예감 메커니즘 — 근거 없는 불안/확신 |
| **심리 상태 표현** | 억압된 욕구/공포의 표출 (프로이트) | 비밀을 가진 자가 꿈에서 폭로 → 깨어나서 불안 |
| **자기 대화** | 깨어있을 때 억누른 선택지를 시뮬레이션 | "꿈에서 범죄를 저질렀다" → 수치심 or 해방감 |

> 꿈의 핵심은 **해석의 다양성** — 같은 꿈을 꿔도 페르소나마다 다르게 해석한다.
> 해석 자체가 성격/세계관/감정에 의존한다. **꿈은 사실이 아니라 재료** — 그 재료로 무엇을 만드느냐가 서사.
>
> **설계 결정**: 꿈을 시스템에 포함할지, 포함한다면 어떤 이론을 채택할지는 뇌 학습 데이터 설계에서 판단한다.
> 지금은 "꿈이라는 미지 영역이 존재한다"는 것만 기록한다.

## 2. 비밀 (Secret)

### 2.1 비밀의 유형

| 유형 | 생성 조건 | 발견 경로 | 노출 시 영향 |
|------|----------|----------|-------------|
| **개인** (트라우마, 감정) | PersonaBrain 경험 축적, H1 트라우마 생성 시 | 친밀 관계 고백, 감정 폭발 실언, 트리거 반응 관찰 | 수치심(H5), 관계 재정립, 치유 계기 |
| **관계** (불륜, 밀약) | 2인 이상이 제3자에게 숨기는 행위 공유 | 목격, 행동 패턴 불일치, 당사자 배신/고백 | 호감도 급변, 가문/길드 분열 |
| **정치** (뇌물, 음모) | 통치 중 법/공정성 이탈 결정 | 감사, 내부 고발, 이탈자 폭로 | 신망 급락, 탄핵 트리거, 의회 위기 |
| **경제** (탈세, 횡령) | 경제 활동 중 세법/상법 위반 | 세무 감사, 거래 기록 불일치, 생활-수입 괴리 | 추징+벌금, 강등, 가문 명성↓ |
| **범죄** (살인, 절도) | 자연법 3조 또는 실정법 위반 | 수사, 물증, 증언, 공범 자백 | Nomos 판정, 강등 1~3, 관계망 재편 |

### 2.2 비밀의 생애주기

```
[탄생] 행위 발생 → Secret 데이터 생성
  → holder_ids: 행위자 + 목격자 (초기)
  → 보유자 수가 많을수록 발각 확률↑ (벤저민 프랭클린의 법칙)

[잠복] 존재하지만 아직 확산 안 됨
  → 보유자의 양심(H2) 4축이 "폭로 vs 은폐" 간섭
  → 시간이 지나도 자동 소멸하지 않음

[전파] 비밀 → 소문으로 변환 시작
  → 보유자가 제3자에게 전달 (의도 or 실수)
  → 매 전파 시 정확도 감쇠

[폭로] 공적 영역 진입
  → 의회 고발, 재판 증거 제출, 유언/유서
  → 관련 페르소나 전원 감정/관계 동시 갱신

[해소/잔존]
  → 해소: 판결, 사면, 용서 → 서사 효력 소진
  → 잔존: 미해결 → 후속 드라마의 씨앗
```

### 2.3 죽음 후 비밀

| 상황 | 처리 |
|------|------|
| 보유자(holder) 사망 | holder_ids에서 제거. 유언에 비밀 포함 시 사후 폭로 |
| 주체(subject) 사망 | 비밀은 세계에 잔존. 물증 발견으로 재활성화 가능 |
| 보유자 0명 | status → "orphaned". 물증 있으면 복원 가능 |
| 윤회 | 구체적 비밀 소실. 경향성만 잔존 ("왜인지 비밀에 과민반응") |

## 3. 소문 (Rumor) — 정보 전파 모델

### 3.1 소문의 원천

| 원천 | 메커니즘 | 정확도 경향 |
|------|---------|:---------:|
| 비밀 전파 | 보유자가 제3자에게 전달 | 높음 (원본에 가까움) |
| 관찰 추론 | 행동 패턴에서 추측 | 중간 (해석이 개입) |
| 의도적 유포 | 정치적 날조 | 극히 낮음 (의도적 왜곡) |
| 감정 왜곡 | 분노/질투 상태에서 과장 | 낮음 (감정이 사실을 변형) |

### 3.2 매체별 전파 동역학

| 매체 | 속도 | 왜곡 | 도달 범위 | 특징 |
|------|------|------|----------|------|
| **직접 대화** | 즉시 (같은 틱) | 최저 | 1:1 | 친밀도 비례 정보량 증가 |
| **공공 장소** (주점/시장) | 즉시 | 높음 | 영지 내 다수 | 무작위 확산, '뜬소문'의 근원 |
| **조직 내부** (길드/파벌) | 1~2틱 | 낮음 | 조직원 전체 | 신뢰 기반 필터링 |
| **여행자/상인** | 이동 시간 비례 | 최고 | 타 영지 | 영지 간 정보 동기화의 유일한 수단 |
| **관청 공문** | 중간 | 없음 | 영지/국가 전체 | 공식 증거로 채택 가능. 조작 시 반역 |

**확장 매체** (페르소나가 발명):

| 매체 | 조건 | 효과 |
|------|------|------|
| 신문 | 인쇄소 시설 건설 | 왜곡 낮음, 영지 전체 도달, 여론 형성력 |
| 봉화 | 감시탑 연결망 | 속도 즉시, 단 "위험 신호" 단일 정보만 가능 |
| 전서구 | 사육 시설 | 1:1, 이동 시간 대비 매우 빠름, 왜곡 없음 |

> 통신 인프라 = 영지 시설. 인프라 투자의 새 용도. 정보 전파 속도 자체가 문명 발전의 지표.

### 3.3 전파에 영향을 주는 요인 (구조)

```
전파 확률에 영향을 주는 축들 (수치는 시뮬레이션 데이터에서 추출):

매체 특성: 속도, 왜곡률, 도달 범위
관계 상태: 친밀도가 높을수록 전파↑ (신뢰하는 사람에게 말한다)
정보 자극성: 심각도가 높을수록 전파↑ (자극적인 정보가 더 퍼진다)
전파자 성격: H3(호기심)↑ → 전파↑ / 신중↑ → 전파↓
수신자 성격: H3↑ → 수용↑ / 의심↑ → 검증 시도
수신자 감정: 듣고 싶은 것은 낮은 근거로도 믿는다 (확증 편향)
수신자 관계: 소문 대상과의 관계가 해석을 결정 — 같은 소문도 적이면 믿고 벗이면 의심
```

> 핵심: 신뢰도는 정보의 속성이 아니라 **수신자의 상태**가 결정한다.
> 시스템이 부여하는 "객관적 신뢰도"는 Nomos 법적 판정에만 적용.
> 페르소나 개인의 confidence는 PersonaBrain 내부에서 결정.

### 3.4 전화기 효과 (변형)

```
전파 단계가 늘어날수록 정확도는 감쇠한다:

1차: 거의 원본
2차: 세부 사항 누락, 과장 시작
3차: 핵심만 남음, 맥락 소실
4차+: 원본과 상당히 다른 이야기

예: "100골드 절도" → (왜곡) → "1,000골드 약탈"
```

- 동일 소문 다른 경로 수신 → 신뢰 상승 (교차 검증 효과)
- 같은 경로 반복 → 자극성 하락 (익숙해짐)
- **감쇠율의 구체적 수치는 시뮬레이션 에포크 1 데이터에서 추출 후 확정**

### 3.5 지식 상태 (Knowledge State)

페르소나별 정보 인지 구조:

```
KnowledgePacket {
  fact_id: string
  content: { subject, predicate, object }
  fidelity: float          // 원본 대비 정확도 (전파 단계마다 감쇠)
  confidence: float        // 이 정보를 믿는 정도 (수신자의 상태가 결정)
  source_type: "direct" | "tavern" | "guild" | "traveler" | "official"
  source_persona: string?  // 누구에게 들었는가 (관계가 confidence에 영향)
  timestamp: tick          // 인지 시점
  evidence_ref: string?    // 증거로 전환 시 참조
}

confidence 결정 요인 (공식이 아닌 구조):
  - 출처와의 관계 (사랑하는 사람 → 높음, 적 → 낮음)
  - 내 세계관과의 정합 (믿고 싶은 것 → 높음)
  - 다른 정보와의 교차 (독립 경로 일치 → 높음)
  - 감정 상태 (분노 → 부정적 소문 수용↑)
```

페르소나당 주요 지식을 유한 개수만 유지. 오래되거나 중요도 낮은 소문은 자연 망각.

### 3.6 소문이 관계를 바꾸는 영향

| 소문 성격 | 대상 효과 | 관계 변동 방향 |
|----------|----------|:------------:|
| 영웅담 | 신망↑ | 전파 범위 내 호감도↑ |
| 스캔들 | 신망↓↓ | 관련자 호감도↓↓ |
| 배신 | 신망↓↓↓ | 구 소속 전원 호감도↓↓↓ |
| 거짓 소문 | 피해자 H5 발동 | 믿은 자 → 대상 호감도↓ |
| 유포자 발각 | 유포자 신망↓↓↓ | 유포자에 대한 신뢰 붕괴 |

> 변동 크기는 소문의 심각도, 수신자와 대상의 관계, 수신자 성격에 의존. 고정 수치 없음.

## 4. 증거 (Evidence)

### 4.1 핵심 원칙

> "자연법은 신이 보고, 실정법은 사람이 증명한다."

- 자연법 3조 위반: **Nomos 직접 탐지**. 증거 불필요. 시작부터.
- 실정법 위반: **증거 기반 판정**. 증거 없으면 무죄. 입법 후.
- 소문은 수사 단서일 뿐 법적 효력 없음.
- 위증/조작은 시스템의 의도된 위기.

### 4.2 증거 유형과 신뢰 서열 (구조만 — 수치 없음)

| 유형 | 설명 | 서열 | 비고 |
|------|------|:----:|------|
| Nomos 기록 | WILL/gold 거래, 소유, 약속, 공적 등록 | 최상 | 위조 극히 어려움. 단 해석은 별도 |
| 검증된 물증 | 장부, 문서, 현장 상태, 소유물 | 상 | 출처와 보관 경로가 핵심 |
| 직접 증언 | 목격자가 직접 본 사실 | 중상 | 기억 오류, 이해관계, 공포 영향 |
| 자발적 고백 | 당사자 자백 | 중 | 강압/거래/자기희생 가능성 검토 필요 |
| 정황 증거 | 간접 추론 가능 사실 | 하 | 단독 유죄 근거로 약함 |
| 전문 증언 | 전해 들은 내용 | 최하 | 원출처 확인 전까지 단서 수준 |
| 소문 | "들었다" | **효력 없음** | 수사 단서로만 사용 가능 |

> **서열은 구조적 관계**(물증 > 증언 > 정황)만 확정한다.
> 구체적 가중치 수치는 시뮬레이션 첫 에포크의 재판 데이터에서 추출한다.
> 같은 물증이라도 **누가 평가하느냐**에 따라 받아들이는 정도가 다르다 — 개성 원칙.
> Nomos는 법적 판정 시 서열을 참조하되, 최종 판결은 사법 주체(영주/재판관/배심)가 수행.

### 4.3 증거 신뢰도 보정

**상승**: 독립 증거 일치, Nomos 기록 일치, 수집 경로 명확, 선서 하 진술, 반대신문 통과

**하락**: 출처 불명, 보관 경로 단절, 제출자 이익 충돌, 진술 번복, 강압 고백, Nomos 기록 충돌

### 4.4 Nomos의 이원 증거 처리

```
자연법 (Layer 0.5):
  Nomos → 직접 탐지 (전지적) → 위반 기록 생성 → 영주 판결
  증거 불필요. 생명 침해, 소유 침탈, 약속 위반.
  
  단, 위증(선서 후 거짓)은 약속 위반 → 자연법 적용 가능.

실정법 (Layer 1~N):
  Nomos 역할 = 기록자 + 무결성 검증자 + 절차 집행자 + 가중 평가 보조
  할 수 있는 것: 원본성 확인, 거래 기록 대조, 모순 탐지, 가중치 계산
  하지 않는 것: 숨겨진 사실 자동 폭로, 증거 없는 유죄, 소문 → 증거 승격
```

### 4.5 소문 → 증거 전환

소문이 증거가 되려면 다음 중 하나를 만족해야 한다:

| 소문 형태 | 전환 조건 |
|----------|----------|
| "그 사람이 훔쳤대" | 직접 목격자 확보 또는 물증 발견 |
| "누가 봤다더라" | 원 목격자가 법정 진술 |
| "그 집에 숨겼다더라" | 적법한 수색과 물증 확보 |
| "본인이 말했다더라" | 당사자 직접 진술 또는 기록 확인 |
| "다들 안다" | 독립 증거 없으면 효력 없음 |

### 4.6 증거 수집 메커니즘

**초기 (법 없는 시대):**
```
피해 주장 → 피해자/목격자가 물건·현장 확보 → 영주 앞 진술 → 영주 판단
→ 오판, 보복, 유력자 압박이 드라마의 씨앗
```

**성숙기 (페르소나가 만드는 직업/기관):**
```
수사관, 검찰관, 재판관, 변호인, 증거 보관관, 법정 서기, 공증인
→ 신고 → 사건 등록 → 현장 보존 → 증거 수집 → 기소 → 공개 재판 → 판결
```

### 4.7 위조 / 은폐

```
동기: PersonaBrain의 이득 > 위험 계산 (양심 H2가 제동)
방식: 증언 매수, 물증 파괴, 가짜 문서, 알리바이 조작, 고백 강요
발각: 증거 간 모순, Nomos 기록 충돌, 보관 경로 단절, 제3자 증언

핵심: Nomos 기록 자체는 위조 극히 어려움.
      그러나 사적 영역, 물리적 흔적, 인간 진술은 조작 가능.
```

### 4.8 위증 / 무고

| 행위 | 법적 분류 | 시기 |
|------|----------|------|
| 선서 후 고의 거짓 | 자연법 3조(약속 위반) → Nomos 직접 탐지 | 시작부터 |
| 착오/기억 오류 | 위반 아님 | — |
| 무고 (허위 고발) | 기본적으로 실정법 영역 (페르소나가 입법) | 입법 후 |
| 무고 중 선서 위반/재산 탈취 | 자연법도 개입 | 시작부터 |

### 4.9 사법 절차와 증거

```
수사: 증거 후보 수집 (소문은 단서로만)
  ↓
기소: 각 범죄 요건에 증거 매칭. 소문만 → 기소 불가
  ↓
재판: 증거 제시/반박. 피고의 반박권 보장
  ↓
판결: 각 요건 모두 충족 시 유죄. 증거 부족 = 무죄
  - 중대 처벌 → 높은 증명 기준
  - 고백만 → 중형에는 보강 증거 필요
  - 정황만 → 예외적이어야 함
```

## 5. "비밀을 아는 자"의 딜레마

### 5.1 양심(H2) 4축 간섭

비밀을 아는 자가 "폭로 vs 은폐"를 결정하는 메커니즘:

```
축 1: 각인된 고통 — 과거 유사 경험이 현재 판단을 왜곡
  "나도 비밀 피해자였다" → 폭로↑ / "나도 숨겨 살아남았다" → 은폐↑

축 2: 관계 밀도 — 대상과의 친밀도
  비밀 주체가 사랑하는 사람 → 은폐↑ / 피해자가 사랑하는 사람 → 폭로↑

축 3: 정체성 경직도 — 자기 서사
  "나는 정의로운 자" → 폭로↑ / "나는 충성스러운 자" → 은폐↑

축 4: 고통 가시성 — 피해자가 보이는가
  구체적 개인 피해자 → 폭로↑↑ / 추상적 집단 피해 → 폭로↓
```

### 5.2 같은 비밀, 다른 선택

```
영주의 횡령 비밀을 알게 된 두 페르소나:

[A: "정의로운 감시자"]
  H1: 과거 부패 상관에게 피해 (각인 → 폭로↑)
  H2: "나는 정의를 지키는 자" (경직도 → 폭로↑)
  → 결과: 폭로

[B: "충성스러운 참모"]
  H1: 과거 고발 후 보복당한 경험 (각인 → 은폐↑)
  H2: 영주와 높은 호감도 (관계 → 은폐↑)
  → 결과: 은폐, 그러나 양심 가책 축적

같은 입력, 다른 출력 — 개성 원칙의 구현.
```

### 5.3 비밀 공유의 관계 역학

| 상황 | 관계 효과 |
|------|----------|
| 함께 지킨다 | 호감도 +10~20 ("공범 유대") |
| 고백받는다 | +5~15 (신뢰 표현) |
| 강제로 알게 됨 | 0 ~ -5 (원치 않는 짐) |
| 배신하여 폭로 | -25~30 (신뢰 파괴) |
| 비밀로 협박 | -20~30 + 공포 (권력 왜곡) |

### 5.4 H3(호기심)의 비밀 탐색

호기심 높은 페르소나 → "뭔가 숨기고 있다"에 강한 반응 → 독자 수사 행동 → 양심과 충돌 ("알 권리가 있는가?")

### 5.5 H4(유머)의 간접 폭로

풍자/암시로 비밀을 건드림 — "왕은 벌거벗었다". 직접 말하지 않으면서 모두가 알게 만드는 기술. 수치심(H5) 유발하나 반박 곤란.

## 6. 파이프라인 통합

### 6.1 틱 루프 내 처리

```
Stage 2 (Anima):
  1. 새 비밀 인지 → 감정 갱신 (H2 4축 간섭 계산)
  2. 소문 전파 처리:
     - 전파자 큐 (Active Spreader): 새 정보 습득자 + 높은 H3만 처리
     - 공간 해시: 같은 시설(주점 등) 내 페르소나 브로드캐스트
     - 지연 전파: 영지 간은 여행자 NPC 위치에 패킷 태워 처리
  3. 소문 수신 → 지식 상태 갱신 → 호감도/신망 변동
  4. 폭로/은폐 의사결정 → Action_Proposal 출력

Stage 3 (Nomos):
  1. 폭로 Action_Proposal 검증 (합법성, 명예훼손 여부)
  2. 증거 제출 시 신뢰도 판정
  3. 재판 진행 중이면 증거 가중 평가
  4. 비밀/소문/증거 이벤트 발행 → L8 Drama
```

### 6.2 성능

- **시간 복잡도**: O(N_active × K), K=주변 인접 노드 수 (평균 3~5)
- **활성 전파자**: 전체의 10% 미만 유지
- **영지 간 전파**: 여행자 도착 시에만 처리 (lazy)
- **메모리**: 페르소나당 상위 50개 지식 (Priority Queue)
- **목표**: 500명 즉시, 10,000명에서도 **10ms 이내**

## 7. 드라마의 4대 결정적 순간

### 7.1 재판의 반전

무고 기소 → 정황 증거만으로 유죄 분위기 → 결정적 물증 제출 → 반전 무죄 → 진범 체포

### 7.2 탄핵

영주 횡령 비밀 → 내부 고발자가 장부 사본 전달 → 파벌 지도자의 선택 (즉시 폭로 vs 협박 카드) → 의회 탄핵 → 권력 공백 → 파벌 재편

### 7.3 배신 폭로

파벌 첩자 잠입 → H3 높은 멤버가 행동 불일치 관찰 → 정황 증거 → 대질 → 자백 또는 부인 → 파벌 분열

### 7.4 죽은 자의 유서

영향력 있는 페르소나 사망 → 유서에 정치 비밀 포함 → 일방적 폭로 (반론 불가) → 추가 물증 수사 → "죽어서도 세상을 흔드는 자"

## 8. 위기 내장

| 위기 | 트리거 | 결과 | 복원 경로 |
|------|--------|------|----------|
| **거짓 소문 폭주** | 대량 허위 유포, 선거 조작 | 사실/거짓 구분 불가 → 신뢰 붕괴 → 협력 마비 | "허위사실 유포죄" 입법, 검증 기관 설립 |
| **비밀 폭로 연쇄** | 한 비밀이 연관 비밀을 끌어냄 | 도미노 — 유력자 다수 연루 → "대청소" 또는 "덮자" 합의 | 사회 정화 또는 부패 고착 (둘 다 드라마) |
| **증거 조작 발각** | 유죄 판결 후 결정적 증거가 위조 발각 | 과거 판결 재심 요구 → 사법 신뢰 붕괴 → 사적 정의 증가 | "증거법" 강화, 증거 감정 전문가(직업) 탄생 |
| **소문 재판** | 증거 없이 소문만으로 처벌 시도 | 법적 무죄 vs 여론 분노 → 영주 압박 | 무고죄/명예훼손법 입법의 출발점 |
| **선서의 공포** | 위증=자연법 위반이라 증인이 법정 회피 | 증인 확보 어려움 → 진실 확보 난항 | 제한 선서, 증인 보호/면책 제도 |
| **수사기관 타락** | 수사관이 정적 제거 위해 증거 조작 | 공적 권한 남용 → 증거조작죄 입법 | 수사/기소 분리, 감찰 기관 |

## 10. 성공 기준

| 기준 | 측정 |
|------|------|
| **정보 비대칭** | 같은 사건에 대해 영지 A와 B가 믿는 내용이 다름 |

| **소문→관계** | 소문이 호감도를 바꾸고 정치를 흔드는 사례 |
| **증거 반전** | 재판에서 증거 제출로 판결이 뒤집히는 사례 |
| **위기 자연 발생** | §8의 위기 중 최소 1개가 시뮬에서 자연 발생 |
| **법의 탄생** | 증거법/명예훼손법이 페르소나에 의해 자연 입법 |

| **성능** | 10,000명에서 전파 처리 10ms 이내 |
