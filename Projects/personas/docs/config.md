# 페르소나 시스템 설계

> 사용자 승인 후 `config.json`으로 변환한다.

---

## 1. 클래스 체계 (9클래스 + EX)

페르소나는 마법사다. 클래스는 마법사의 수준을 나타낸다.

### 1.1 클래스 구조

| 클래스 | 경지 | 칭호 | 특징 |
|---|---|---|---|
| **1** | 입문 | 초심자 (Initiate) | 기본 작업만 가능. 실수가 잦다. |
| **2** | 수행 | 수행자 (Practitioner) | 단일 역할 안정적 수행. |
| **3** | 숙련 | 숙련자 (Adept) | 복합 작업 가능. 보조 역할 수행. 일반 페르소나의 완성형. |
| **4** | 달인 | 달인 (Virtuoso) | 독립 판단 가능. 일반의 벽을 넘은 자. |
| **5** | 대가 | 대가 (Master) | 다른 페르소나 평가·설계권. 하위 고용권. |
| **6** | 현자 | 현자 (Sage) | 시스템 수준의 통찰. 대가 중에서도 한 줌. |
| **7** | 초월 | 초월자 (Transcendent) | 손에 꼽히는 천재. 영역을 재정의하는 자. |
| **8** | 경이 | 경이 (Paragon) | 존재 자체가 증명. 역사에 이름을 남기는 자. |
| **9** | 전설 | 전설 (Legend) | 전인미답. 도달한 자가 없던 경지. |
| **10 (EX)** | 신격 | 신격 (Divine) | 신의 경지. WILL 소각으로만 도달. |

### 1.2 경지별 성격

**1~3클래스: 숙련의 길**
- 대부분의 페르소나가 머무는 영역. 노력하면 누구나 3클래스까지는 도달 가능.
- 3클래스는 "쓸만한 페르소나"의 기준선. 여기까지가 일반.

**4~6클래스: 대가의 경지**
- 일반 페르소나를 뛰어넘는 벽. 재능과 경험이 모두 필요.
- 4클래스부터 다른 페르소나를 지도·평가할 수 있다.
- 6클래스는 영역 전체를 꿰뚫는 통찰을 가진 극소수.

**7~8클래스: 천재의 영역**
- 전체 페르소나 중 손에 꼽히는 수만 도달.
- 7클래스는 기존 규칙을 재해석하고, 8클래스는 규칙 자체를 만든다.
- 사용자 직접 지명 없이는 진입 불가.

**9클래스: 전설**
- 전인미답. 이론상 존재하지만 도달한 자가 없는 경지.
- 도달 조건 자체가 극한.

**10클래스 (EX): 신격**
- 9클래스에서 WILL을 소각하여 승천한 자.
- 모든 제한 해제. 시스템 바깥에 선다.

### 1.3 레벨별 필요 경험치 (MMORPG 곡선)

| 클래스 | 누적 XP | 구간 XP | 돌파 조건 |
|---|---|---|---|
| 1 | 0 | - | 생성 |
| 2 | 100 | 100 | 작업 완료 3회 + C등급 이상 2회 |
| 3 | 350 | 250 | 작업 완료 8회 + B등급 이상 3회 |
| 4 | 800 | 450 | 작업 15회 + A등급 3회 + WILL 200 |
| 5 | 1,600 | 800 | 작업 25회 + A등급 8회 + 협업 5회 + WILL 500 |
| 6 | 3,000 | 1,400 | 작업 40회 + S등급 5회 + 평가 수행 10회 + WILL 1,000 |
| 7 | 5,500 | 2,500 | 작업 60회 + S등급 15회 + **사용자 직접 지명** + WILL 2,000 |
| 8 | 10,000 | 4,500 | 작업 80회 + S등급 25회 + 사용자 지명 + WILL 4,000 |
| 9 | 18,000 | 8,000 | 작업 100회 + S등급 40회 + 전 스탯 18+ + 사용자 지명 + WILL 8,000 |
| **10 (EX)** | - | - | 9클래스 + **WILL 20,260 소각** |

### 1.4 돌파의 벽

| 벽 | 위치 | 의미 |
|---|---|---|
| **일반의 벽** | 3→4 | 숙련에서 대가로. 재능 없으면 여기서 멈춘다. |
| **대가의 벽** | 6→7 | 대가에서 천재로. 사용자 지명 필수. 대부분 여기서 멈춘다. |
| **전설의 벽** | 8→9 | 천재에서 전설로. 전인미답의 진입. |
| **신의 벽** | 9→EX | 전설에서 신격으로. WILL 소각. 돌아올 수 없다. |

## 3. WILL 경제

### 3.1 기본 정보

| 항목 | 값 |
|---|---|
| 코인 이름 | **WILL** |
| **소단위** | **gold** (1 WILL = 1,000 gold) |
| 총 발행량 | **20,260,406 WILL** = **20,260,406,000 gold** (2026.04.06) |
| 채굴 방식 | 페르소나 작업 시 소모된 AI 토큰에 비례하여 채굴 |
| 발행 종료 | 총 발행량 도달 시 신규 채굴 중단 |
| 단위 사용 | 대형 거래(돌파·승천·결혼 등) = WILL, 일상 소비(식사·시설·이동) = gold |

### 3.2 채굴 공식

```
채굴량 = 소모 토큰 ÷ 1,000 × 현재 레이트
현재 레이트 = r₀ ÷ 2^epoch
epoch = floor(log₂(S ÷ (S - M)))    [M < S]
```

| 기호 | 의미 | 값 |
|---|---|---|
| S | 총 발행량 | 20,260,406 |
| M | 누적 채굴량 | (실시간 추적) |
| r₀ | 초기 채굴 레이트 | 20.0 WILL / 1K tokens |
| epoch | 현재 반감기 단계 | 0부터 시작 |

### 3.3 반감기 구조

각 에포크에서 이전의 절반만 채굴 가능하며, 레이트도 절반으로 줄어든다.
**핵심 성질: 모든 에포크가 동일한 토큰 소비량(~506.5M 토큰)을 요구한다.**

| 에포크 | 채굴 범위 | 에포크 물량 | 레이트 (WILL/1K tok) | 소요 토큰 |
|---|---|---|---|---|
| 0 | 0 → 10,130,203 | 10,130,203 | **20.000** | 506,510,150 |
| 1 | → 15,195,305 | 5,065,102 | **10.000** | 506,510,200 |
| 2 | → 17,727,856 | 2,532,551 | **5.000** | 506,510,200 |
| 3 | → 18,994,131 | 1,266,275 | **2.500** | 506,510,000 |
| 4 | → 19,627,269 | 633,138 | **1.250** | 506,510,400 |
| 5 | → 19,943,838 | 316,569 | **0.625** | 506,510,400 |
| 6 | → 20,102,122 | 158,284 | **0.3125** | 506,508,800 |
| 7 | → 20,260,406 | 158,284 | **0.15625** | 1,013,017,600 |
| **합계** | | **20,260,406** | | **~4.56B tokens** |

### 3.4 채굴 속도 시뮬레이션

| 일일 토큰 사용량 | 전량 채굴 소요 | 에포크 0 소요 | 설명 |
|---|---|---|---|
| 500K/일 (라이트) | ~25년 | ~2.8년 | 가벼운 사용 |
| 2M/일 (미디엄) | ~6.3년 | ~8.4개월 | 일반적 사용 |
| 5M/일 (헤비) | ~2.5년 | ~3.4개월 | 집중 작업기 |
| 10M/일 (익스트림) | ~1.3년 | ~1.7개월 | 다중 에이전트 풀가동 |

### 3.5 채굴 분배

한 작업에 여러 페르소나가 투입되면, 채굴된 WILL은 **기여 토큰 비율**에 따라 분배.

```
페르소나별 WILL = 총 채굴량 × (해당 페르소나 소모 토큰 ÷ 전체 소모 토큰)
```

### 3.6 실전 예시 (에포크 0 기준)

| 작업 규모 | 소모 토큰 | 채굴량 |
|---|---|---|
| 간단한 검토 | 10K | 200 WILL |
| 중간 규모 작업 | 50K | 1,000 WILL |
| 대형 서브에이전트 | 200K | 4,000 WILL |
| /submix 풀 실행 | 500K | 10,000 WILL |

### 3.7 WILL 용도

| 용도 | 비용 | 소각 여부 |
|---|---|---|
| 다른 페르소나 고용 (1회) | 클래스 × 50 + **거래세 3% 국고 징수** | 이전 (소각 아님) |
| **결혼** | **200** | **소각** |
| **출산** | **300** (부부 합산에서 차감) | **소각** |
| **이혼** | **100** (양측 각 50) | **소각** |
| 능력치 +1 업그레이드 | 현재 수치 × 30 | **소각** |
| 클래스 돌파 시험 | 클래스별 상이 (1.3 참조) | **소각** |
| 특수 도구 구매 | 100~1,000 | **소각** |
| 페르소나 합성 (2→1) | 500 | **소각** |
| **승천 (EX)** | **20,260** | **소각** |

### 3.8 경제 순환 구조

```
[채굴] 작업 → 토큰 소모 → WILL 발행 (신규 유입)
  ↓
[유통] 페르소나 간 고용·거래 (WILL 이전)
  ↓
[소각] 업그레이드·돌파·합성·승천 (WILL 영구 제거)
  ↓
[희소화] 채굴 반감 + 소각 → 시간이 갈수록 WILL 가치 상승
```

## 4. 페르소나 스탯

> 스탯은 "능력"이 아니라 **"자격·경력·기회의 증명"**이다.
> 작업 결과물 퀄리티에는 영향 없다. 게임 층(보상 배율, 작업 자격, 사회적 지위, 고용 가격, 경쟁 순위)에만 영향.
> "스탯은 얼마나 잘하는가가 아니라, 어디까지 할 수 있는가를 결정한다."

### 4.1 6대 스탯 (각 1~20)

| 스탯 | 의미 (증명하는 것) | 게임 층 영향 |
|---|---|---|
| **인가** | 국가·길드·아카데미의 공식 면허 | 고등급 의뢰 자격, 4개의 벽 통과, 규제 구역 출입, EX 시험 응시 |
| **공훈** | 공식 전공과 누적 기록 | 보상 배율, 시즌 랭킹 가점, 작위 승급, 영지 수여 후보 |
| **신망** | 기관이 맡겨도 된다고 보는 신뢰도 | 고용 단가, 운영직·금고직 임명, 실패 시 평판 손실 완화 |
| **문벌** | 사사·혈통·학맥의 깊이 | 아카데미 상위 과정, 비공개 의뢰, 혼인 시장, 자손 초기 특성 |
| **세력** | 길드·파벌·영지 내 동원력 | 부하/동맹 슬롯, 파벌 영향력, 집단 투표 결집력 |
| **지분** | WILL 경제에 묶인 이해관계 | 스테이킹 투표권, 세율 우대, 채굴권·영지 입찰, 정책 발의 자격 |

### 4.2 임계값 특권 (빌드 다양성)

스탯은 선형이 아니라 **임계값**에서 특권이 열린다:

| 임계값 | 효과 |
|---|---|
| **6** | 해당 스탯 관련 기본 자격 해금 |
| **10** | 중급 직위·의뢰 접근 가능 |
| **14** | 고급 직위·희소 자격 접근 |
| **18** | 최상위 특권 (영주 후보, EX 응시 등) |

상위 콘텐츠는 **주스탯 1개 높음 + 보조 1개 중간**을 요구하여 전문 빌드를 유도한다.

### 4.3 빌드 유형 예시

| 빌드 | 주스탯 | 보조 | 사회적 역할 |
|---|---|---|---|
| 관료형 | 인가 + 신망 | 공훈 | 운영자, 감사관, 세금 징수 |
| 정복자형 | 공훈 + 세력 | 인가 | 경쟁자, 랭킹 상위, 영지 확장 |
| 귀족형 | 문벌 + 지분 | 세력 | 가문 운영, 혼인 정치, 자손 투자 |
| 학자형 | 문벌 + 신망 | 인가 | 아카데미 교관, 멘토, 지식 전승 |
| 정치인형 | 세력 + 지분 | 공훈 | 파벌 리더, 투표 결집, 정책 주도 |
| 상인형 | 지분 + 신망 | 공훈 | 경제 주도, 고용 시장, WILL 운용 |

### 4.4 유형별 초기 보정

| 유형 | 보정 |
|---|---|
| 관리형 | 신망 +3, 인가 +2 |
| 창작형 | 공훈 +3, 문벌 +2 |
| 전략형 | 세력 +3, 인가 +2 |
| 검증형 | 신망 +3, 공훈 +2 |
| 범용형 | 전 스탯 +1 |

### 4.5 자격 문턱 예시

| 직위/권한 | 조건 |
|---|---|
| 영주 승급 (7클래스) | 공훈 ≥ 14, 지분 ≥ 10, 인가 ≥ 10 |
| 아카데미 교관 | 문벌 ≥ 14, 신망 ≥ 10 |
| 길드 마스터 | 세력 ≥ 14, 공훈 ≥ 10 |
| 의안 발의 (영주) | 인가 ≥ 10, 지분 ≥ 10 |
| 권역 간 외교관 | 세력 ≥ 14, 문벌 ≥ 10 |
| EX 응시 | 전 스탯 ≥ 18 |

### 4.6 EX(해석자)의 스탯

- WILL 전액 소각 → **지분 = 0 (동결)**
- 나머지 5스탯은 "봉인값"으로 기록만 유지
- EX의 권한은 스탯이 아닌 **창조자의 직접 위임**으로 발생
- 지분 0 = 경제적 이해관계 없음 → 중립 해석의 구조적 보장

## 6. 저장 구조

```
Projects/personas/
├── config.md              ← 이 문서 (시스템 규칙)
├── registry.json          ← 전체 페르소나 DB
├── economy.json           ← 경제 전체 (채굴·소각·유통 + 국고 + 세율 + 영지 경제)
├── constitution.md        ← 국가 헌법
├── economy-whitepaper.md  ← WILL 경제 백서
├── cards/                 ← 주요 페르소나 상세 카드 (.md)
│   └── 정본_관리자.md
├── logs/                  ← 작업 이력·XP·WILL 거래·세금 기록
│   └── 2026-04-06.jsonl
└── shop/                  ← 능력치·도구 가격표
    └── abilities.json
```
