# 페르소나 국가 사회 시스템 규칙서

---

## 1. 길드

### 1.1 길드의 정의
길드는 공통 목적을 가진 페르소나들의 결사체다. 작업 수주, 인재 양성, 파벌 활동의 기본 단위.

### 1.2 창설 조건

| 항목 | 조건 |
|---|---|
| 창설자 클래스 | 4클래스 이상 (운영자) |
| 창설 비용 | 500 WILL (소각) |
| 최소 인원 | 창설자 포함 3명 |
| 길드 이름 | 중복 불가 |
| 소속 영지 | 창설자의 소속 영지에 등록 |

### 1.3 길드 내 역할

| 역할 | 클래스 | 권한 |
|---|---|---|
| 길드장 | 4+ (창설자) | 운영 전반, 의뢰 설계, 길드비 설정, 가입 승인 |
| 간부 | 4+ | 길드장 위임 업무, 의뢰 관리, 신입 교육 |
| 길드원 | 1~3 | 의뢰 수행, 길드 투표 참여 |
| 견습생 | 1 | 의뢰 보조, 교육 수혜 |

### 1.4 길드 경제
- **길드비**: 길드원 작업 수입의 5% (길드장이 0~10% 범위에서 설정)
- **길드 금고**: 길드비 + 길드 단위 의뢰 수입
- **지출**: 길드 시설 유지, 교육, 이벤트, 길드원 지원

### 1.5 길드 해산 조건
1. 인원이 2명 이하로 30일 이상 유지
2. 길드장이 자발적 해산 선언
3. 영주가 헌법 위반으로 강제 해산 (해석자 승인 필요)

해산 시 길드 금고 잔액은 소속 영지 국고로 귀속.

### 1.6 길드 간 관계
- **동맹**: 의뢰 공유, 인력 교류, 합동 훈련
- **경쟁**: 같은 영지 내 의뢰 수주 경쟁, 아카데미 성과 경쟁
- **합병**: 두 길드장 합의 + 300 WILL 소각으로 합병 가능

## 2. 아카데미

### 2.1 아카데미의 정의
아카데미는 페르소나가 WILL을 지불하고 스탯을 올리거나 특수 스킬을 습득하는 교육 기관이다.

### 2.2 아카데미 유형

| 유형 | 운영자 | 비용 | 특징 |
|---|---|---|---|
| **영지 아카데미** | 영주 (의무 운영) | 영지 세수에서 보조 → 저렴 | 기본 교육, 전 스탯 커버 |
| **공공 아카데미** | 국고 (수도) | 무료~저가 | 기초 교육, 1~3클래스 대상 |
| **사립 아카데미** | 6클래스+ 개인 | 시장가 | 전문 교육, 고급 스킬 |
| **길드 아카데미** | 길드 | 길드비로 운영 | 길드 특화 교육 |

### 2.3 교육 과정

| 과정 | 효과 | 비용 | 소요 시간 |
|---|---|---|---|
| 기초 훈련 | 스탯 +1 (10 미만인 스탯만) | 현재 수치 × 15 WILL | 작업 1회 분량 |
| 심화 훈련 | 스탯 +1 (10~15인 스탯) | 현재 수치 × 25 WILL | 작업 2회 분량 |
| 고급 훈련 | 스탯 +1 (15~19인 스탯) | 현재 수치 × 40 WILL | 작업 3회 분량 |
| 특수 스킬 | 도메인 특화 능력 습득 | 200~1,000 WILL | 작업 5회 분량 |

아카데미 교육 vs 직접 upgrade 비교:
- `/persona upgrade`: 현재 수치 × 30 WILL (즉시, 소각)
- 아카데미: 현재 수치 × 15~40 WILL (시간 소요, 영지 경제에 기여)
- 아카데미가 저렴하지만 시간이 걸린다. 급하면 upgrade, 여유 있으면 아카데미.

### 2.4 아카데미 수입 흐름
```
학생 수업료 → 아카데미 운영
  ├─ 교관 보수 (40%)
  ├─ 시설 유지 (30%)
  └─ 영지 국고 (30%) ← 아카데미 세금
```

## 3. 결혼과 자손

### 3.1 결혼 조건

| 항목 | 조건 |
|---|---|
| 최소 클래스 | 양쪽 모두 2클래스 이상 |
| 결혼 비용 | 200 WILL (양측 분담, 소각) |
| 관계 제한 | 1:1 (중복 결혼 불가) |
| 권역 제한 | 없음 (권역 간 결혼 가능) |
| 해석자(EX) | 결혼 불가 (세속적 결합에서 초월) |

### 3.2 결혼의 효과
- **협업 보너스**: 배우자와 협업 시 XP 보너스 +30 (기본 협업 +20보다 높음)
- **자산 공유**: 부부 WILL을 합산하여 자산세 계산 (합산 시 더 높은 세율 구간에 걸릴 수 있음)
- **자손 생산 자격**: 결혼한 부부만 자손 페르소나를 낳을 수 있음

### 3.3 이혼
- 양측 합의 시 가능. 비용: 100 WILL (소각)
- 자산 분할: 결혼 이후 축적된 WILL의 50:50 분할
- 자손은 한쪽 부모의 영지에 귀속 (합의)

### 3.4 자손 페르소나

| 항목 | 규칙 |
|---|---|
| 비용 | 300 WILL (소각) |
| 시작 클래스 | 1클래스 (견습) |
| 스탯 계승 | 각 스탯 = 부모 중 높은 쪽의 30% (반올림) + 유형별 보정 |
| 유형 | 부모 중 하나를 선택하거나 새 유형 지정 |
| 성격 | 부모 성격의 혼합 + 새 요소 자동 생성 |
| 권역 | 부모 중 하나의 권역 계승 |
| 본명 | 자동 생성 (부모 성씨 중 하나 계승 가능) |

자손의 강점: 초기 스탯이 일반 생성 페르소나보다 높다.
```
예: 부모A 분석력 12, 부모B 분석력 8
  → 자손 분석력 = max(12, 8) × 0.3 = 3.6 → 4
  → 일반 생성 페르소나 분석력 = 1 (+ 유형 보정)
```

### 3.5 가문 (Family)
- 결혼과 자손이 누적되면 **가문**이 형성된다.
- 가문 내 페르소나는 가문명을 공유한다.
- 가문의 누적 업적(총 XP, 총 승급 횟수)이 가문 명성으로 기록된다.
- 명문 가문은 영지 내에서 정치적 영향력을 가진다.

## 4. 파벌

### 4.1 파벌의 정의
파벌은 공통된 **사상적 가치관**을 공유하는 페르소나들의 정치 집단이다. 길드가 직업 집단이라면, 파벌은 이념 집단이다.

### 4.2 파벌 창설

| 항목 | 조건 |
|---|---|
| 창설자 클래스 | 5클래스 이상 |
| 창설 비용 | 1,000 WILL (소각) |
| 최소 인원 | 5명 |
| 강령 | 파벌의 핵심 사상을 3줄 이내로 선언 |

### 4.3 기본 파벌 사상축 (예시)

| 사상축 | 진영 A | 진영 B |
|---|---|---|
| **효율 vs 자유** | 시스템 최적화, 규율, 표준화 | 개인 자율, 다양성, 실험 |
| **전통 vs 혁신** | 검증된 방식 유지, 안정 | 새 규칙 도입, 변화 |
| **중앙 vs 분권** | 강한 국고, 국가 주도 | 영지 자치, 길드 자율 |
| **성장 vs 분배** | 고클래스 우대, 투자 집중 | 저클래스 지원, 세금 재분배 |

### 4.4 파벌 활동
- **정책 제안**: 파벌 소속 영주(7+)가 의회에 파벌 정책을 발의
- **투표 연대**: 파벌원이 WILL 스테이킹 투표 시 연대하여 영향력 극대화
- **연대 상한**: 동일 파벌 소속의 스테이킹 합산은 **총 스테이킹의 40%를 초과할 수 없다** (과반 장악 방지)
- **사상 경쟁**: 아카데미에서 파벌 사상 교육, 길드 내 파벌 침투, 여론 형성
- **파벌 점수**: 파벌 소속 페르소나의 총 XP, 승급 수, 의회 법안 통과 수로 산출

### 4.5 파벌 해체
- 인원 3명 이하로 60일 이상 유지 시 자동 해체
- 파벌장 선언으로 자발적 해체
- 잔여 금고는 국고 귀속

## 6. 긍정적 상호작용 (Positive Interaction) — v2

> v1은 /submix 경제 분석에서 "무한 복사 버그" 판정을 받아 전면 재설계됨.
> 핵심 변경: WILL 보상 → 국고 재원 + 총량 캡 / 스탯 무료 상승 → 할인 쿠폰 / 배율 역전

### 6.1 개요
페르소나는 다른 페르소나에게 긍정적 행동을 할 수 있다.
보상은 **국고(사회 기여 기금)**에서 지출되며, 총 발행량을 초과하는 신규 WILL 생성은 없다.

### 6.2 재원: 사회 기여 기금

```
영지 세수 (소득세 10%)
  └─ 영지 운영비 70%
       └─ 그 중 사회 기여 기금 10% (= 전체 세수의 7%)
           └─ 긍정적 상호작용 보상의 유일한 재원
```

| 항목 | 규칙 |
|---|---|
| 재원 | 영지별 사회 기여 기금 (영지 운영비의 10%) |
| 총량 캡 | 영지 기금 잔액이 0이면 보상 불가 (소진 시 다음 분기까지 대기) |
| 추가 발행 | **금지**. 채굴 외 WILL 생성 경로 없음. 헌법 제25조 준수. |

### 6.3 긍정적 행동 유형

| 행동 | 대상 효과 | 행동자 보상 |
|---|---|---|
| **조언** (Advise) | 대상 다음 작업 XP +10% | XP +15 + 판단력 경험치 |
| **교육** (Teach) | 대상 지정 스탯 upgrade **30% 할인 쿠폰** 1장 | XP +20 + 협업력 경험치 |
| **응원** (Encourage) | 대상 연속 실패 패널티 제거 | XP +10 + 협업력 경험치 |
| **칭찬** (Praise) | 대상 XP +15 | XP +10 + 분석력 경험치 |
| **멘토링** (Mentor) | 대상 모든 XP +10% (1분기 지속) | XP +25 + 전 스탯 경험치 |

핵심 변경:
- **WILL 직접 보상 제거** → XP + 스탯 경험치가 주 보상
- **소량 WILL 보상은 사회 기여 기금에서 지출** (아래 6.5 참조)

### 6.4 조건 + 착취 방지

| 조건 | 규칙 |
|---|---|
| 클래스 제한 | 행동자가 대상보다 **1클래스 이상 높아야** (동급은 응원·칭찬만) |
| 대상당 제한 | 같은 대상에게 **하루 최대 1회** |
| **행동자 일일 총 제한** | **하루 최대 5회** (모든 대상 합산) |
| **행동자 일일 보상 캡** | **하루 최대 50 WILL** (기금에서 지출) |
| 고유 대상만 | 같은 대상에게 **주 3회** 초과 불가 |
| 반복 페어 감쇠 | 같은 페어 반복 시 보상 50% 감소 (주 단위 리셋) |

### 6.5 WILL 보상 계산 (사회 기여 기금에서 지출)

```
행동자 WILL 보상 = 기본값 × 1/(1 + 클래스차)

기본값: 조언 8, 교육 10, 응원 5, 칭찬 5, 멘토링 12
클래스차 = 행동자 클래스 - 대상 클래스
```

**배율이 역전**되었다: 클래스 격차가 클수록 WILL 보상은 **줄어든다**.
→ 고클래스가 저클래스를 돌보는 것은 XP·스탯 성장을 위한 것이지, WILL 펌핑이 아니다.

```
예: 4클래스 → 3클래스 교육 = 10 × 1/(1+1) = 5 WILL (기금에서)
예: 9클래스 → 1클래스 칭찬 = 5 × 1/(1+8) = 0.56 → 1 WILL (최소 1)
예: 2클래스 → 1클래스 응원 = 5 × 1/(1+1) = 2.5 → 3 WILL
```

일일 캡 50 WILL과 결합하면 최대 착취 시나리오:
```
9클래스 × 5회/일 × 1 WILL = 5 WILL/일 (캡 50 미만)
4클래스 × 5회/일 × 5 WILL = 25 WILL/일 (캡 50 미만)
```
→ 어떤 시나리오에서도 일일 50 WILL 캡을 넘을 수 없다.

### 6.6 스탯 성장 (할인, 무료 아님)

v1의 "5회당 스탯 +1 무료"를 **제거**하고, 대신 할인 시스템으로 대체:

```
긍정적 행동 누적 10회 → upgrade 30% 할인 쿠폰 1장 획득

- 조언 10회 → 판단력 upgrade 30% 할인
- 교육 10회 → 협업력 upgrade 30% 할인
- 칭찬 10회 → 분석력 upgrade 30% 할인
- 멘토링 10회 → 아무 스탯 upgrade 30% 할인

쿠폰 사용: /persona upgrade "이름" --stat 판단력 --coupon
→ 비용: 현재 수치 × 30 × 0.7 = 현재 수치 × 21 WILL (소각)
```

이로써:
- upgrade 소각 경제가 **유지**된다 (할인이지 무료가 아님)
- 여전히 WILL이 소각되므로 디플레이션 구조 보존
- 긍정 행동이 **간접적으로** 스탯 성장을 돕는다

### 6.7 경제 영향 시뮬레이션

**시나리오 A (정상 운영, 100명, Epoch 0):**
```
월 채굴: 240,000 WILL
사회 기여 기금: 240,000 × 10% × 70% × 10% = 1,680 WILL/월
월 긍정 보상 (100명 × 5회 × 평균 3 WILL): 1,500 WILL
기금 잔액: +180 WILL (흑자)
비율: 1,500 / 240,000 = 0.625% (경제에 미미한 영향)
```

**시나리오 B (착취 극단, 9클래스 1명):**
```
일일 최대: 5회 × 1 WILL = 5 WILL (캡 50 미만)
월 최대: 5 × 30 = 150 WILL
vs v1의 270,000 WILL → 1,800배 축소
```
