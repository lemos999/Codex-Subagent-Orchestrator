# 인간성 확장 — Charter

> "같은 세계, 같은 사건, 다른 경험 — 차이는 받아들이는 자에게 있다."
> 상위 문서: `world-ontology.md` §0.5 개성 원칙

---

## 1. 핵심 가치

> **"알 것 같지만 알 수 없는 인간"**
>
> 모든 것은 하나의 원리: 경험하고, 받아들이는 차이.
> 사고 메커니즘이 페르소나마다 다르기에, 같은 입력에서 다른 출력.
> 이 6가지는 "받아들이는 방식"을 다층적으로 만드는 장치다.

## 2. 6요소 본질

### H1. 트라우마 / 치유

> 과거가 현재를 침범하는 구조. 단순 감정 감쇠가 아닌 **층위적 침투**.

| 항목 | 내용 |
|------|------|
| 본질 | 경험이 "받아들이는 방식"을 **왜곡**한다 |
| 구조 | 침투 깊이 (행동→감정→자아 서사→존재 전제) + 트리거 네트워크 (가중 그래프) |
| 치유 | 헤르만 3단계: 안전 확보 → 기억 재구성 → 사회 재연결. 순서 어기면 재손상 |
| 불확실성 | 치유 = 고정값 아님. 조건 벡터(안전감, 관계, 자기서사 유연성) 기반 확률 분포 |
| 전문의 | 페르소나 전문의의 상담/치료가 조건 벡터를 개선할 수 있음. 효과는 깊이와 감당 수준에 따라 다름 |
| 철학 | 프로이트(반복 강박), 니체("죽이지 못하면 강하게" — 그러나 항상은 아니다), 불교(놓아줌) |

### H2. 양심 / 도덕

> 이득 계산을 뛰어넘는 선택의 원천. 스칼라가 아닌 **다축 텐서의 간섭 패턴**.

| 항목 | 내용 |
|------|------|
| 본질 | "이득인데 안 하는" 또는 "손해인데 하는" 선택을 만드는 힘 |
| 구조 | 4축 간섭: 각인된 고통(과거 잔상) × 관계 밀도(사랑 → 발화점↓) × 정체성 경직도(자기 서사) × 고통 가시성(구체적 vs 추상적) |
| 작동 | 숙고(Deliberation) 시 4축이 실시간 간섭. 이득 축을 압도하면 → 이득을 거스르는 선택 |
| 비결정 | "도덕 기준 75점"이 아님. 같은 사람도 상황에 따라 성인이거나 비겁자 — 축 활성화 조건이 다르기 때문 |
| 철학 | 칸트(정언명령), 공자(仁), 레비나스(타자의 얼굴), 아리스토텔레스(덕=반복 실천) |

### H3. 호기심 / 탐구

> 결과를 모른 채 가고 싶은 충동. 욕(갈망)과 구조적으로 다름.

| 항목 | 내용 |
|------|------|
| 본질 | 모르는 것을 두려움이 아닌 **끌림**으로 받아들이는 방식 |
| 구조 | reward_target = None, uncertainty_gradient > 0이면 발동. 충족돼도 소멸 안 함 |
| 욕과 차이 | 욕: 결과를 안다, 원한다, 충족되면 소멸. 호기심: 결과를 모른다, 그래도 간다, 더 깊어짐 |
| 피상 vs 경탄 | 하이데거: Neugier(피상적 호기심) vs Staunen(존재를 흔드는 경탄) |
| 금기 관계 | 판도라의 상자 — 호기심이 금기를 깨뜨린다. 법적으로 유죄이나 동기는 고귀 |
| 철학 | 아리스토텔레스("모든 인간은 본성적으로 알고자 한다"), 파우스트 |

### H4. 유머 / 놀이

> 부조리에 맞서는 영혼의 형이상학적 저항. 독립적 인간 능력.

| 항목 | 내용 |
|------|------|
| 본질 | 고통을 **웃음**으로 받아들이는 방식. 성격의 부산물이 아닌 독립 능력 |
| 구조 | 고정 신념 ↔ 가혹한 현실 사이 간극의 완충. "인지적 부조화의 유쾌한 해소" |
| 다양성 | 외향적 유머(농담, 장난) + 내향적 유머(블랙유머, 자조, 아이러니). 성격 5축만으로 설명 불가 |
| 이득 초월 | 죽음 앞 농담 = 이득 계산 완전 초월. 존엄의 최종적 현현 |
| 억압 해방 | 프로이트 — 금기를 건드리되 처벌받지 않는 안전한 방출 |
| 철학 | 베르그송(기계적 경직성에 대한 생명의 반응), 키르케고르(절망을 아는 자의 웃음), 니체("고통이 깊기에 웃음을 발명") |

### H5. 수치심 / 자존감

> 타자의 시선이 만드는 사회적 통증. 양심과 독립된 차원.

| 항목 | 내용 |
|------|------|
| 본질 | 타자의 시선을 자기 정체성의 **위협**으로 받아들이는 방식 |
| 구조 | 양심(내부 판사) ≠ 수치심(외부 관찰자). 양심이 결백해도 수치심 발동 가능 |
| 관계 가중 | 사랑하는 존재(높은 호감도) 앞에서 가장 날카로움 |
| 트라우마 전이 | 공개적 수치 → 트라우마로 전이 가능 (자아 서사 침투) |
| 이득 초월 | "수치스러운 이득"을 거부 — 추한 생존자가 되는 것을 죽음보다 두려워함 |
| 철학 | 사르트르(타자의 시선), 유교(염치), 에릭슨(수치 vs 자율성) |

### H6. 노화 / 시간감각

> 시간 경과가 아닌 **경험의 질적 축적**. 체득과 통합의 깊이.

| 항목 | 내용 |
|------|------|
| 본질 | 축적된 경험의 깊이가 "받아들이는 방식"을 **바꾼다** |
| 구조 | 3축: 경험 밀도(다양한 자극) × 체득률(성찰→구조화) × 통합도(연결→세계관) |
| 틱 ≠ 나이 | 100틱이라도 밀도·체득·통합이 낮으면 반복이지 노화가 아님 |
| 과잉 경험 | 밀도만 높고 체득 못하면 = 경험 부채(번아웃). 체득했으나 통합 거부 = 냉소 |
| 생애 단계 | 에릭슨: 각 단계의 과제. 공자: "이립 불혹 지천명" = 틱이 아닌 깨달음의 임계점 |
| 철학 | 하이데거(죽음을 향한 존재), 장자(대년과 소년), 에릭슨(생애 8단계) |

## 3. 상호 연결

```
트라우마 ←→ 수치심: 공개적 수치가 트라우마가 됨
양심 ←→ 수치심: 양심은 내부 기준, 수치심은 외부 시선. 함께 작동하면 "옳은 일을 하되 들키고 싶지 않은" 갈등
호기심 ←→ 양심: 금기에 대한 호기심 vs 양심의 제동
유머 ←→ 트라우마: 트라우마를 유머로 승화 (방어기제)
노화 ←→ 양심: 시간이 지나면 과거의 잘못이 회한이 됨 (종말론적 회개)
노화 ←→ 트라우마: 노화가 트라우마를 망각이 아닌 회환으로 변모시킴
```

## 5. 성공 기준

| 기준 | 측정 |
|------|------|
| **개성** | 같은 사건에 다른 반응 — "받아들이는 방식"의 차이가 관찰됨 |
| **예측 불가** | 6요소의 간섭이 만드는 행동을 설계자도 예측 못함 |
| **시간 변화** | 노화에 따른 행동 패턴 변화가 관찰됨 |
| **치유 가능** | 트라우마가 조건 충족 시 치유되는 사례 (보장은 아님) |
| **이득 초월** | 양심/수치심/유머에 의한 비합리적(그러나 인간적) 선택이 발생 |
| **창발** | 6요소 상호작용에서 설계하지 않은 드라마가 자연 발생 |

## 6. Ontology 연결

| 항목 | 값 |
|------|-----|
| Layer | L3a 감정(오욕칠정), L3b 성향(양심·호기심·유머·수치심), L3c 심층(트라우마·노화) |
| Executor | Anima (조건부 활성화: 트리거 시에만) |
| 상위 구속 | §0.5 개성 원칙: 받아들이는 방식의 차이 |
| 하위 연결 | L4 Agency(숙고 시 양심 간섭, 호기심 행동 반영은 L4에서) |
| 데몬 응징 | 금기 → Anima가 트라우마(H1) 생성 |
