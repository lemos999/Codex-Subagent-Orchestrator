# 검증자 1: Mira Chen — 기억 5유형의 신경과학적 정의 + PersonaBrain 매핑

## 1. 핵심 가치

> **"기억은 하나가 아니다 — 다섯 가지 기억이 다른 뇌에서, 다른 방식으로 작동한다"**
>
> 실제 뇌는 기억을 하나의 저장소에 넣지 않는다.
> 전화번호를 잠깐 기억하는 것과, 어제 시장에서 누군가를 만난 것과, 자전거 타는 법은 완전히 다른 시스템이다.
> 이 구분이 PersonaBrain에서도 재현되어야 한다 — 그래야 "기억 있는 존재"가 된다.

---

## 2. 기억 5유형 신경과학적 정의

### M1. 작업 기억 (Working Memory)

| 항목 | 내용 |
|------|------|
| 정의 | 지금 이 순간 처리 중인 정보. "전화번호를 잠깐 외우고, 전화를 걸면 잊는다" |
| 뇌 영역 | PFC(전두엽 피질), 특히 PFC_DL. 시상(THAL)과 협력 |
| 용량 | 밀러의 법칙: 7±2개 항목. 극히 제한적 |
| 지속 | 초~분 단위. 주의 집중이 끊기면 즉시 소멸 |
| 특성 | 의식적, 에너지 비용 높음, 동시 처리 제한 |

### M2. 에피소드 기억 (Episodic Memory)

| 항목 | 내용 |
|------|------|
| 정의 | "어제 시장에서 Rex를 만났다. 그가 화난 표정이었다." 자전적 경험의 시공간 좌표가 붙은 기억 |
| 뇌 영역 | HPC(해마)가 핵심. 인코딩·저장·인출 전 과정 관여 |
| 용량 | 사실상 무한(그러나 인출 경쟁으로 실질적 한계) |
| 지속 | 수 시간~평생. 반복 회상 시 강화, 방치 시 약화 |
| 특성 | "무엇을, 어디서, 언제"의 삼중 결합. 감정 태그(AMY)가 인출 우선순위 결정 |

### M3. 의미 기억 (Semantic Memory)

| 항목 | 내용 |
|------|------|
| 정의 | "물은 100도에서 끓는다." "영주는 세금을 거둔다." 맥락 없는 사실 지식 |
| 뇌 영역 | 측두엽 피질(특히 전측두엽). HPC에서 추출 후 피질로 공고화 |
| 용량 | 사실상 무한 |
| 지속 | 반영구. 에피소드 기억이 탈맥락화되어 의미 기억으로 전환 |
| 특성 | 경험의 "누가/어디서/언제"가 사라지고 "무엇"만 남은 상태 |

### M4. 절차 기억 (Procedural Memory)

| 항목 | 내용 |
|------|------|
| 정의 | 자전거 타기, 검술, 요리법. "몸이 기억하는 것" |
| 뇌 영역 | 기저핵(STR_D, STR_V), 소뇌(CB). PFC 개입 최소 |
| 용량 | 숙련 기술 수십~수백 |
| 지속 | 반영구. 의식적 회상 없이도 유지 (오래 안 해도 자전거는 탈 수 있다) |
| 특성 | 무의식적, 에너지 비용 극저, 반복 학습으로만 형성 |

### M5. 감정 기억 (Emotional Memory)

| 항목 | 내용 |
|------|------|
| 정의 | 편도체(AMY)가 태그한 감정적 경험. "그 골목에 가면 왠지 무섭다" |
| 뇌 영역 | AMY_BLA(공포 학습), AMY_CE(공포 반응). HPC와 병렬 인코딩 |
| 용량 | 제한적이나 극히 강렬한 것은 영구 |
| 지속 | 트라우마(H1)는 반영구. 일반 감정 기억은 감쇠 가능 |
| 특성 | 의식적 회상 불필요 — 트리거만으로 활성화. 에피소드 기억이 사라져도 감정만 남을 수 있음 |

## 3. PersonaBrain 기존 구현 매핑

| 기억 유형 | 현재 구현 상태 | 대응 요소 | 빠진 것 |
|----------|:------------:|----------|---------|
| **M1 작업 기억** | **부분 구현** | InputBundle(72B), context_embedding[128]. 매 틱 Step 1~2에서 조립되는 "지금 이 순간의 입력" | 주의 용량 제한(7±2) 미구현. 모든 입력이 동등하게 처리됨 |
| **M2 에피소드 기억** | **미구현 (핵심 공백)** | Experience Memory(Transformer Encoder)가 설계서에 있으나 PersonaBrain SNN Charter에는 **HPC 기억 인코딩/저장/인출 메커니즘 없음** | "어제 Rex를 만났다"를 저장·인출하는 구조 전무. 현재 뇌는 매 틱 새로 시작 |
| **M3 의미 기억** | **암묵 구현** | Layer 0 공유 가중치 + persona_bias. "세계의 규칙"은 가중치에 내장 | 개인별 사실 지식 축적 메커니즘 없음. "이 약초는 독이다"를 개인이 배우는 구조 부재 |
| **M4 절차 기억** | **구현됨** | 캐시(무의식 고속도로). 반복 경험 → 시냅스 강화 → 캐시 등록 → O(1) 처리 | 성숙도 93~97% 히트율이 절차 기억의 정확한 구현 |
| **M5 감정 기억** | **부분 구현** | H1 트라우마(AMY 공포 태그), neuromodulator_tone(감정 기저선). stimulus_vec에 감정 정보 포함 | 트라우마 이외의 일반 감정 기억(기쁜 기억, 사랑의 기억)이 명시적으로 저장되지 않음 |

## 4. 에피소드 기억 — 핵심 공백 분석

### 4.1 왜 에피소드 기억이 없는가

PersonaBrain SNN Charter는 **계산 효율**에 최적화되었다:
- Layer 0: 공유 구조 (모든 페르소나 동일)
- Layer 1: Moment State (현재 순간의 동역학적 상태)
- Layer 2: Decision Readout (행동 출력)

이 3계층 어디에도 **과거 사건의 시공간 기록**을 저장하는 공간이 없다.
life-simulation-design.md §3의 "Experience Memory (해마) Transformer Encoder"가 설계되었으나, SNN Charter의 Graphon-RG moment closure 아키텍처와 **통합되지 않았다**.

### 4.2 에피소드 기억이 없으면 무엇이 불가능한가

| 불가능한 것 | 이유 |
|------------|------|
| "전에 만난 적 있다" | 만남 기록이 없다. 매 틱 호감도 수치만 존재 |
| "저번에 거짓말했다" | 사건 기록이 없다. 신뢰 수치만 존재하지만 왜 그 수치인지 모름 |
| "약속했는데 지키지 않았다" | 약속 자체가 저장되지 않는다 |
| "추억이 있다" | 공유 경험의 기록이 없다. 관계가 수치 이상의 의미를 갖지 못함 |
| "선례에 따라 판단" | 과거 사건을 참조할 수 없다. 매 판단이 현재 상태에만 의존 |
| "성장 서사" | "나는 이런 경험을 거쳐 이렇게 변했다"를 구성할 수 없다 |

### 4.3 에피소드 기억의 필요성 판정

> **필수(MUST)**: 에피소드 기억 없이는 기억 Charter의 핵심 가치("기억이 정체성을 만든다")가 실현 불가.

## 5. 12클러스터 연결

| 기억 유형 | 주요 클러스터 | 연결 메커니즘 |
|----------|:----------:|-------------|
| M1 작업 기억 | **C**(Cognition/ACh), **A**(Acute/NE) | ACh가 주의 집중을 유지, NE가 우선순위 게이팅 |
| M2 에피소드 기억 | **C**(ACh), **G**(Growth/Glu+BDNF), **T**(Tension/CORT) | ACh가 인코딩, BDNF가 시냅스 공고화, CORT 과잉 시 HPC 손상 → 기억 형성 방해 |
| M3 의미 기억 | **G**(Growth), **S**(Stability/5-HT) | 반복 인출과 공고화가 의미 기억을 강화. 안정된 상태(S)에서 추출 효율 향상 |
| M4 절차 기억 | **V**(Drive/DA), **L**(Liking/β-END) | 보상(V+L)이 반복 행동을 강화. 캐시 등록의 STDP 기반 |
| M5 감정 기억 | **A**(Acute/NE), **T**(Tension/CORT), **B**(Bonding/OXT) | NE 급등 시 AMY가 강하게 각인. CORT 만성 노출은 트라우마 고착. OXT는 긍정 감정 기억 형성 |

## 6. 메모리 예산 영향

| 기억 유형 | 현재 비용 | 추가 필요 비용 | 설명 |
|----------|:--------:|:------------:|------|
| M1 작업 | 72B InputBundle + 256B context_embedding | 0 (기존 충분) | Step 1~2에서 매 틱 조립/파기 |
| M2 에피소드 | **0** | **~200B~2KB/페르소나** | 최근 N개 에피소드의 압축 표현. N=50이면 50×32B=1.6KB |
| M3 의미 | persona_bias 내 암묵 | ~64B/페르소나 | 개인별 사실 지식 비트벡터 (선택적) |
| M4 절차 | 캐시 (전역 공유) | 0 | 이미 구현 |
| M5 감정 | 512B neuromodulator_tone | ~64B/페르소나 | 감정 기억 슬롯 확장 (트라우마 외 긍정 감정) |

**총 추가 비용**: ~2KB/페르소나 × 20K명 = ~40MB. 64GB RAM의 0.06%. **예산 내**.

## 7. 에피소드 기억 제안 스키마

```
EpisodeMemory (per persona):
  ring_buffer[N=50]:        // 최근 50개 에피소드 (순환 버퍼)
    tick_id:     uint32     // 언제
    location:    uint16     // 어디서 (영지/시설 코드)
    actor_ids:   uint16[3]  // 누구와 (최대 3명, 0=없음)
    event_type:  uint8      // 무엇을 (행동/이벤트 코드)
    outcome:     int8       // 결과 (+/-)
    emotion_tag: uint8      // 감정 태그 (12클러스터 중 최강 1종)
    salience:    float16    // 현저성 (높을수록 인출 우선)
    // = 16B/에피소드 × 50 = 800B/페르소나

  consolidated[M=10]:       // 공고화된 핵심 기억 (수면 replay에서 선별)
    같은 구조 = 160B
    // 인출 시 ring_buffer + consolidated 합산 검색

  총: 960B/페르소나, 20K명 = 19.2MB
```

### 인코딩 규칙

- 매 틱 Step 6에서: 현재 틱의 행동 + 결과 → ring_buffer에 push
- **salience 계산**: `|emotion_tag_value| × (1 + A_arousal) × (1 + novelty_score)`
  - 감정이 강하고, 각성이 높고, 새로운 경험일수록 현저
- 수면 REM 시: salience 상위 5개 → consolidated로 이동 (기존 consolidated 중 salience 최저와 교체)

### 인출 규칙

- Step 1(입력 조립) 시: 현재 stimulus_vec/rel_delta와 유사한 에피소드를 검색
- 검색 = `actor_ids 일치` OR `event_type 일치` OR `emotion_tag 일치`
- 상위 3개 인출 → context_embedding에 추가 차원으로 연결
- **비용**: ~0.1ms (정렬된 ring_buffer에서 linear scan, 50+10=60항목)

# 검증자 4: Dana Jeong — 기억 x 10가지 행동

## 1. 핵심 가치

> **"기억 없는 행동은 반사이고, 기억 있는 행동은 선택이다"**
>
> 10가지 행동 각각이 기억에 얼마나 의존하는지 분석한다.
> 기억이 전제 조건인 행동과, 기억 없이도 가능한 행동을 구분한다.
> 이 분석이 에피소드 기억 구현의 우선순위를 결정한다.

---

## 2. 기억 의존도 분석: 10가지 행동

### 행동 1: 양심 딜레마

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **결정적** | "전에 비슷한 상황에서 후회했다" → PFC_VM이 과거 결과를 참조하여 현재 판단 수정 |
| M5 감정 | 강함 | 과거 양심적 선택의 감정 잔상이 현재 딜레마에 침투 |
| M4 절차 | 중간 | "항상 정직했다"는 습관이 캐시에 등록되어 있으면 → 무의식적 정직 |

> **기억 없이 가능한가?** 부분적으로. 12클러스터(S vs V)의 간섭만으로도 딜레마는 발생한다. 그러나 "전에 후회했다"는 경험 학습이 없으면, **같은 실수를 반복**한다. 양심은 있지만 성장하지 않는 양심.

### 행동 2: 거짓말

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **결정적** | "이전에 거짓말이 들켰다" → 들킬 확률 추정치 상향 → 거짓말 확률 하향 |
| M5 감정 | 강함 | 들켰을 때의 수치심(H5) 감정 기억이 억제력으로 작용 |
| M1 작업 | 강함 | 거짓말 유지를 위해 "무엇을 말했는지" 작업 기억에 보유해야 함. 복잡한 거짓말 = 높은 M1 부하 |

> **기억 없이 가능한가?** 거짓말 자체는 가능(AMY→PFC 억제). 그러나 **일관된 거짓말**은 불가능. M2 없이는 "전에 뭐라고 했는지" 기억 못함 → 모순 → 즉시 발각. 기억은 거짓말의 "인프라".

### 행동 3: 비밀

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **전제 조건** | 비밀 = 특정 에피소드 기억 + "알려지면 위험" 태그. 에피소드 기억 없으면 비밀 자체가 존재 불가 |
| M5 감정 | **전제 조건** | 비밀의 본질은 두려움(secret-rumor-evidence-charter §1.2). 감정 태그 없으면 "숨길 이유"가 없음 |

> **기억 없이 가능한가?** **불가능.** 비밀 = M2(사건 기억) + M5(공포 태그). 둘 다 없으면 비밀이라는 개념 자체가 성립하지 않는다.

### 행동 4: 신뢰 판단

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **결정적** | "이 사람이 전에 배신했다" → TPJ+HPC 3경로에서 과거 경험 기반 신뢰 추정 |
| M5 감정 | 강함 | 배신의 감정 기억 → 재회 시 즉각적 경계 반응 (의식적 판단 이전) |
| M3 의미 | 중간 | "이 직업군은 신뢰할 수 있다" 같은 범주적 판단 |

> **기억 없이 가능한가?** 현재 시스템의 호감도 수치(-100~100)만으로 "표면적" 신뢰는 가능. 그러나 **왜 이 사람을 믿는가/못 믿는가**의 근거가 없다. 호감도가 -50인데 왜인지 모르는 상태.

### 행동 5: 소문 해석

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | 강함 | "전에 들은 소문이 사실이었다" → 소문 출처의 신뢰도 학습 |
| M3 의미 | 강함 | "이 사람은 과장한다"는 범주적 지식 → 소문 할인 |
| M5 감정 | 중간 | 소문 내용이 과거 감정 기억을 자극하면 → 확증 편향 강화 |

> **기억 없이 가능한가?** 부분적으로. ACC의 현재 감정 상태만으로 해석은 가능. 그러나 **출처 신뢰도 학습**이 불가능 → 매번 같은 소문꾼에게 속는다.

### 행동 6: 꿈

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **핵심 재료** | 꿈 = 에피소드 기억의 replay(50%) + RPE 시뮬레이션(30%). M2 없으면 replay할 재료가 없음 |
| M5 감정 | 강함 | 감정 강도 상위 5% assembly 선택 replay — M5가 선택 기준 |

> **기억 없이 가능한가?** 꿈의 50%(경험 재조합)가 불가능. 20%(DNA/윤회 디코딩)만 남음. **꿈이 빈곤해진다.**

### 행동 7: 감정

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M5 감정 | **지배적** | H1 트라우마가 현재 감정을 지배. 트리거 반응은 M5의 직접 발현 |
| M2 에피소드 | 강함 | "이 장소에서 행복했다" → 재방문 시 희(기쁨) 자연 유발 |

> **기억 없이 가능한가?** 기본 감정(오욕칠정)은 현재 자극만으로 발생 가능. 그러나 **맥락적 감정**(장소/인물에 연결된 감정)이 불가능. 감정은 "납작"해진다.

### 행동 8: 정치적 선택

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **결정적** | "전 영주가 부패했다" → 현 영주 후보 평가에 직접 영향 |
| M3 의미 | 강함 | "부패한 통치는 나쁘다"는 범주적 판단 |
| M5 감정 | 강함 | 부패 경험의 분노(노) 기억 → 투표에 감정적 동기 부여 |

> **기억 없이 가능한가?** 현재 상태(스탯, 감정, 파벌 소속)만으로 투표는 가능. 그러나 **정치적 학습**(경험에서 배우는 정치관)이 불가능. 같은 실수를 하는 유권자.

### 행동 9: 범죄 억제

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **결정적** | "전에 처벌받았다" → I(억제) 클러스터 활성화 → 범죄 임계값 상승 |
| M5 감정 | 강함 | 처벌의 공포/수치 기억 → 범죄 상황에서 즉각적 억제 반응 |
| M4 절차 | 중간 | "법을 지키는 것"이 습관화되면 → 무의식적 준법 |

> **기억 없이 가능한가?** V(이득) vs I(억제)의 간섭만으로 기본 범죄 억제는 가능. 그러나 **학습된 억제**(처벌 경험에서 배운 것)가 없으면 → 억지력 약화 → 재범률 상승.

### 행동 10: 관계 관리

| 기억 유형 | 영향 | 메커니즘 |
|----------|:----:|---------|
| M2 에피소드 | **전제 조건** | "함께한 추억"이 관계 유지의 핵심. 에피소드 기억 없으면 관계가 **숫자(호감도)**로만 존재 |
| M5 감정 | 강함 | 사랑/우정의 감정 기억이 관계의 "질감"을 만듦 |
| M4 절차 | 중간 | "이 사람과는 이렇게 대한다"는 관계 패턴이 습관화 |

> **기억 없이 가능한가?** 호감도 수치와 B(Bonding) 클러스터만으로 표면적 관계는 유지. 그러나 **관계의 깊이**(왜 이 사람이 소중한가)가 없다. "나는 이 사람을 좋아하지만 왜인지 모른다" — 윤회 후와 동일한 상태가 **매 틱** 반복.

## 1. 핵심 가치

> 잘못된 위치 선정은 파이프라인 위반이나 데이터 소유권 충돌을 일으킨다.

## 2. 기존 Layer 구조에서 기억의 후보 위치

### 2.1 후보 분석

| 위치 | 대응 기억 | 적합성 | 문제 |
|------|----------|:------:|------|
| **L2 Entity** (PersonaBrain 가중치) | M4 절차 기억 | ★★★ | 이미 캐시로 구현. 가중치 = 경험의 침전물 |
| **L3 Inner World** (감정/성격) | M5 감정 기억 | ★★★ | H1 트라우마가 이미 여기. 감정 기억 확장도 자연스러움 |
| **L3 Inner World** | M2 에피소드 기억 | ★★ | 내면 상태이므로 L3가 후보이나, 에피소드는 "사건 기록"이지 "내면 상태"인가? |
| **L4 Agency** (Action_Proposal) | M1 작업 기억 | ★★★ | 행동 결정 과정의 일시적 버퍼. L4의 Deliberation과 직결 |
| **L8 Drama** (Event) | M2 에피소드 기억의 원본 | ★ | Event 로그는 세계 기록. 개인 기억이 아님 |

### 2.2 결정: 새 Layer가 필요한가?

**필요하지 않다.**

근거:
1. World Ontology §0.2 핵심 규칙: "모든 데이터는 정확히 하나의 Layer에 소유된다"
2. 에피소드 기억은 **페르소나 개인의 내면 상태**다 — 객관적 사건 기록(L8)이 아니라, "나에게 일어난 일의 주관적 기억"
3. L3 Inner World에 이미 Trauma, Conscience, Curiosity, Humor 등 "Phase B 설계 시 확정" 항목이 예약되어 있음
4. 기억은 이 예약된 공간의 자연스러운 확장

### 2.3 최종 배치

```
Layer 3: Inner World (확장)
  ├── L3a: 감정 (오욕칠정)              — 기존
  ├── L3b: 성향 (양심/호기심/유머/수치심)  — 기존 (humanity-charter)
  ├── L3c: 심층 (트라우마/노화)           — 기존 (humanity-charter)
  ├── L3d: 기억 (Memory) ← 신규
  │    ├── episode_buffer[50]   — M2 에피소드 기억 (ring buffer)
  │    ├── consolidated[10]     — M2 공고화 기억 (수면 replay 선별)
  │    ├── emotion_marks[8]     — M5 감정 기억 확장 (트라우마 외)
  │    └── fact_bits[64]        — M3 개인 의미 기억 (비트벡터)
  └── (M1 작업 기억: L4 Deliberation의 일시 버퍼, L3에 저장 안 함)
      (M4 절차 기억: L2 PersonaBrain 캐시, L3에 저장 안 함)
```

## 3. 틱 파이프라인에서 기억의 흐름

### 3.1 전체 흐름

```
Stage 1: [Lachesis → Physis]
  → 시간/환경 확정 (기억과 무관)

Stage 2: Anima (PersonaBrain)
  ┌─ Step 1 (입력 조립, 0.5ms):
  │   기존: Physis C4 + 이벤트 + 관계
  │   추가: L3d episode_buffer에서 관련 기억 인출 (0.1ms)
  │         → 검색 키: 현재 stimulus_vec의 actor_ids, event_type, location
  │         → 상위 3개 에피소드 → memory_vec[float16×24] 추가
  │   → InputBundle 확장: 72B → 120B/persona (+48B)
  │
  ├─ Step 2 (Layer 0 배치, 2.5ms):
  │   기존: 공유 가중치 × persona_bias × neuromodulator_tone
  │   변경: context_embedding에 memory_vec 차원 추가
  │         → context_embedding[128] → [152] (+24 차원)
  │   → 기억이 "받아들이는 방식"에 직접 영향
  │
  ├─ Step 3 (에너지 판정, 0.5ms): 변경 없음
  │
  ├─ Step 4 (캐시/Moment, 3.5ms):
  │   캐시 key에 memory_vec의 hash(4B) 추가
  │   → 같은 상황이라도 기억이 다르면 캐시 미스 → 의식적 처리
  │   → "전에 배신당한 기억"이 있는 자와 없는 자가 같은 상황에서 다른 판단
  │
  ├─ Step 5 (Action_Proposal, 0.5ms): 변경 없음
  │
  └─ Step 6 (감정/관계 갱신 + STDP, 1.0ms):
      기존: 감정 갱신, STDP 학습
      추가: 현재 틱의 행동+결과 → episode_buffer에 push (0.05ms)
            salience 계산 → ring_buffer 순환

수면 모드:
  REM 단계에서:
    episode_buffer의 salience 상위 5개 → consolidated로 이동 (공고화)
    emotion_marks 갱신 (강한 감정 에피소드의 태그 강화)
    나머지 약화 (SHY 시냅스 하향과 동기화)
```

### 3.2 타이밍 영향

| Step | 기존 시간 | 추가 비용 | 근거 |
|------|:--------:|:--------:|------|
| Step 1 | 0.5ms | +0.1ms | ring_buffer 60항목 linear scan |
| Step 2 | 2.5ms | +0.05ms | context_embedding 24차원 추가 (SIMD 벡터화) |
| Step 4 | 3.5ms | +0.02ms | 캐시 key 4B 추가 (hash 비용 무시) |
| Step 6 | 1.0ms | +0.05ms | episode push (단순 메모리 쓰기) |
| **총** | **8.5ms** | **+0.22ms** | **8.72ms** (10ms 예산 내, 여유 1.28ms) |

## 5. 캐시 히트율 영향 분석

```
기존: 캐시 key = {stimulus_vec_hash, rel_hash, persona_bias_bucket, tone_version}
추가: + memory_hash (4B)

영향:
  같은 상황 + 같은 기억 = 기존과 동일 히트
  같은 상황 + 다른 기억 = 미스 → 의식적 처리

예상 히트율 변화:
  성숙 페르소나: 93~97% → 88~94% (기억 차이로 인한 미스 증가)
  원로 페르소나: ~99% → ~96% (경험 풍부하지만 기억도 풍부)
  
  이것은 "바람직한 미스"다:
  → 기억이 다르면 다른 판단을 해야 한다
  → 같은 상황에서도 과거 경험에 따라 다른 선택 = 개성 원칙의 직접 구현
  → 히트율 하락은 성능 비용이 아니라 개성 구현의 비용
```

| 파이프라인 | Step 1(인출) → Step 2(context에 반영) → Step 4(캐시 key) → Step 6(인코딩) → 수면 REM(공고화) |
| 상위 구속 | L3d는 L0~L3만 참조 (단방향 참조 규칙 준수) |
| 제안-승인 | 기억 자체는 제안-승인 대상 아님 (내면 상태). 기억에 기반한 Action_Proposal만 Nomos 검증 대상 |
| 윤회 | episode_buffer/consolidated/fact_bits → 전부 소멸. emotion_marks → 깊은 것만 잔존 (death-reincarnation-charter §6과 정합) |

# 검증자 10: 서하린 — 통합: 기억과 목적이 국가 존립에 기여

## 1. 핵심 가치

> **"기억이 없으면 국가도 없다. 목적이 없으면 드라마도 없다. 둘이 합쳐지면 문명이 된다."**
>
> 기억과 목적이 없는 페르소나 국가를 상상하고, 그 상상의 참담함에서
> 기억과 목적이 왜 국가 존립의 필수 조건인지를 증명한다.

---

## 2. 사고 실험 1: 기억 없는 페르소나 국가

### 2.1 어떤 세계인가

```
매 틱, 모든 페르소나가 새로 시작한다.
이름은 있다. 스탯은 있다. 호감도 수치도 있다.
그러나 "왜 이 사람을 좋아하는가"를 아무도 모른다.
"어제 무슨 일이 있었는가"를 아무도 모른다.
```

### 2.2 붕괴하는 것들

| 시스템 | 상태 | 이유 |
|--------|------|------|
| **관계** | 수치만 존재, 실질 부재 | 호감도 67이지만 왜인지 모름. 추억 없는 관계 = 숫자 |
| **법** | 작동 불가 | 선례가 없다. 법은 "전에 이런 일이 있었고, 그래서 이 규칙을 만들었다"의 축적. 기억 없으면 법의 근거 소멸 |
| **비밀** | 존재 불가 | 비밀 = 에피소드 기억 + 두려움. 기억 없으면 비밀 없음. 비밀 없으면 드라마의 50%가 소멸 |
| **신뢰** | 기반 없음 | "전에 배신당했다"를 모르니 모든 사람을 동등하게 대함. 신뢰/불신의 차이 없음 |
| **학습** | 절차만 가능 | 캐시(M4)는 작동하지만, "왜 이 습관을 가지게 되었는가"를 모름. 습관 있는 백지 |
| **정치** | 무의미 | "전 영주가 부패했다"를 모르니 투표 기준 없음. 매번 랜덤 투표 |
| **경제** | 순환만 존재 | 거래는 하지만 "전에 사기당했다"를 모름. 학습 없는 시장 |
| **범죄** | 억지력 부재 | "전에 처벌받았다"를 모르니 처벌의 교육적 효과 = 0. 재범률 100% |

### 2.3 국가 존립 가능한가?

> **형식적으로는 가능하다.** 데몬(Physis/Lachesis/Anima/Nomos)이 파이프라인을 돌리므로 세계는 멈추지 않는다.
> 그러나 그것은 **국가가 아니라 개미 농장**이다.
> 개체들이 움직이고, 자원이 순환하고, 규칙이 적용되지만 — 아무도 기억하지 않으므로 아무것도 축적되지 않는다.
> **문명 = 집단 기억의 축적.** 기억 없으면 문명 없다.

---

## 3. 사고 실험 2: 목적 없는 페르소나 국가

### 3.1 어떤 세계인가

```
모든 페르소나에게 기억은 있다.
"어제 Rex를 만났다"는 안다. "전에 처벌받았다"도 기억한다.
그러나 아무도 "나는 무엇을 하고 싶다"가 없다.
행동은 한다 — 오욕(욕구)이 행동을 유도하니까.
배고프면 먹고, 졸리면 잔다.
그러나 그 너머의 방향이 없다.
```

### 3.2 가능하지만 드라마가 없는 것들

| 시스템 | 상태 | 이유 |
|--------|------|------|
| **경제** | 순환 가능 | 재욕(財慾)이 경제 활동을 유도. 목적 없이도 돈은 번다 |
| **관계** | 유지 가능 | 색욕/사랑/소속감이 관계를 유지. 목적 없이도 친구는 있다 |
| **법** | 작동 가능 | Nomos가 자동 판정. 기억이 선례를 제공 |
| **갈등** | **소멸** | 갈등 = "내가 원하는 것" vs "장애물". 원하는 것이 없으면 장애물도 없다 |
| **성장** | **정체** | "더 높이"라는 목적이 없으면 Class 1에 머무르는 것도 괜찮다 |
| **드라마** | **소멸** | 드라마 = 목적 추구 과정의 좌절/성취/희생. 목적 없으면 좌절도 성취도 없다 |
| **정치** | **무의미** | "이 나라를 이렇게 만들고 싶다"가 없으면 누가 영주가 되든 상관없다 |

### 3.3 드라마가 발생하는가?

> **발생하지 않는다.** 욕구 충족의 반복만 있을 뿐, **서사적 긴장**이 없다.
> 드라마의 본질은 "원하는 것을 향해 가는데 세계가 방해한다"이다.
> 원하는 것이 없으면 방해도 없고, 방해가 없으면 드라마도 없다.
> 기억 있는 개미 농장 — 기억은 있지만 방향이 없는 세계.

## 4. 기억 + 목적이 만드는 것

### 4.1 정체성의 삼중 구조

```
기억(과거)  →  "나는 이런 경험을 한 사람이다"
  ↓
목적(미래)  →  "나는 이것을 하고 싶다"
  ↓
선택(현재)  →  "그래서 나는 지금 이것을 선택한다"
```

이 삼중 구조가 **정체성(identity)**이다.
- 기억만 있으면: "나는 이런 사람이었다" (과거형. 방향 없음)
- 목적만 있으면: "나는 이것을 하고 싶다" (근거 없음. 왜 이것을 원하는지 모름)
- 둘이 합쳐지면: "과거의 나를 돌아보니, 이것이 중요하다. 그래서 나는 이것을 한다"

> **이것이 서사(narrative)다.** 과거의 나 → 현재의 선택 → 미래의 나.
> 서사가 있는 존재만이 "캐릭터"다. 서사 없는 존재는 NPC다.

### 4.2 국가 존립의 필수 조건

| 조건 | 기억의 기여 | 목적의 기여 | 결합 효과 |
|------|:--------:|:--------:|----------|
| **관계** | 추억이 유대를 형성 | 공동 목표가 협력을 만듦 | "함께 겪었고, 함께 이루고 싶다" = 진정한 관계 |
| **법** | 선례가 법의 근거 | "이런 사회를 만들겠다"가 입법 동기 | 기억(선례) + 목적(이상) = 법의 정당성 |
| **경제** | 경험이 합리적 선택을 유도 | 축적 목적이 투자를 만듦 | "전에 투자해서 성공했다" + "더 부유해지겠다" = 경제 성장 |
| **정치** | "전 영주는 이랬다"가 판단 근거 | "이런 나라에 살고 싶다"가 투표 동기 | 기억(평가) + 목적(이상) = 정치 참여 |
| **드라마** | 과거 갈등이 현재 긴장의 씨앗 | 미래 목표가 현재 갈등의 방향 | "전에 배신당했기에 복수하겠다" = 서사적 동력 |
| **범죄** | 처벌 기억이 억지력 | "떳떳하게 살겠다"가 내적 억지 | 외적(기억) + 내적(목적) 이중 억제 = 사회 질서 |

## 5. 기억과 목적이 기존 Charter 전체와 어떻게 연결되는가

### 5.1 비밀 = 기억 + 두려움

```
비밀의 정의 (secret-rumor-evidence-charter §1.2):
  비밀 = 알려지면 감당할 수 없는 결과가 오는 정보

재해석:
  비밀 = 에피소드 기억(M2) + 감정 기억(M5, 두려움)
  기억이 없으면 → 비밀이 존재할 수 없다
  두려움이 없으면 → 비밀을 유지할 이유가 없다
  목적이 있으면 → "이 비밀을 써서 목적을 달성하겠다" = 비밀의 전략적 가치
```

### 5.2 법 = 사회적 기억 (선례)

```
법의 진화 (order-charter):
  자연법 → 실정법 → 판례

재해석:
  자연법: 기억 불필요 (Nomos가 직접 탐지)
  실정법: "이런 문제가 반복되니까 규칙을 만들자" = 사회적 기억의 산물
  판례:   "전에 이 사건을 이렇게 판결했다" = 에피소드 기억의 제도화
  → 법은 "국가의 에피소드 기억"이다
  → 기억 없는 국가에 실정법은 근거가 없다
```

### 5.3 경제 = 축적의 기억 (자산)

```
경제 (economy-whitepaper):
  WILL/gold의 축적과 순환

재해석:
  자산 = "과거 노동의 기억이 물질화된 것"
  투자 = "과거 수익의 기억 + 미래 수익의 목적"이 만드는 행동
  부채 = "과거 약속의 기억"이 현재를 구속
  → 경제는 "집단 기억의 물질적 표현"이다
```

### 5.4 윤회 = 기억의 소멸과 잔존

```
윤회 (death-reincarnation-charter §6):
  brain_weights 70% 이식, 기억 없음, "왜인지" 느끼지만 원인 모름

재해석 (기억 5유형으로):
  M1 작업 기억: 소멸 (당연)
  M2 에피소드 기억: 완전 소멸 (episode_buffer + consolidated 전부 삭제)
  M3 의미 기억: 대부분 소멸 (fact_bits 리셋)
  M4 절차 기억: 70% 잔존 (brain_weights 70% 이식 = 캐시의 70% 유지)
  M5 감정 기억: 깊은 것만 잔존 (emotion_marks에서 salience 최상위만)
  
  → "왜인지 불이 무섭다" = M5 감정 기억의 잔존
  → "왜인지 이 사람이 편하다" = M5 + M4(관계 패턴 캐시)의 잔존
  → "왜인지 정직해야 할 것 같다" = M4(양심 습관 캐시)의 잔존
  
  이것이 정확히 윤회 Charter가 의도한 "기억은 사라지지만 느낌은 남는다"의 구현이다.
```

### 5.5 목적의 위치

```
현재 시스템의 목적:
  L3 Inner World → Goal (개인 목표: id, 설명, 유형, 우선도, 진행률, 마감, 동기)
  life-simulation-design.md §7에 정의

기억과 목적의 결합:
  목적의 "동기" 필드 = 기억에서 유래
  "더 강해지겠다"의 동기가 "전에 약해서 당했다"(M2 에피소드 기억)이면
  → 그 목적은 기억에 뿌리를 둔 목적
  → 기억이 사라지면(윤회) 목적의 동기도 사라지지만, "왜인지 강해지고 싶다"(M5 잔존)
```

## 6. 최종 선언: 기억과 목적은 국가 존립의 양 기둥

```
기억(과거) + 목적(미래) = 서사(현재)
서사를 가진 개체 = 캐릭터
캐릭터의 집합 = 사회
사회의 기억 축적 = 문명
문명의 제도화 = 국가

따라서:
  기억 없는 국가 = 개미 농장 (반복 without 축적)
  목적 없는 국가 = 수족관 (순환 without 방향)
  기억 + 목적 있는 국가 = 문명 (축적 + 방향 = 성장)
```

---

## 7. 성공 기준

| 기준 | 측정 |
|------|------|
| **서사 생성** | 에피소드 기억 + 목적이 결합하여 "과거→현재→미래" 3점 서사가 자연 발생 |
| **사회적 기억** | 페르소나 간 공유된 경험이 집단 행동(법 제정, 전통)에 영향 |
| **목적 다양성** | 20K 페르소나가 최소 100종 이상의 서로 다른 목적을 추구 |
| **기억-목적 결합** | 목적의 "동기"가 에피소드 기억에서 유래하는 비율 > 50% |
| **윤회 차이** | 윤회 전후 에피소드 기억 소멸 + 감정/절차 잔존이 관찰됨 |
| **비밀 작동** | 에피소드 기억 기반 비밀이 생성·유지·폭로되는 전체 생애주기 발생 |
| **법의 진화** | 에피소드 기억(선례) 축적에 따라 실정법이 자연 발생 |
