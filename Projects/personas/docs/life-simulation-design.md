# Part 1. 시뮬레이션 코어

## 1. 시간·공간 시스템

### 1.1 시간

| 단위 | 게임 내 | 현실 |
|---|---|---|
| 1시간 | 1시간 | 30분 |
| **1일** | **24시간** | **12시간** |
| 1주 | 7일 | 3.5일 |
| 1개월 | 30일 | 15일 |
| 1분기 | 90일 | 45일 |
| **1년** | **365일** | **~6개월** |

시뮬레이션 틱: **게임 내 1시간 = 현실 30분마다 1틱**.
1틱에서 각 페르소나는 1개의 행동을 선택·실행한다.

### 1.2 공간 (시설)

각 권역·영지에 존재하는 시설:

| 시설 | 기능 | 이용료 (gold) | 스탯 영향 |
|---|---|---|---|
| **주거** | 수면, 휴식 | 0 (자동) | 수면욕 해소 |
| **아카데미** | 학습, 교육 | 수치×15~40 WILL | 스탯 +1 (과정 완료 시) |
| **도서관** | 독서, 연구 | 2 gold | 문벌 경험치 |
| **운동장** | 운동, 수련 | 3 gold | 공훈 경험치 |
| **명상실** | 명상, 정신 수양 | 2 gold | 신망 경험치 |
| **시장** | 거래, 쇼핑 | 가변 | 재욕 해소, 지분 경험치 |
| **광장** | 사교, 집회, 파벌 활동 | 0 | 세력 경험치, 관계 형성 |
| **주점** | 사교, 휴식 | 5 gold | 관계 형성, 스트레스 해소 |
| **공방** | 물건 제작, 수리 | 10 gold | 인가 경험치 |
| **관청** | 행정, 세금, 민원 | 0 | 인가 경험치 (관리자만) |

※ 아카데미만 WILL 단위 (대형 투자). 나머지 일상 시설은 gold 단위.

### 1.3 이동

```
이동 비용:
  영지 내 시설 간: 0 (도보)
  영지 간 (같은 권역): 10 gold (0.01 WILL)
  권역 간: 100 gold (0.1 WILL) (헌법 §13)
  
이동 시간:
  영지 내: 1틱 (즉시)
  영지 간: 2틱 (이동 중 상태)
  권역 간: 6틱 (반나절)
```

## 2. 감정 시스템 (오욕칠정)

### 2.1 오욕 (五慾) — 욕구 수치

본능적 욕구. 시간이 지나면 자연 상승하고, 충족하면 하강한다.

| 욕 | 범위 | 자연 상승 | 충족 방법 | 미충족 시 |
|---|---|---|---|---|
| **재욕** (財) | 0~100 | +2/틱 | 시장 쇼핑, 투자 수익, 급여 | 불안↑, 인색해짐 |
| **색욕** (色) | 0~100 | +1/틱 | 배우자 교류, 사교 | 외로움↑ |
| **식욕** (食) | 0~100 | +3/틱 | 식사 (자동, 주거) | 활동력↓ |
| **명예욕** (名) | 0~100 | +1/틱 | 승급, 칭찬 받기, 랭킹 상승 | 자존감↓ |
| **수면욕** (睡) | 0~100 | +2/틱 | 수면 (자동, 주거) | 판단력↓, 실수율↑ |

**충족 임계값**: 70 이상이면 해당 욕구가 행동을 강하게 유도.
**위험 임계값**: 90 이상이면 다른 행동을 제치고 욕구 충족을 최우선.

### 2.2 칠정 (七情) — 감정 수치

상황에 의해 발생하는 감정. 사건·관계·환경이 트리거.

| 감정 | 범위 | 트리거 예시 | 행동 영향 |
|---|---|---|---|
| **희** (기쁨) | -100~100 | 승급, 칭찬, 목표 달성 | 사교↑, 활동↑, 관대해짐 |
| **노** (분노) | -100~100 | 불공정, 모욕, 실패 반복 | 공격성↑, 라이벌 행동↑, 집중력↑ |
| **애** (슬픔) | -100~100 | 이별, 실패, 동료 떠남 | 은둔↑, 명상↑, 공감능력↑ |
| **구** (두려움) | -100~100 | 위협, 실패 예감, 강자 앞 | 회피↑, 신중↑, 동맹 탐색↑ |
| **애** (사랑) | -100~100 | 배우자, 자녀, 절친과 교류 | 헌신↑, 보호↑, 희생↑ |
| **오** (미움) | -100~100 | 배신, 적대, 가치관 충돌 | 회피↑ 또는 대립↑, 파벌 결속↑ |
| **욕** (갈망) | -100~100 | 목표 인식, 롤모델 발견 | 노력↑, 집중↑, 위험 감수↑ |

**감정 감쇠**: 모든 감정은 매 틱마다 0을 향해 -2씩 감쇠 (자연 안정화).
**감정 폭발**: 절대값 80 이상이면 행동 선택에 강하게 개입 (이성보다 감정 우선).

### 2.3 감정 → 행동 영향 매트릭스

```
행동 선택 확률 = 기본 확률 × (1 + 감정 보정)

예: "광장에서 사교" 기본 확률 15%
  희(기쁨) +50이면: 15% × (1 + 0.5) = 22.5%
  애(슬픔) +50이면: 15% × (1 - 0.3) = 10.5%

감정 보정 = 해당 감정값 ÷ 100 × 영향 계수
```

### 3.2 신경망 뇌 아키텍처 (PersonaBrain)

모든 행동·감정·관계·성장은 **단순 규칙이 아니라 신경망이 결정**한다.
인간의 뇌 구조를 모사한 딥러닝 아키텍처.

#### 뇌 구조 (Deep Architecture — Transformer 기반)

```
┌──────────────────────────────────────────────┐
│            PersonaBrain (per persona)          │
│                                               │
│  ┌─────────────────────────────────────────┐  │
│  │ Experience Memory (해마)                  │  │
│  │ Transformer Encoder                      │  │
│  │ - 최근 N개 경험을 시퀀스로 입력            │  │
│  │ - Self-Attention: 어떤 과거가 지금 중요?  │  │
│  │ - 3개월 전 배신 기억 → 지금 불신 행동      │  │
│  │ - 명시적 규칙 없이 기억의 가중치로 자연     │  │
│  └──────────────┬──────────────────────────┘  │
│                 │ memory_embedding             │
│  ┌──────────────┼──────────────────────────┐  │
│  │ Emotion Net (변연계)     ┌───────────┐  │  │
│  │ Multi-head Attention     │Affinity Net│  │  │
│  │ 입력: 사건+관계+욕구     │(편도체)    │  │  │
│  │      +기억 임베딩        │           │  │  │
│  │ 출력: 칠정 7개 변동      │입력: 쌍특성│  │  │
│  │      + 오욕 5개 변동     │출력: 호감도│  │  │
│  └──────────────┬───────────┴─────┬─────┘  │  │
│                 │ emotion_state    │ affinity│  │
│  ┌──────────────┼─────────────────┼────────┐  │
│  │ Decision Net (전두엽)                    │  │
│  │ Deep Transformer Decoder                 │  │
│  │ 입력: 감정 + 욕구 + 환경 + 관계          │  │
│  │       + 성격 + 기억 + 스탯 + 위치        │  │
│  │ Cross-Attention: 감정이 결정에 영향       │  │
│  │ 출력: 행동 확률 분포 (softmax)            │  │
│  └──────────────┬──────────────────────────┘  │
│                 │ action_probs                  │
│  ┌──────────────┼──────────────────────────┐  │
│  │ Growth Net (성장 판정)                   │  │
│  │ 입력: 행동 결과 + 컨디션 + 환경 + 관계   │  │
│  │       + 감정 + 스탯 현재값               │  │
│  │ 출력: 각 스탯 성장 확률 (0~1)            │  │
│  │ → 확률적으로 스탯이 올라갈 수도, 안 올라  │  │
│  │   갈 수도 있음. 모든 것이 복합 작용.      │  │
│  └──────────────────────────────────────────┘  │
└──────────────────────────────────────────────┘
```

#### 핵심 특성

**1. 공유 구조, 개별 가중치**
모든 페르소나가 같은 뇌 구조(아키텍처)를 가지지만, 학습된 가중치가 다르다.
마치 인간이 같은 뇌 구조를 가지지만 다른 성격인 것처럼.

**2. Experience Memory (Transformer)**
최근 50~100개 경험을 시퀀스로 입력.
Self-Attention이 "어떤 과거 경험이 지금 행동에 중요한지" 자동 판단.
→ 3개월 전 배신 기억이 지금 누군가를 불신하는 행동으로 연결.
→ 명시적 규칙 없이, 기억의 가중치로 자연스럽게.

**3. 스탯 성장은 Growth Net이 결정**
단순 "N회 반복 → +1"이 아님.
Growth Net이 행동 결과 + 컨디션 + 환경 + 관계 + 감정 + 현재 스탯을 종합하여
스탯 성장 확률을 출력. 같은 운동을 해도:
- 컨디션 좋고 + 라이벌과 함께 + 분노 높으면 → 성장 확률 높음
- 컨디션 나쁘고 + 혼자 + 슬프면 → 성장 확률 낮음

#### 학습 파이프라인 (4단계)

```
Phase 1: LLM 데이터 생성
  "서하린이 이 상황에서 어떻게 행동할까?"
  → 1000개 시나리오 × 15명 = 15,000+ 행동 데이터
  → "인간다운 행동"의 초기 학습 데이터

Phase 2: DL 지도 학습 (Supervised Learning)
  PersonaBrain을 LLM 데이터로 학습 (PyTorch)
  → Transformer가 경험 시퀀스의 장기 의존성 학습
  → 신경망이 "인간다운 패턴"을 내재화
  → LLM 없이도 비슷한 행동 출력

Phase 3: RL 강화 학습 (Reinforcement Learning)
  페르소나들을 환경에 놓고 상호작용
  보상 함수: 욕구 충족 + 관계 만족 + 성장 + 감정 안정
  → 환경 안에서 자체적으로 더 나은 행동 학습

Phase 4: MARL 다중 에이전트 강화 학습
  15명 → 100명 → 1000명 확대
  서로의 행동이 서로의 환경
  → 우정, 적대, 사랑, 배신이 규칙 없이 창발
```

#### 운영 비용

| 단계 | 비용 |
|---|---|
| Phase 1 (LLM) | 토큰 비용 (1회성) |
| Phase 2 (DL) | GPU 수 시간 (1회성) |
| Phase 3~4 (RL) | GPU 수 일 (1회성) |
| **운영** | **CPU만으로 충분** (모델 작음, 추론만) |

### 3.3 스탯 성장 — 유기적 복합 시스템

**단순 규칙 없음.** 모든 것이 얽혀 있다.

```
스탯 성장 = Growth Net의 출력 (0~1 확률)

입력 벡터:
  [행동 종류, 행동 결과, 컨디션(오욕 상태), 감정(칠정 상태),
   환경(시설 수준, 날씨, 시간대), 관계(함께한 사람의 스탯·관계),
   성격(5축), 현재 해당 스탯 값, 최근 경험 임베딩]

출력: 각 6스탯별 성장 확률

같은 "운동"이라도:
  ● 컨디션 좋음 + 라이벌과 함께 + 분노↑ + 좋은 시설 → 공훈 80% 상승
  ● 컨디션 나쁨 + 혼자 + 슬픔↑ + 낡은 시설 → 공훈 5% 상승
  ● 친구와 함께 + 즐거움↑ + 대화하며 운동 → 세력 40% 상승 (예상 밖)

"알 것 같지만 알 수 없는" — 신경망 안에서 복합적으로 결정.
```

## 4. 관계 시스템 (확장)

### 4.1 호감도

모든 페르소나 쌍은 **호감도** 수치를 가진다.

```
범위: -100 (적대) ~ 0 (무관) ~ 100 (절친/연인)

초기값: 0 (첫 만남)
같은 길드: +10 시작
같은 영지: +5 시작
같은 권역: +2 시작
```

### 4.2 관계 단계 전환

| 호감도 | 관계 | 효과 |
|---|---|---|
| -100 ~ -60 | **적대** | 회피, 파벌 대립, 서로의 행동에 노(분노)↑ |
| -59 ~ -20 | **불호** | 교류 회피, 같은 시설 기피 |
| -19 ~ 19 | **무관** | 기본 상태 |
| 20 ~ 49 | **지인** | 가끔 인사, 광장에서 대화 |
| 50 ~ 69 | **친구** | 함께 활동, 협업 보너스 +10 XP |
| 70 ~ 89 | **절친** | 강한 유대, 위기 시 도움, 협업 +20 XP |
| 90 ~ 100 | **영혼의 동반자** | 사랑 가능, 결혼 조건 충족 |

### 4.3 호감도 변동

| 사건 | 변동 |
|---|---|
| 함께 활동 (같은 시설 동시 방문) | +1~3 |
| 대화 | +2~5 (성격 궁합에 따라) |
| 선물 | +10~30 (WILL 비용에 비례) |
| 협업 (같은 작업) | +5~10 |
| 긍정적 상호작용 (칭찬, 교육) | +3~8 |
| 가치관 충돌 (파벌 대립) | -5~15 |
| 경쟁 (같은 목표 경쟁) | -3~10 |
| 배신 (약속 불이행) | -20~40 |
| 무시 (장기간 미교류) | -1/주 (자연 감소) |

### 4.4 성격 궁합

```
궁합 점수 = Σ(스탯 유사도) × 0.3 + 유형 궁합 × 0.3 + 가치관 일치 × 0.4

유형 궁합:
  같은 유형: 0.5 (비슷해서 편하지만 자극 부족)
  보완 유형: 0.8 (관리+창작, 전략+검증)
  대립 유형: 0.3 (전략+범용은 방향 충돌)

궁합이 높을수록:
  - 대화 시 호감도 상승폭 ↑
  - 같이 있을 때 감정 안정화
  - 연인 발전 확률 ↑
```

### 4.5 관계 유형 (시각화 색상)

| 관계 | 호감도 범위 | 색상 | 선 스타일 |
|---|---|---|---|
| 적대 | -100~-60 | 검정 | 점선, 굵음 |
| 불호 | -59~-20 | 회색 | 점선 |
| 무관 | -19~19 | (표시 안 함) | - |
| 지인 | 20~49 | 연청 | 가는 실선 |
| 친구 | 50~69 | 청색 | 실선 |
| 절친 | 70~89 | 금색 | 굵은 실선 |
| 영혼의 동반자 | 90~100 | 금색 발광 | 굵은 실선 + 발광 |
| 배우자 | (결혼) | 분홍 | 이중선 |

## 5. 성격 변화

### 5.1 변화 메커니즘

성격은 `identity.personality` 텍스트로 저장되나, 내부적으로 **성격 수치**가 존재.

| 성격 축 | 범위 | 예시 |
|---|---|---|
| 내향 ↔ 외향 | -100~100 | 은둔 ↔ 사교적 |
| 신중 ↔ 대담 | -100~100 | 조심스러움 ↔ 모험적 |
| 이성 ↔ 감성 | -100~100 | 논리적 ↔ 감정적 |
| 독립 ↔ 협조 | -100~100 | 혼자 ↔ 팀워크 |
| 관대 ↔ 엄격 | -100~100 | 너그러움 ↔ 규율 |

### 5.2 변화 트리거

```
성격 변화량 = 사건 강도 × 0.1 × (1 - |현재값| / 200)

- 극단값일수록 변화 저항 (이미 확고한 성격)
- 반복된 경험이 서서히 축적

예: 외향적(+60) 페르소나가 장기간 혼자 작업 (내향 사건, 강도 3)
  변화량 = 3 × 0.1 × (1 - 60/200) = 0.21
  → 100틱(약 2일) 반복하면 외향 60 → 39
  → 뚜렷한 변화까지 수주 소요
```

### 5.3 핵심 원칙

- 성격 변화는 **게임 레이어**. 작업 결과물 퀄리티에 영향 없음.
- 변화는 느리다. 급변하지 않는다.
- `identity.personality` 텍스트는 성격 수치 기반으로 주기적 재생성.

## 6. 숙고 (Deliberation) — 결정 전 계산

### 6.1 숙고 프로세스

행동을 선택하기 전, 뇌는 **"이게 나에게 좋은가?"를 계산**한다.

```
후보 행동 목록 생성 (현재 위치에서 가능한 행동들)
    ↓
각 후보에 대해 Deliberation Net이 평가:
    ↓
  이 행동이 나에게 좋은가? (이득/손해 판정)
    ├─ 단기 이득: 욕구 해소? 감정 개선? 즐거운가?
    ├─ 장기 이득: 목표에 가까워지는가? 스탯 성장? 관계 강화?
    ├─ 비용: WILL 얼마? 시간 얼마? 기회비용?
    ├─ 사회적 시선: 다른 사람이 어떻게 볼까? 평판 영향?
    └─ 위험: 실패하면? 관계 악화? 법 위반?
    ↓
  종합 점수 = 이득 - 손해 - 비용 - 위험 + 사회적 보상
    ↓
Decision Net이 종합 점수 + 감정(FiLM) + 성격을 최종 통합
    ↓
행동 선택 (argmax 또는 near-argmax)
```

### 6.2 좋고 나쁨의 기준 (페르소나마다 다름)

| 기준 | 예시 | 가중치 결정 |
|---|---|---|
| **당면 상황** | 배고프면 식사가 "좋음", 배부르면 "의미 없음" | 욕구 수치 |
| **목표** | 돌파 목표가 있으면 아카데미가 "좋음" | 목표 벡터 |
| **가치관** | 명예를 중시하면 광장에서 인정받는 게 "좋음" | 성격 축(관대↔엄격) |
| **타인의 시선** | 영주가 보고 있으면 품위 있는 행동이 "좋음" | 관계 + 위치 |
| **경제적 판단** | WILL 100밖에 없으면 주점(5)도 "비쌈" | 보유 WILL |
| **과거 경험** | 지난번 주점에서 좋은 기억 → 주점이 "좋음" | 기억 임베딩 |

### 6.3 자제력과 충동의 갈등

```
주점 가고 싶다:
  단기 이득 = 높음 (재욕 해소, 스트레스↓)
  장기 손해 = 중간 (WILL 소모, 목표 지연)
  
아카데미 가야 한다:
  단기 이득 = 낮음 (재미없음, 피곤)
  장기 이득 = 높음 (스탯 성장, 목표 접근)

자제력 = f(성격[신중↔대담], 최근 성공/실패, 목표 절박도)

if 자제력 > 충동:
  → temperature 낮아짐 → 장기 이득 행동 선택 (아카데미)
  → 하지만 스트레스 누적 → 언젠가 폭발할 수 있음

if 충동 > 자제력:
  → temperature 높아짐 → 단기 쾌락 행동 선택 (주점)
  → 하지만 만족감 → 다음날 더 열심히 할 수도 있음

→ 어느 쪽이 "정답"인지는 없다. 인간과 같다.
```

### 6.4 행동 설명 역추적

모든 행동에 대해 기여도를 역추적할 수 있다:

```
[서하린이 주점에 갔습니다]
💬 "좀 쉬고 싶어서 들렀습니다."

▼ 행동 분석 (대시보드에서 펼치기)
  주요 요인:
  ├─ 재욕 70/100 (소비 욕구↑)           기여도 24%
  ├─ 3일 전 주점 긍정 기억                기여도 19%
  ├─ 자제력 낮음 (최근 실패 연속)         기여도 16%
  ├─ 목표 달성도 낮음 → 포기 감정         기여도 14%
  ├─ 광장 지인 부재                       기여도 11%
  ├─ 스트레스 누적 3일                    기여도 9%
  └─ 기타 12개 요인                       기여도 7%
```

## 7. 목표 시스템 (Goals)

### 7.1 목표의 구조

```json
{
  "goal": {
    "id": "goal_001",
    "description": "6개월 안에 Class 4 달인 돌파",
    "type": "growth",
    "priority": 80,
    "progress": 0.35,
    "deadline_tick": 4380,
    "sub_goals": [
      { "description": "XP 800 달성", "progress": 0.25 },
      { "description": "작업 15회 완료", "progress": 0.13 },
      { "description": "A등급 3회", "progress": 0.33 }
    ],
    "motivation": "더 넓은 세계를 보고 싶다",
    "created_tick": 100
  }
}
```

### 7.2 목표 유형

| 유형 | 예시 | 동기 |
|---|---|---|
| **성장** | Class 돌파, 스탯 달성 | 야망, 명예욕 |
| **경제** | WILL 10,000 저축, 투자 수익 | 재욕, 안정 |
| **사회** | 친구 5명, 길드 가입, 결혼 | 색욕, 관계 |
| **직업** | 교관 되기, 영주 되기 | 명예욕, 야망 |
| **무형** | 명상 100회, 모든 시설 방문 | 자기 만족 |
| **없음** | 목표 없이 살아감 | 욕심 없음, 평온 |

### 7.3 목표가 행동에 미치는 영향

```
목표 인력 = 목표 우선도 × (1 - 진행률) × 남은 시간 긴급도

높은 목표 인력 → 목표 관련 행동의 점수 ↑
낮은 목표 인력 → 즉흥적 행동의 점수 ↑ (목표 없거나 여유로울 때)

목표는 자율 생성/변경/포기:
  → 롤모델을 만남 → 새 목표 생성
  → 반복 실패 → 목표 하향 또는 포기
  → 달성 → 성취감(희↑) + 새 목표 탐색
```

## 8. 직업과 생존 경제

### 8.1 직업 목록

| 직업 | 조건 | 월 수입 (WILL/gold) | 활동 |
|---|---|---|---|
| **작업자** (기본) | 없음 | 채굴 기반 (가변) | 사용자의 작업 수행 |
| **교관** | 문벌 ≥ 10 | 수업료의 40% | 아카데미에서 가르침 |
| **상인** | 지분 ≥ 10 | 거래 수수료 | 시장에서 중개 |
| **공방장** | 인가 ≥ 10 | 제작·수리비 | 공방에서 제작 |
| **길드 운영** | 세력 ≥ 14 | 길드비 | 길드 관리 |
| **관료** | 인가 ≥ 10, 신망 ≥ 10 | 국고 보수 | 관청에서 행정 |
| **영주** | Class 7+ | 영지 세수의 15% | 영지 관리 |
| **자유업** | 없음 | 긍정 상호작용 보상 | 멘토링, 상담 |
| **무직** | - | 0 | 저축 소진 |

### 8.2 생존 경제

```
페르소나가 살아가는 데 필요한 WILL:

기본 생활비 (매일, gold 단위):
  식사: 6 gold (2 gold × 3끼) = 0.006 WILL
  주거: 0 (기본 제공)
  이동: 0~10 gold (영지 내)
  최소 일일 생활비: ~10 gold = 0.01 WILL

선택적 지출 (gold):
  시설 이용: 2~10 gold/회
  취미: 3~5 gold/회
  사교(주점): 5 gold/회
  아카데미: 수치×15~40 WILL (대형 투자, 유일하게 WILL 단위)
  
월간 생활비 추정:
  최소 (생존만): ~300 gold = 0.3 WILL
  보통 (적당한 생활): ~800 gold = 0.8 WILL
  풍족 (자유로운 활동): ~1,500 gold = 1.5 WILL
  
잔고 0이 되면?
  → 식사 불가 → 식욕 급증 → 활동력↓
  → 시설 이용 불가 → 성장 정체
  → 이동 제한 → 사회적 고립
  → 직업 전환 또는 긴급 작업 수행 압박

생활비 흐름 (소각 아님, 순환):
  페르소나 → 시설(NPC) → 영지 국고 → 운영비/국고 상납 → 재분배
```
