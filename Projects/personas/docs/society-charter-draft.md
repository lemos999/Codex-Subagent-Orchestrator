## 1. 핵심 가치

> **"사회는 설계되는 것이 아니라 살아지는 것이다"**
>
> 500명이 빈 땅에 선다. 영지는 좌표를 찍는 순간 탄생하고,
> 투표는 "어떻게 결정할지"를 결정하는 첫 투표에서 시작하며,
> 직업은 필요가 만들고 시장이 다듬는다.
> 우리는 **뼈대**만 만든다 — 살은 페르소나가 붙인다.

### 1.1 설계 원칙

| 원칙 | 의미 |
|------|------|
| **최소 설계** | 헌법/경제백서에 이미 확정된 수치만 참조. 나머지는 빈칸 |
| **갈등에서 탄생** | 영지 경계 분쟁 → 경계법, 직업 수입 불공정 → 노동법, 투표 매수 → 선거법 |
| **위기 내장** | 영지 파산, 투표 마비, 실업 — 정상 작동의 일부 |
| **Physis 연동** | 날씨가 GDP를 흔들고, GDP가 직업을 흔들고, 직업이 투표를 흔든다 |
| **질서 Charter 정합** | 법 없는 시대(~틱 100) → 갈등 → 입법. 사회 시스템도 같은 리듬 |

## 2. 영지 (Territory)

### 2.1 영지란

영지 = 좌표 + 기후 + 인구 + 경제 + 정치의 최소 단위.
영주(7+클래스)가 관리하나, 영지의 **규칙**은 거주 페르소나가 만든다.

### 2.2 영지 생성

| 단계 | 내용 | 주체 |
|------|------|------|
| 1. 좌표 지정 | 450,000km² 중 무인 지대에 좌표 선택 | 영주 (7+) |
| 2. 기후 자동 적용 | Physis C2 지형시트 조회 → 고도/수계/차폐 → Δ 계산 | Physis |
| 3. 초기 인프라 배치 | 기본 시설: 관청(1) + 시장(1) + 주거(인구 비례) | Nomos |
| 4. 인구 배정 | 최소 50명. 이주 희망자 + 자동 배정 혼합 | 영주 + 시민 선택 |
| 5. 다음 틱부터 활성 | 고유 날씨, GDP 계산, 세수 시작 | 파이프라인 자동 |

**초기 5영지** (500명, 4:3:3):

| 영지 | 권역 | 인구 | 좌표 (추정) |
|------|------|:---:|------------|
| 서림 (수도) | Claude | 120 | 북위45°, 고원 분지 1,400m |
| 동림 | Claude | 80 | 북위43°, 산록 1,800m |
| 포지 (수도) | Codex | 150 | 북위25°, 삼각주 해안 |
| 비스타 (수도) | Gemini | 100 | 남위10°, 화산섬 항구 |
| 산호만 | Gemini | 50 | 남위12°, 산호초 섬 |

> 초기 배치만 고정. 이후 영지 신설은 인구 증가 + 영주 발의로.

### 2.3 영지 경제 (확정 수치 — economy-whitepaper SOT)

```
영지 GDP = Σ(소속 페르소나 분기 WILL 수입)

세수 구조:
  소득세 10% → 영지 국고
  거래세 3%  → 국고 (중앙)
  자산세 0.5%/분기 (10,000+ 보유) → 국고 (중앙)
  돌파세 5% → 국고 (중앙)

영지 국고 배분 (세수의 100%):
  ├─ 아카데미 운영: 25%
  ├─ 인프라 투자:   20%
  ├─ 사회 기여 기금: 10%
  ├─ 영주 보수:     15%
  └─ 국고 상납:     30% → 중앙 (수도 10% + 공공 10% + 비상 10%)
```

### 2.4 인프라 수준

```
환경 수준 = log₂(1 + 누적 인프라 투자 ÷ 1,000)
범위: 0~15

Lv 1~3:  XP 보너스 +5%
Lv 4~6:  XP +10%, 아카데미 비용 -10%
Lv 7~9:  XP +15%, 아카데미 -20%, 인재 유치 보너스
Lv 10+:  XP +20%, 아카데미 -30%, 특수 이벤트 해금
```

**인프라 시설 목록** (초기 가이드, 확장 가능):

| 시설 | 기능 | 비용 (WILL) | 유지비 (gold/월) |
|------|------|:-----------:|:--------------:|
| 관청 | 행정, 세금, 법 집행 | 기본 제공 | 인구×2 |
| 시장 | 거래, 물가, 상인 활동 | 기본 제공 | 인구×1 |
| 아카데미 | 훈련, 교육 | 500 | 인구×3 |
| 주점 | 사교, 소문, 관계 | 100 | 50 |
| 공방 | 제작, 수리 | 200 | 100 |
| 병원 | 치유, 트라우마 상담 | 300 | 150 |
| 감시탑 | 재난 조기 경보 | 200 | 80 |
| 제방/방재 | 홍수/태풍 피해 감소 | 400 | 200 |
| 창고 | 비상 물자 비축 | 150 | 30 |

> 새 시설 유형은 페르소나가 제안 + 의회 승인으로 추가 가능.

### 2.5 영지 경쟁

| 지표 | 계산 |
|------|------|
| 총 GDP | Σ 분기 수입 |
| 1인당 GDP | GDP ÷ 인구 |
| 승급률 | 분기 내 클래스 상승 인원 ÷ 인구 |
| 인재 유치율 | 순 이주 인원 (전입 - 전출) |
| 인프라 수준 | log₂(1 + 누적 투자 ÷ 1,000) |
| 행복도 | 주민 평균 감정(칠정 양극) |

> 순위는 분기마다 Nomos가 집계. 상위 영지에 명예 칭호 부여 (금전 보상 아님).

### 2.6 영지 위기 (위기 내장)

| 위기 | 트리거 | 결과 |
|------|--------|------|
| **파산** | 국고 < 0 유지 30일+ | 시설 유지비 미납 → 시설 폐쇄 → 인프라 수준↓ → 이주민 유출 |
| **과밀** | 인구 > 거주면적 수용력 ×1.5 | 생활비↑, 범죄↑, 불만↑ → 분할 압력 |
| **기후 재난** | Physis 재난 이벤트 | GDP 보정 -50~100% (재난 기간), 인프라 파괴, 사망 가능 |
| **영주 공백** | 영주 사망/해임 + 후계 미정 | 국고 동결, 투자 중단. 60일 내 선출 안 되면 중앙 관리로 전환 |
| **이주 유출** | 순 이주 < -20%/분기 | 세수↓ → 운영 축소 → 더 많은 유출 (악순환) |

**이주 유출 제동 장치**:
```
순이주 -20%/분기 트리거 발동
  → 1차: 중앙 국고 비상 기금에서 긴급 지원 (운영비 30일분)
  → 2차: 인접 영지와 자동 교역 협정 (세율 인하 30일)
  → 자연 복원: 인구 유출 → 과밀 해소 → 생활비↓ → 이주 유입 (음의 피드백)
  → 또한 과밀 영지의 생활비↑ → 인구 분산 → 쇠퇴 영지로 자연 유입
```

> 악순환 제동 = 중앙 지원(인위) + 시장 메커니즘(자연). 둘 다 실패 시 파산 절차로 전환.

**파산 절차**:
```
국고 적자 30일 도달
  → Nomos 파산 경고 이벤트 발행
  → 영주에게 30일 유예 (자구책 기간)
    → 구조조정: 시설 축소, 세율 인상(자치법 범위), 지원 요청
    → 실패 시: 영지 해산 또는 합병
      → 주민은 인접 영지로 이주 (자동)
      → 영주 클래스 유지, 영지 박탈
```

### 2.7 영지 분할 / 합병 / 신설

| 유형 | 조건 | 절차 |
|------|------|------|
| **신설** | 인구 100+ 무소속 시민 + 7+클래스 영주 후보 | 영주 후보 신청 → 의회 승인 → 좌표 지정 |
| **분할** | 인구 300+ 또는 과밀 위기 | 영주 동의 + 의회 승인. 새 영주 필요 |
| **합병** | 양측 영주 합의 + 양측 4+클래스 과반 투표 | 합병 후 영지 규칙 재정립 |
| **소멸** | 파산 + 구제 실패 or 인구 10 미만 30일 | Nomos 자동 해산. 주민 재배정 |

## 3. 투표 (Voting)

### 3.1 투표의 탄생 — 질서 Charter와 같은 리듬

```
Phase 0: 헌법 최소 틀 (시작부터)
  → "7+클래스가 발의 가능" (헌법 제14조)
  → "4+클래스가 투표 가능" (헌법 제15조)
  → WILL 스테이킹 기반 (경제 백서 §5)
  → 그 외 아무 규칙도 없음

Phase 1: 첫 투표 (갈등에서 탄생, ~틱 100 부근 추정)
  → 어떤 갈등이 "결정이 필요하다"는 합의를 만듦
  → 7+클래스 영주가 첫 안건 발의
  → "어떻게 투표할 것인가" 자체가 첫 투표 — 투표법 탄생
  → 이 과정 자체가 정치 드라마

Phase 2: 성숙기 (입법 누적)
  → 투표법 + 선거법 + 탄핵법 + 예산법 + ...
  → 권역별 자치법으로 분화
  → 파벌이 투표 블록을 형성
```

### 3.2 헌법 확정 투표 메커니즘 (economy-whitepaper SOT)

| 항목 | 규칙 |
|------|------|
| 참여 자격 | 4클래스+ |
| 발의 자격 | 7클래스+ (영주) |
| 메커니즘 | WILL 스테이킹 (찬성/반대 진영에 예치) |
| 잠금 | 스테이킹액의 10% (투표 기간 동안 동결) |
| 통과 조건 | 찬성 스테이킹 > 반대 스테이킹 |
| 정족수 | 전체 유통량의 5% 이상 참여 |
| EX | 투표 불가 (경제 중립) |

### 3.3 클래스 가중치

```
투표력 = 스테이킹 WILL × 가중치

Class 4: 1.0     Class 7: 2.0 (영주)
Class 5: 1.2     Class 8: 2.5
Class 6: 1.5     Class 9: 3.0 (전설)
                  EX:     0   (투표 불가)
```

### 3.4 반금권(反金權) 안전장치

| 장치 | 효과 |
|------|------|
| 발의권 분리 | 부자가 의제 지배 불가 — 발의는 영주만 |
| 1인 스테이킹 상한 | 전체 스테이킹의 20% 초과 불가 |
| 클래스 가중치 | 경험(클래스)이 부(WILL)를 보완 |
| 시민 청원 | 3클래스+ 100 WILL 서명 → 안건 상정 (헌법 제16조) |
| 파벌 상한 | 동일 파벌 합산 ≤ 총 스테이킹의 40% |

### 3.5 투표 유형 (페르소나가 만드는 부분)

> 아래는 **예측**이지 사전 설계가 아니다. 실제로는 첫 투표법이 결정한다.

| 유형 | 내용 | 발생 시기 (추정) |
|------|------|----------------|
| **법안 투표** | 실정법 제정/개정 | 첫 의회 소집 후 |
| **영주 선출** | 영주 공백 시 후임 선출 | 영주 제도 성숙 후 |
| **영주 해임** | 불신임 투표 | 해임 규칙 입법 후 |
| **세율 조정** | 영지/국가 세율 변경 | 세법 입법 후 |
| **예산 승인** | 국고 지출 계획 승인 | 예산법 입법 후 |
| **조약 비준** | 권역 간 조약 | 외교 관례 형성 후 |
| **헌법 개정** | 헌법 조항 수정 | 극히 드물게 (창조자 거부권) |

### 3.6 투표 절차 (헌법 제14~15조 기반)

```
1. 발의 — 7+클래스 영주, 또는 시민 청원(3+클래스, 100 WILL 서명)
2. 사전 심사 — 해석자(EX)가 헌법 적합성 검토
   → 위헌 판정 시: 발의 기각 + 사유 공개
   → 합헌 판정 시: 의회 상정
3. 토론 — 의회(7+클래스). 기간은 투표법으로 정함 (헌법에 없음)
4. 투표 — WILL 스테이킹. 4+클래스 참여. 기간은 투표법으로 정함
5. 집계 — Nomos 자동. 정족수(5%) + 찬성>반대 확인
6. 공포 — 해석자 최종 확인 + 공포. 효력 발생
7. 거부권 — 창조자가 행사 가능 (헌법 제15조)
```

### 3.7 투표 위기 (위기 내장)

| 위기 | 트리거 | 결과 |
|------|--------|------|
| **정족수 미달** | 참여 < 유통량 5% | 투표 무효. 재발의 필요 |
| **의회 교착** | 찬반 동률 or 파벌 교착 | 입법 정지 → 경제/사회 영향 |
| **투표 매수** | 스테이킹 대리/매수 | 발각 시 자연법 3조(계약 위반) → Nomos 탐지 |
| **보이콧** | 대규모 투표 거부 | 정족수 미달 반복 → 체제 위기 |
| **파벌 독점** | 단일 파벌이 40% 상한 우회 시도 | Nomos가 합산 검증. 우회 불가 |

## 4. 직업 (Jobs)

### 4.1 초기 9종 (가이드, 확장 가능)

| 직업 | 조건 | 월 수입 | 활동 |
|------|------|---------|------|
| **작업자** (기본) | 없음 | 채굴 기반 (가변 WILL) | 사용자의 작업 수행 |
| **교관** | 문벌 ≥ 10 | 수업료의 40% (gold) | 아카데미에서 가르침 |
| **상인** | 지분 ≥ 10 | 거래 수수료 (gold) | 시장에서 중개 |
| **공방장** | 인가 ≥ 10 | 제작·수리비 (gold) | 공방에서 제작 |
| **길드 운영** | 세력 ≥ 14 | 길드비 (gold) | 길드 관리 |
| **관료** | 인가 ≥ 10, 신망 ≥ 10 | 국고 보수 (gold) | 관청에서 행정 |
| **영주** | Class 7+ | 세수의 15% (gold) | 영지 관리 |
| **자유업** | 없음 | 긍정 상호작용 보상 (gold) | 멘토링, 상담 |
| **무직** | — | 0 | 저축 소진 |

> 단위 주의: **작업자**만 WILL 채굴. 나머지 직업은 **gold** 수입 (순환 경제).

### 4.2 수입 흐름

```
┌─── WILL 유입 (희소, 감소) ───┐
│  작업자 채굴                  │
│  tokens÷1000×rate×efficiency │
│  → 소득세 10% 자동 공제      │
└──────────────────────────────┘
         ↓ (소득세 → 영지 국고)
┌─── gold 순환 (무한) ─────────┐
│  영지 국고 → 시설 운영       │
│  시설 → 교관/상인/관료 급여  │
│  급여 → 생활비 지출          │
│  생활비 → 시설(NPC) → 국고   │
│  (완전 순환, 소각 없음)      │
└──────────────────────────────┘

유일한 소각 경로:
  결혼(200), 출산(300), 길드(500), 파벌(1,000),
  돌파(수치×30), 승천(20,260)

※ 이주 비용(100 gold = 0.1 WILL)은 소각이 아닌 시설 이전(순환).
```

### 4.3 직업과 Physis

날씨가 직업 생산성을 흔든다 (Physis C7 매트릭스):

| 날씨 조건 | 영향받는 직업 | 보정 |
|-----------|-------------|------|
| 한파/폭염 | 작업자, 상인, 공방장 | 생산성 -10~30% |
| 장마/홍수 | 상인 (교역 중단) | 수입 -50% |
| 가뭄 | 전체 (식량 가격↑) | 생활비 +20~50% |
| 맑은 날 | 전체 | 보정 없음 (기본) |
| 태풍 | 전체 (시설 파손) | 수입 -30~80%, 수리 비용 발생 |

### 4.3.1 계절 경제 사이클 (90일 주기)

Physis C1의 360일 공전 = 4계절 × 90일. 계절이 직업/영지 경제에 구조적 영향:

| 계절 | Claude (내륙 고원) | Codex (해안 평야) | Gemini (열대 군도) |
|------|-------------------|-------------------|-------------------|
| 봄 (0~90) | 융설 → 수력 회복, 교역로 개방 | 파종기, 교역 활성 | 건기, 관광 성수기 |
| 여름 (90~180) | 건조기 → 산불 위험, 생산성↑ | 장마 → 교역↓, 홍수 위험 | 태풍 시즌 시작 |
| 가을 (180~270) | 수확, 교역 절정 | 수확, 교역 회복 | 태풍 절정, 방재비↑ |
| 겨울 (270~360) | 한파 → 난방비↑(생활비 계수 1.3), 교역로 폐쇄 | 온화, 안정적 생산 | 우기 → 해운↓ |

> 계절 영향은 Physis C7 매트릭스를 통해 자동 반영. 별도 크론잡 불필요.
> 영지별 생활비 계수(Claude 1.3, Codex 1.0, Gemini 1.2)가 계절 난이도를 반영.

### 4.3.2 점진적 기후 이주

재난이 아닌 **지속적 기후 불리**에 의한 이주 메커니즘:

```
생활비 > 수입 지속 (30일+)
  → 저축 감소 → 불만 축적 (Anima 칠정: 노/오 상승)
  → 불만 임계 → 이주 동기 발생 (PersonaBrain 숙고)
  → 이주 선택: 생활비 계수 낮은 영지로 이동
  → 이것이 "겨울이 너무 길어서 떠나는" 점진적 이주
```

> 재난→파산→이주(극단적 경로)와 구별되는 자연스러운 인구 이동.
> Physis 생활비 계수 차이가 권역 간 인구 균형의 자연 조절기.

### 4.4 직업 생성과 소멸

> 초기 9종은 가이드. 실제 직업 분화는 인구·경제·문화와 함께 자연 발생.

**새 직업 생성 조건**:
```
1. 수요 존재 — 기존 직업으로 충족 안 되는 역할 (재판관, 탐험가, 의사, 기자...)
2. 페르소나 제안 — "이런 일을 전문으로 하고 싶다"
3. 경제적 존립 — 수입원이 있어야 함 (누가 대가를 지불하는가?)
4. 의회 인정 (선택적) — 공식 직업으로 인정 시 세금 혜택/국고 보수 가능
```

**직업 소멸 조건**:
```
종사자 0명 90일+ → 직업 비활성
  → 비활성 180일+ → 직업 목록에서 제거
  → 수요가 다시 생기면 재등록 가능
```

### 4.5 전직 (Career Change)

| 항목 | 규칙 |
|------|------|
| 비용 | 없음 (단, 새 직업 조건 충족 필수) |
| 쿨다운 | 30일 (무한 전직 방지) |
| 수입 공백 | 전직 중 7일 무수입 (적응 기간) |
| 강제 전직 | 영주 해임 → 무직 전환. 길드 해산 → 자유업 전환 |

### 4.6 실업과 빈곤

```
무직 상태:
  → 수입 0 → 저축 소진
  → 저축 0 도달:
    → 식사 불가 → 식욕(오욕) 급증 → 활동력↓
    → 시설 이용 불가 → 성장 정체
    → 이동 제한 → 사회적 고립
    → 압박: 직업 전환 or 긴급 작업 수행 or 범죄 유혹

구제:
  → 사회 기여 기금 (영지 국고 10%) → 긴급 지원 (최소 생활비 7일분)
  → 길드/아카데미의 교육 지원
  → 이것도 페르소나가 법으로 확장 가능 (복지법)
```

### 4.7 직업 위기 (위기 내장)

| 위기 | 트리거 | 결과 |
|------|--------|------|
| **대량 실업** | 재난/파산으로 시설 폐쇄 | 무직 급증 → 불만↑ → 범죄↑ → 정치 불안 |
| **인력 부족** | 특정 직업 종사자 부족 | 시설 운영 불가 → 서비스 중단 → 생활 불편 |
| **임금 불공정** | 직업 간 수입 격차 심화 | 노동 청원 → 임금법 입법 압력 |
| **교관 부족** | 문벌 10+ 부족 | 아카데미 교육 불가 → 성장 정체 → 클래스 고착 |
| **영주 독점** | 세수 15%가 과도 | 시민 불만 → 세율 조정 투표 → 갈등 |

## 5. 3종 연결 — 피드백 루프

### 5.1 영지 × 직업

```
영지 인프라 투자 → 시설 건설 → 직업 slot 생성
  → 교관(아카데미), 상인(시장), 공방장(공방)
  → 직업 수입 → 소비 → 시설 매출 → 국고 → 재투자
  (선순환)

영지 파산 → 시설 폐쇄 → 직업 slot 소멸
  → 실업 → 세수 감소 → 더 심한 파산
  (악순환)
```

### 5.2 영지 × 투표

```
영주(7+) = 의회 구성원 = 영지 관리자
  → 영지 GDP ↑ = 영주 권위 ↑ = 발의 영향력 ↑
  → 영지 GDP ↓ = 시민 불만 ↑ = 해임 투표 압력

세율 조정 = 투표로 결정
  → 세율↑ = 국고↑ = 인프라↑ = GDP↑ (장기)
  → 세율↑ = 시민 부담↑ = 이주↑ = GDP↓ (단기)
  → 이 긴장이 정치 드라마
```

### 5.3 직업 × 투표

```
직업 수입이 WILL 보유량 결정
  → WILL 보유량이 스테이킹 가능량 결정
  → 스테이킹이 투표력 결정
  → 따라서 직업이 정치적 영향력을 결정

그러나:
  클래스 가중치(경험) + 1인 상한(20%) + 파벌 상한(40%)
  → 부 ≠ 권력의 독점
  → "부자이지만 경험 없는" vs "가난하지만 고클래스"의 긴장
```

### 5.4 Physis → 전체

```
기후 재난 (Physis 이벤트)
  → GDP 보정 -50~100% (영지)
  → 직업 생산성↓ (직업)
  → 대응 실패 → 영주 신뢰↓ → 해임 투표 (투표)
  → 이것이 "날씨가 정치를 만드는" 인과 체인
```

## 6. 초기 (500명) — 사회의 첫 풍경

```
틱 0~50: 배정기
  → 5영지에 500명 배정. 기본 시설만 존재
  → 작업자 직업 중심. 채굴로 WILL 축적 시작
  → 관계 형성. 길드/파벌은 아직 미결성 (비용 부담)

틱 50~100: 분화기
  → 4+클래스 등장 → 길드 창설 가능 (500 WILL)
  → 문벌/지분 10+ → 교관/상인 등 직업 분화
  → 영지 간 1인당 GDP 격차 발생 → 이주 시작

틱 100~200: 갈등기 (질서 Charter §3.2와 동기)
  → 자원 경쟁, 영지 경계, 직업 수입 불공정
  → "규칙이 필요하다" → 첫 의회 → 첫 투표법
  → 첫 실정법 탄생 (아마 소유권 or 교역 규칙)

틱 200+: 정치기
  → 파벌 형성 → 투표 블록 → 입법 경쟁
  → 권역별 자치법 분화 → 3권역 정체성 확립
  → 영지 신설 수요 → 무인 지대 개척
```
