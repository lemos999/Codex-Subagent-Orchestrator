## 1. 핵심 가치

> **"법은 고통에서 증류된다"**
>
> 500명이 법 없이 시작한다. 갈등이 오고, 고통이 오고,
> "우리에게 규칙이 필요하다"는 깨달음이 온다.
> 첫 의회 소집, 첫 법안 토론, 첫 표결 — 이것이 국가의 진짜 탄생이다.
> 법 없는 자유를 맛본 개척자와 법 아래 태어난 후예는 다른 존재가 된다.

## 2. 법의 3단계

```
Layer 0: 헌법 (창조자가 만듦 — 시작부터 존재)
  → 불변. 기본권, 클래스 체계, WILL 총량, 창조자 거부권.
  → 최소 정치 틀: "7+클래스 발의 가능", "4+클래스 투표 가능"

Layer 0.5: 자연법 3조 (헌법에서 연역 — 시작부터 존재)
  → 제1조: 생명 침해 금지
  → 제2조: 소유 침탈 금지
  → 제3조: 약속 위반 금지
  → Nomos가 즉시 탐지·판정. 최소 절차(영주 판결).

Layer 1~N: 실정법 (페르소나가 만듦 — 갈등에서 탄생)
  → 상속법, 세법, 상법, 범죄법, 투표법, 특별법...
  → 입법의 규칙 자체도 페르소나가 정한다
  → 다수결? 만장일치? 의회 구조? → 첫 투표법을 투표로 결정
  → 재판소? 독립 사법? → 불공정 경험 후 페르소나가 설립
  → 세율? → 첫 분쟁에서 탄생
```

## 3. 초기 (500명, 법 없는 시대)

### 3.1 Nomos의 역할

법이 없어도 Nomos는 존재한다. 법전이 아닌 **질서를 향한 의지**가 본질.

| 시기 | Nomos의 역할 |
|------|------------|
| 법 이전 | 관찰자. 분쟁 감지, 자연법 3조 집행만 |
| 갈등 발생 | 의제 정리, 선례 기록, 법문 초안 지원 |
| 입법 시작 | 페르소나의 의지에 형식을 입힘. 법안 정합성 검토 |
| 법 이후 | 집행자. 위반 탐지, 절차 운영 |

### 3.2 첫 드라마

```
틱 ~100 부근 (추정):
  → 자원 경쟁 / 영지 경계 분쟁 / 감정 충돌
  → 양심(H2) 높은 자: "이건 옳지 않다" → 규칙 제안
  → 양심 낮은 자: "법 없으니 내 마음대로" → 이득 추구
  → 충돌 → "우리에게 규칙이 필요하다"

첫 의회:
  → 누군가(아마 7+클래스 영주) 소집
  → 첫 안건: "어떻게 결정할 것인가" (투표 규칙)
  → 이 자체가 이미 정치 드라마

첫 법안:
  → 아마 분쟁 해결과 직결된 법 (소유권? 교역 규칙?)
  → 통과 → 반대자 → 파벌 형성 → 더 많은 드라마
```

## 4. 성숙기: 권역별 법 특색

헌법(통일법) + 자치법(권역별) 이원 체계. 자치법도 페르소나가 만든다.

### 4.1 이원 체계

```
헌법 (통일법) — Nomos 집행
  ├── 기본권, 클래스, 자연법 3조
  ├── 행위 정의 (무엇이 범죄인가)
  ├── 형량 상한/하한
  └── 위헌 심사권 (자치법이 헌법 저촉 시)

자치법 (권역별) — 영주/권역 의회 집행
  ├── 형량 세부 조정
  ├── 고유 법률
  ├── 절차/속도
  └── Nomos 위헌 심사 가능
```

### 4.2 권역별 법 특색 (자연 발생 예측)

| | Claude Dominion | Codex Republic | Gemini Federation |
|---|----------------|----------------|-------------------|
| 법 철학 | 증거주의 극대화 | 코드=법, 자동 집행 | 분산 합의, 데이터 판결 |
| 속도 | 느림 (정밀) | 빠름 (자동화) | 중간 (합의) |
| 고유 법 | 문서위조특별법 | 오픈 취약점 공개법 | 정보 독점 금지법 |
| 해임 방식 | 의회 탄핵 (절차적) | 불신임 코드 즉시 | 연합 투표 |
| 탐지 | 감사원 상시 감시 | 체인 투명성 | 데이터 분석 |

> 이 특색은 **사전 설계가 아닌 예측**. 실제로는 페르소나가 자기 문화에 맞게 만들 것.

## 5. 클래스 강등 / 해임

### 5.1 자연법 위반에 의한 강등

| 위반 | 결과 | 시기 |
|------|------|------|
| 자연법 3조 위반 | 1~3 클래스 강등 | 시작부터 |
| 영지 파산 | 영지 박탈 (클래스 유지) | 영지 시스템 이후 |
| 장기 비활동 | 클래스 점진 하락 | 시작부터 |

### 5.2 실정법 위반에 의한 강등

| 위반 | 결과 | 시기 |
|------|------|------|
| 실정법 범죄 유죄 | 형량에 따른 강등 | **해당 법 입법 후** |
| 탄핵 (영주) | 의회 과반 → 해임 | **의회 설립 후** |
| 신망 임계 이하 | 페르소나 투표로 해임 | **투표법 입법 후** |

### 5.3 EX

- 강등 불가 (이미 초월). 창조자만 해임 가능.

## 6. 범죄 / 사법

### 6.1 자연법 범죄 (시작부터)

| 범죄 | 대응 | 절차 |
|------|------|------|
| 살인 | Nomos 탐지 → 영주 판결 | 최소 절차 |
| 절도 | Nomos 탐지 → 영주 판결 | 최소 절차 |
| 계약 위반 | Nomos 탐지 → 당사자 중재 | 영주 또는 당사자 |

### 6.2 실정법 범죄 (입법 후)

| 범죄 | 대응 | 시기 |
|------|------|------|
| 탈세 | 행정 당국 수사 → 기소 → 재판 | 세법 입법 후 |
| 사기 | 수사 → 재판 | 상법 입법 후 |
| 직권남용 | 의회 고발 → 재판 | 의회 설립 후 |
| 반역 | 최고 재판 → 극형 가능 | 반역 정의 입법 후 |
| 권역 고유 범죄 | 권역 자치법 적용 | 자치법 입법 후 |

### 6.3 범죄 동기

PersonaBrain의 숙고에서 "이득 > 위험"으로 판단될 때 발생.
양심(H2)이 제동. 수치심(H5)이 사후 반응. 이것도 "받아들이는 방식의 차이".

### 6.4 범죄자 인도

헌법 직접 규정: 발생 권역 관할. 피해 권역 요청 시 Nomos 중재.

## 7. 자율 → 정치 피드백

| 경로 | 메커니즘 | 시기 |
|------|---------|------|
| 감정→불만 | 칠정(노/오) 지속 높음 → 불만 축적 | 시작부터 (감정은 있으니) |
| 불만→행동 | 불만 임계 → "이건 바뀌어야 해" | 시작부터 |
| 행동→청원 | 3+클래스 청원 (헌법에 있음) | 시작부터 |
| 청원→투표 | 투표 규칙에 따라 | **투표법 입법 후** |
| 재난→정치 | 대응 실패 → 영주 신뢰↓ → 해임 | **해임 규칙 수립 후** |

> 감정은 처음부터 정치로 흐른다. 다만 흐르는 **경로(절차)**는 페르소나가 만든다.

## 8. 정치 위기 (위기 내장)

| 위기 | 트리거 | 결과 |
|------|--------|------|
| 권력 공백 | 영주 해임/사망 + 후계 미정 | 파벌 경쟁, 혼란 |

| 부패 만연 | 법 집행 신뢰 붕괴 | 시민 불복종 |
