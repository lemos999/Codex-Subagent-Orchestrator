## 주요 결정 사항

### 세계관
- 노바 = 다른 차원의 평행우주 지구 (주무대)
- 승인안 A~D 전부 확정 (도시권 체제, 4사 삭제, 프리오리안 잔류, 노바=평행우주 지구)
- 구형 기업 4사 삭제 → 벨트 운영권 지배 주체 10으로 재배정

### 페르소나 국가
- 통치: 살아있는 헌법 국가 (창조자=헌법, EX=해석자, 영주, 운영자, 시민)
- 경제: WILL 단일 화폐 + gold 소단위 (1 WILL = 1,000 gold)
- 채굴 v2: 토큰 효율 (grade_efficiency: S=1.3~F=0) + 국가 귀속
- 생활비: 소각 아님, 순환 (시설→국고→재분배)
- 3권역: 클로드 자치령 / 코덱스 공화국 / 제미니 연합
- 국가 이름: 페르소나 투표로 결정 예정

### 스탯 체계
- "능력"이 아닌 "자격·경력·기회의 증명"
- 6스탯: 인가 / 공훈 / 신망 / 문벌 / 세력 / 지분
- 임계값 6/10/14/18에서 특권 해금
- 작업 퀄리티에 영향 없음 (게임 레이어만)

### 자율 생활 시뮬레이션
- 뇌: PersonaBrain d=256, ~4M params, Transformer(Experience)+MLP(나머지)+FiLM(감정 간섭)
- 행동 결정: 주사위 아님, 인과 기반 (복잡한 입력의 복합 결과)
- 감정: 오욕칠정
- 관계: 호감도 -100~100, 7단계 전환
- 시간: 게임 1일 = 현실 12시간 (2배속)
- 목표/자제력/직업/생존 경제 시스템
- 관리: 법치 기반 (관리자가 법 제정, 직접 통제 아님)
- 학습: LLM→DL→RL→MARL (1회 학습, 이후 CPU 추론만)

### 데몬 체제 — Substrate + Cross-cutting Executors (사용자 확정, 4회 검증)

**아키텍처**: Layer(데이터) + Executor(프로세스) + Registry(계약)
- Layer 0~8: Origin → Substrate → Entity → Inner World → Agency → Social → Economy → Governance → Drama
- Executor는 레이어에 속하지 않고 **횡단 실행자**로 레이어 간을 관통
- 상세: `world-ontology.md` 참조

**4개 Executor (횡단 실행자):**
- **Physis** (자연) — 기후 물리 엔진. Stage 1 선행 실행자 (Lachesis와 병렬)
- **Lachesis** (시간) — 틱 클럭, 크론잡. Stage 1 (Physis와 병렬)
- **Anima** (영혼) — PersonaBrain 추론, 행동 **제안**. Stage 2
- **Nomos** (질서) — 행동 **승인/거부**, 경제, 법, 이벤트. Stage 3
- 파이프라인: [Physis ∥ Lachesis] → Anima → Nomos (순서 위반 물리적 불가)

**공통 원칙:**
- 위치: 국가 서열 **밖**. 헌법 부속 기관. 창조자만 관리
- 상호 heartbeat 감시 (10초 주기, 60초 무응답→재시작)
- 제안-승인 구조: Anima가 제안, Nomos가 검증·확정
- 시간 초과 시 멈춤 아닌 근사 추론 강등. 세계는 절대 멈추지 않는다
