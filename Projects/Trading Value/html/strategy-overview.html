7 Strategies

LONG ONLY Trend Following

MA 정렬 + 모멘텀 확인 후 롱 진입. 가장 기본적이고 검증된 전략. ETH에서 +26.7% OOS 달성.

conviction = trend_w * MA_align + rsi_w * RSI_pullback + vol_w * Volume_confirm + mom_w * Momentum if conviction > threshold: LONG (size = conviction * leverage)

Best for: 강한 추세장 (BTC/ETH bull run)
Weakness: 횡보/하락장에서 손실
Timeframe: 15m ~ 4h

LONG & SHORT Trend Both

추세추종이지만 하락 추세에서 숏도 허용. Conviction이 음수면 숏 포지션.

conviction = weights @ signals (range: -1 to +1) if conviction > 0: LONG if conviction < 0: SHORT size = |conviction| * leverage

Best for: 명확한 방향성 있는 시장
Weakness: 횡보장에서 양쪽 손절
Timeframe: 15m ~ 1h

LONG ONLY Mean Reversion

과매도 시 매수, 평균으로 회귀 기대. RSI/볼린저 밴드 이탈 활용.

signals_inverted = signals * [1, -1, 1, -1, 1] (RSI와 Momentum 반전: 과매도 = bullish) conviction = weights @ signals_inverted Entry: RSI < 30 AND price < BB_lower

Best for: 횡보장, 과매도 반등
Weakness: 강한 하락 추세에서 "떨어지는 칼" 잡기
Timeframe: 5m ~ 15m

LONG ONLY Breakout

가격이 저항선을 돌파할 때 추세 시작을 포착. 거래량 급증 확인.

signals_boosted = signals * [1.5, 0.5, 1.5, 1.5, 0.5] (Trend + Volume + Momentum 강화, RSI + Volatility 약화) conviction = weights @ signals_boosted Entry: price > Donchian_high AND volume_spike

Best for: 레인지 탈출, 신고가 돌파
Weakness: 거짓 돌파(fakeout)
Timeframe: 15m ~ 4h

LONG & SHORT Grid Trading

현재 가격 주위에 일정 간격으로 매수/매도. 횡보장의 왕.

sideways_score = 1.0 - |trend| - |momentum| distance = (price - entry) / entry if |distance| > grid_spacing: conviction = -sign(distance) * sideways * 0.5 (가격 올라가면 매도, 내려가면 매수)

Best for: 횡보장, 레인지 바운드
Weakness: 강한 추세에서 재고 리스크
Timeframe: 1m ~ 15m

LONG ONLY Momentum Rotation

9종목 중 최근 수익률 상위 3개만 투자. 약한 종목은 자동 청산.

for each asset: momentum_rank = sort(all_9_assets, by=ROC_24h) if this_asset in top_3: conviction = 1.0 - rank * 0.15 (1st=1.0, 2nd=0.85, 3rd=0.70) else: conviction = 0.0 (FLAT)

Best for: 섹터 로테이션, 강한 종목 집중
Weakness: 급격한 로테이션 시 지연
Timeframe: 1h ~ 1d

MARKET NEUTRAL Pair Trading

두 종목의 가격 비율이 평균에서 이탈하면 되돌림에 베팅. 시장 방향 무관.

ratio = price_A / price_B z_score = (ratio - mean_100) / std_100 if z > 2.0: SHORT A (overvalued) if z < -2.0: LONG A (undervalued) conviction = min(|z| / 4, 1.0)

Best for: 상관관계 높은 종목 (BTC/ETH, NVDA/QQQ)
Weakness: 비율이 비정상(non-stationary)이면 실패
Timeframe: 1h ~ 4h

Tournament Cycle

Week 1-4: Exploration Phase
5,000 variants run freely. No elimination. Collecting data. Minimum 30 trades per variant needed.
Week 5: First Tournament Round
Rank by CTS (Calmar 35% + Sortino 25% + Consistency 20% + DSR 10% + Frequency 10%). Bottom 20% eliminated.
Every 2 Weeks: Evolution
Bottom 20% replaced: 70% CMA-ES mutations from top 20%, 30% pure random for diversity.
Ongoing: Ensemble Production
Top 20 variants form weighted ensemble. Inverse-variance weighting. Correlation filter (max 3 with rho > 0.7).

Ranking Metric: CTS (Composite Tournament Score)

CTS = 0.35 * Calmar + 0.25 * Sortino + 0.20 * Consistency + 0.10 * DSR + 0.10 * Frequency
ComponentWeightFormulaPurpose
Calmar Ratio35%Annual Return / Max DrawdownConsistent growth with controlled risk
Sortino Ratio25%Return / Downside DeviationPenalize losses only, not upside volatility
Consistency20%Recent WR * 0.6 + Total WR * 0.4Stable win rate over time
DSR10%Sharpe - multiple testing correctionFilter lucky variants
Frequency10%min(trades / 100, 1.0)Enough data for significance
