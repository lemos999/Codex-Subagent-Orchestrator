- **문제 (명확성 축):** B-3이 A-4를 대체하는지, 보충하는지, 병렬인지 불명확.
- **개선안:** B-3을 "A-4 검증 4단계의 [1671] 특화 파라미터"로 명시.
  ```
  ## B-3. 검증 프로토콜
  A-4 검증 4단계를 따르되, 3단계(맥락)의 파라미터를 아래로 특화한다:
  - 세계관: 1670~1671년 경신대기근 조선
  - 톤앤매너: 하드코어, 친절함 배제
  - 고증: 시대적 아이템/건축/의복/언어
  ```

### [권고] F-B3: B-4 파트 분류가 A-9 시스템 분류와 연결 안 됨

- **위치:** B-4 (1053~1059행)
- **현재 상태:** "시스템 기획 / 콘텐츠 기획 / 레벨 디자인 기획 / 전투 기획" — 4개 파트. 그러나 A-9의 22종 시스템 템플릿이 이 4개 파트에 어떻게 매핑되는지 안내 없음.


### [제안] F-B4: B-6 운용 규칙의 태그 체계가 불완전

- **위치:** B-6 (1069~1074행)
- **현재 상태:** 태그 3종 (✅ 확정 / ⚠️ 미결 / 🔵 방향확정 수치 미정). Part C에서는 모두 ✅만 사용됨. ⚠️와 🔵가 실전에서 사용된 흔적 없음.
