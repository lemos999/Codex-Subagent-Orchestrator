# Generic Domain Heuristics

> 도메인 팩이 없을 때 사용하는 범용 휴리스틱.

## 가치 기여도 평가 (Value Contribution Test)

기본 VCT 4문항 사용 (SKILL.md 정의):
1. 압박 (Tension) — 해결할 문제가 있는가?
2. 선택권 (Agency) — 의미 있는 선택지가 있는가?
3. 피드백 (Feedback) — 결과가 관측 가능한가?
4. 학습 (Mastery) — 반복할수록 개선 가능한가?

## Charter 가이드

Primary Outcome 예시 참조 (도메인 불문):
- 효율성 / 안정성 / 사용자 만족 / 비용 절감 / 혁신 / 성장 / 규정 준수

## 맥락 검증 축

1. **도메인 관행:** 해당 분야의 일반적 관행/표준에 위배되지 않는가?
2. **제약조건:** Phase 0에서 수집한 제약조건을 위반하지 않는가?
3. **표현 일관성:** 산출물 전체의 용어/표현 수준이 일관적인가?
